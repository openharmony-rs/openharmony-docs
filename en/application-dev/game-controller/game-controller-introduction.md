Game Controller Kit enables games to adapt to controller peripherals (only gamepads are supported currently). It addresses player control issues and ensures a smooth user experience. Game developers can easily implement functions such as monitoring controller online/offline events and listening for button and axis events by integrating this service.

## Available Capabilities

Game Controller Kit supports the following capabilities:

- Monitor device online/offline events and query details of all online devices.

- Listen for axis and button events of gamepads.
