## A

### ArkUI Gestures

ArkUI gestures received by the Web component as a universal component, acting on the Web component itself rather than the web page.

### ArkWeb

ArkWeb provides the Web component for displaying web page content in applications.

### ArkWeb Gestures

Gestures automatically generated by the ArkWeb component from received touch events, acting on the web page.

### Adaptive Page Content Layout

A layout mode where the Web component size adapts to the web page content.

### Async Render Mode

A rendering mode where the Web component is independently displayed as a graphics surface node.

## B

### Black Screen

An issue where the web page content cannot be displayed normally and appears black.

### Breakpad

A crash information processing project. The project source code used to compile the minidump_stackwalk tool.

## C

### Canvas

A canvas component, which is an ArkUI component used for custom drawing and supports same-layer rendering.

### Canvas Node

A graphics canvas node where Web rendering follows the system's display output.

### Color Scheme

A declaration of the color scheme supported by a web page.

### CORS Policy

Cross-Origin Resource Sharing policy, a security mechanism used to control cross-origin requests.

### Crashpad

A process crash information processing tool provided by the Chromium kernel, used to record process crash information.

## D

### DOMContentLoaded

The event triggered when the HTML document has been completely loaded and parsed.

### Dark Color Scheme

A dark color scheme suitable for low-light environments.

### Data Detector

Automatically identifies information such as phone numbers and URLs on a page and provides interactive operations.

### DevTools

A set of tools for debugging frontend pages and monitoring web errors.

### DSS

Display Subsystem (DSS), a system component responsible for display composition.

### Drag and Drop

The operation of dragging an element from one location to another and dropping it.

### Dynamic Component

A component that can be dynamically mounted and removed, created through NodeController and BuilderNode.

### Dump File

A crash dump file, a binary format file that records process crash information, with the suffix dmp.

## F

### File Descriptor

A file identifier used to pass PDF file information.

### Fit Content

A web layout mode that automatically adapts web content to the container size while maintaining the correct display ratio.

### Flash Screen

A display anomaly where page content rapidly flickers or switches.

### Focus Chain

The path along which focus moves between components within an application.

### Force Dark Mode

A mode that forcibly converts web page elements to a dark style.

### FrameNode

Frame node, the basic node type in ArkUI used to build the node tree.

## G

### Gesture

Interactive actions performed by the user through touching the screen, such as tapping, long pressing, swiping, etc. The Web component supports handling various gesture events.

### Gesture Blocking

Blocking gesture events of the Web component.

### Gesture Conflict Handling

Handling conflicts between ArkUI gestures and ArkWeb gestures.

### Gesture Recognition

Identifies gestures generated by user interactions with the touchscreen.

## H

### HAP

HarmonyOS Ability Package, the deployment unit that contains application code, resources, and configuration files.

### Hole Area

The area on the screen where the camera or sensor is cut out.

## I

### Immersive Effect

The effect of extending the interface into non-safe areas to maximize screen utilization.

### Irregular Screen

A screen with a special shape, such as a notch screen, punch-hole screen, etc.

## J

### JSBridge

JavaScript bridge, a mechanism for JavaScript interaction between the application and the web page.

## L

### Light Color Scheme

A light color scheme suitable for normal lighting environments.

### Local PDF Document

A PDF document stored in the application's local resource directory.

## M

### Media Playback Hosting

Media playback of the Web component is hosted by the application side.

### Mixed Content Policy

A policy for controlling the mechanism of mixed HTTP and HTTPS content.

## N

### Named Destination

A predefined target location in a PDF document that can be used to navigate directly to a specified position.

### NativeEmbedStatus

Same-layer embedding status, the lifecycle status type of a same-layer render tag.

### NavDestination

A navigation destination component used for page navigation, which is part of the Navigation route.

### Navigation

A navigation component, which is a UI component used to manage page navigation and routing.

### Navigation Bar

The UI area at the bottom of the system used for navigation interaction.

### Network Hosting

Network requests of the Web component are hosted by the application side.

### Network PDF Document

A PDF document accessed via a network URL.

### NodeRenderType

Node render type, an enum that defines the node rendering method.

### Notch Screen

A special-shaped screen with a camera cutout at the top.

## P

### PDF Document in Application Sandbox

A PDF document stored in the application sandbox directory.

### Page Flickering

A brief visual flickering phenomenon that occurs when a page loads or navigates.

### Pre-render Web page

In scenarios where a web page is launched or navigated, a Web component is pre-created in the background to load data and complete rendering, enabling fast display.

### Pre-start render process

Before entering a web page, an empty Web component is created in advance to start the web rendering process, preparing for subsequent use.

### Print Adapter

A component used to render web page content and generate PDF file information.

## R

### Render Mode

The rendering method of the Web component content.

### Render Process Start Failed

The state where the rendering process fails to start successfully.

### RenderMode

Render mode type, an enumeration that defines the rendering modes of the Web component.

### RenderProcessMode

Render process mode type, an enumeration that defines how the Web rendering process runs.

### Resource Protocol

Application resource access protocol, used to access resource files within an application.

### Responsive Layout

A page layout technique that enables web pages to automatically adjust their layout according to different device screen sizes.

### registerNativeEmbedRule

An interface for registering same-layer embedding rules, used to specify the tag type for same-layer rendering.

## S

### Safe Area

The area on the screen that is not obscured by device hardware or system UI.

### SafeAreaEdge

Enumerates the safe area edge types, defining the expansion direction of the safe area.

### SafeAreaType

Safe area type, an enumeration that defines the types of safe area expansion.

### Same-layer Component

An ArkUI component that is rendered synchronously with H5 tags.

### Same-layer Render

A technology that synchronously renders H5 tags with ArkUI components.

### Same-layer Tag

An embed or object tag used for same-layer rendering in an H5 page.

### Scheme Handler

A scheme handler, used to intercept and process network requests of specific protocols initiated by the Web component.

### Screen Distortion

An issue where the page content is displayed in a disordered manner or with abnormal colors.

### Search Bar

A UI component for entering search keywords.

### Status Bar

The area at the top of the device that displays system status. Tap to return to the top of the page.

### Subsystem

A subsystem, which is a functional module division unit in the OpenHarmony architecture.

### Surface Node

A graphics surface node used for independent display.

### Sync Render Mode

A rendering mode in which the Web component, acting as a graphics canvas node, follows the system's display pipeline.

### System Navigation Bar

The bottom navigation bar UI area provided by the system.

### System Unsafe Area

System UI areas such as the status bar, punch-hole area, and navigation bar.

### SYNC_RENDER

Synchronous rendering mode, where the Web component acts as a graphics canvas node and follows the system's display rendering mode.

### Soft Keyboard Avoidance

Layout adjustments made by the web page to avoid the soft keyboard display area.

## T

### TextInput

A text input component. An ArkUI component for text input, supporting same-layer rendering.

### Texture Export Mode

A rendering mode in which component content is rendered through texture export.

### Theme Mode

The light or dark display theme provided by the system.

### Touch Event

An interaction event triggered when the user touches the screen.

## U

### Unsafe Area

The area on the screen that is obscured by device hardware or system UI.

### UserAgent

User agent, a string that identifies the browser type and version.

## V

### ViewTree

The view tree, which is the target component tree where components are mounted.

### Visual Flickering

A visual flickering phenomenon that occurs when page content switches.

## W

### Web Dark Mode

Enumerates the dark mode states of the Web component.

### Web Layout Mode

Enumerates the layout modes of the Web component.

### Web Page Content Height

The height value of the actual web page content.

### Web Render Process

The web rendering process, which is used to render web content.

### WebResourceResponse

The web resource response class, which is a data structure used to construct custom resource responses.

### Waterfall Page

A type of web page that uses a waterfall layout.

### WebNativeMessagingExtensionAbility

Web native messaging extension ability, a component that provides background services for browser extensions.

### White Screen

An issue where the web page content cannot be displayed normally and appears blank.

## X

### XComponent

XComponent, an ArkUI component used for custom rendering that supports same-layer rendering.
