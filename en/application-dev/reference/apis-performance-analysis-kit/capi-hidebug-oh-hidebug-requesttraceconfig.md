## Overview

Defines the configuration structure of request trace collection.

**Since**: 24

**Related module**: [HiDebug](capi-hidebug.md)

**Header file**: [hidebug_type.h](capi-hidebug-type-h.md)

## Summary

### Member Variables

| Name| Description|
| -- | -- |
| const char* identifier | Prefix of the name of the file generated by trace collection. Only the first 20 characters of the string are used as the prefix. They can contain only uppercase letters, lowercase letters, and underscores (_). If the value does not meet the requirements, it defaults to an empty string.|
| uint32_t bufferSizeKb | Cache size of the trace file, in KB. The value range is [1024, 15360]. If the input parameter is out of the valid range, the parameter will be set to the nearest boundary value.|
| uint32_t durationMs | Trace collection duration, in milliseconds. The value range is [1000, 15000]. If the input parameter is out of the valid range, the parameter will be set to the nearest boundary value.|
| uint32_t reserved | Reserved field. The value can be set to **0**.|
