## Overview

Defines the **ArkUI_EmbeddedComponentOption** parameter for **EmbeddedComponent**, which is used to set the configuration option of the component when creating or configuring **EmbeddedComponent**. It is applicable to embedded UI development scenarios where other components need to be embedded and managed within a page.
