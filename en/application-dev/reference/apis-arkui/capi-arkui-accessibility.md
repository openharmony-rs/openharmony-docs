## Overview

Describes the native capabilities provided by ArkUI Accessibility, such as querying accessibility nodes and reporting accessibility events. It is applicable to scenarios where auxiliary applications and accessibility services need to access and respond to ArkUI accessibility information, helping you build accessibility features and improve the accessibility experience of applications.
