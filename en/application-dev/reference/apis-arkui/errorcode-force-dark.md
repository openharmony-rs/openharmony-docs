## 106205 Invalid Input Parameter for the Color Inversion Capability

**Error Message**

Force dark config is invalid.

**Description**

This error code is reported when the input parameter for the color inversion capability is incorrect.

**Possible Causes**

The input parameter passed for the color inversion capability is invalid.

**Solution**

Correct the input parameter to meet the color inversion capability's requirements.
