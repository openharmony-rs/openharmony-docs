## Overview

Defines a text component controller. [OH_ArkUI_TextController_Create](capi-native-type-h.md#oh_arkui_textcontroller_create) can be used to create a text component controller object. [OH_ArkUI_TextController_Destroy](capi-native-type-h.md#oh_arkui_textcontroller_destroy) can be used to destroy the text component controller object. After the controller is created, [OH_ArkUI_TextController_SetStyledString](capi-native-type-h.md#oh_arkui_textcontroller_setstyledstring) can be used to set the styled string of the text component.
