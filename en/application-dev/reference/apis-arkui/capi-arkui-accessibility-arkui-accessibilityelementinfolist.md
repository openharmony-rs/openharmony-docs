## Overview

Provides a **List** instance of encapsulated [ArkUI_AccessibilityElementInfo](capi-arkui-accessibility-arkui-accessibilityelementinfo.md), which is used to store and manage accessibility element information in batches. It is suitable for scenarios where information of multiple accessibility elements needs to be obtained at once.
