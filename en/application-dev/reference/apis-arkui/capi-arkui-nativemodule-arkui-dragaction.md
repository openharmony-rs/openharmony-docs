## Overview

Defines a drag action, which is used to proactively initiate dragging, where you proactively call an API to start dragging, as opposed to passively responding to drag events. You can learn about the creation, configuration, and execution mechanism of **ArkUI_DragAction** in the context of the proactive drag process. For details, see [Binding Drag Events](../../ui/ndk-drag-event.md).
