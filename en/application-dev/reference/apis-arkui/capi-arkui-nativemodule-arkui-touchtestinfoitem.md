## Overview

Defines touch test information items. The touch test information items contain information about subcomponents involved in a touch test. You can obtain the subcomponent information via the [OH_ArkUI_TouchTestInfoItem_GetXXX](./capi-ui-input-event-h.md#oh_arkui_touchtestinfoitem_getx) APIs.
