## Overview

Defines a provider for third-party accessibility actions, which is used to implement callback functions. You can register and manage accessibility-related callbacks through this struct to implement custom accessibility interaction logic. It is suitable for scenarios where ArkUI accessibility capabilities need to be extended or customized.
