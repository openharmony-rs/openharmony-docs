## Overview

Provides the RenderNode capabilities of ArkUI on the native side, which can be used to create, destroy, and manage RenderNodes. You can use **ArkUI_RenderNodeHandle** to organize a RenderNode tree and set rendering attributes such as the background color, size, and position.
