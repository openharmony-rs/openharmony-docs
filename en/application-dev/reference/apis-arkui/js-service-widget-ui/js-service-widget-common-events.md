Different from private events, universal events can be bound to most components. (If a keyup event needs to be triggered to respond to the input of carriage return and space characters on the keyboard, add the click event to the outermost component.)

> **NOTE**
>
>The initial APIs of this module are supported since API version 8. Updates will be marked with a superscript to indicate their earliest API version.

| Name| Parameter| Description|
| -------- | -------- | -------- |
| click | - | Triggered when the component is clicked.|
