## 32200001 Failed to Query All Device Information in Multimodal Input

**Error Message**

Multimodal input error.

**Description**

This error code is reported when information about all devices fails to be queried in multimodal input.

**Possible Causes**

An exception occurred in multimodal input.

**Solution**

Restart the device, or collect logs for further analysis.


## 32200002 Insufficient Device Memory

**Error Message**

No memory.

**Description**

This error code is reported when the device memory is insufficient.

**Possible Causes**

The device running the game has insufficient memory.

**Solution**

Free up some memory or restart the system.
