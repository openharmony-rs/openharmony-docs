# OH_Cursor

## Overview

Defines a struct for a result set. Provides APIs to access the result set obtained by querying the RDB store.

**Since**: 10

**Related module**: [RDB](capi-rdb.md)

**Header file**: [oh_cursor.h](capi-oh-cursor-h.md)
