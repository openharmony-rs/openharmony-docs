## 890001 Parameter Error

**Error Message**

Invalid parameter. Possible causes: Parameter verification failed.

**Description**

This error code is reported if an invalid parameter value is passed to the API.

**Possible Causes**

Parameter verification failed. The possible cause is that the value of an input parameter is invalid.

**Solution**

Verify input parameter values.
