## grant<sup>9+</sup>

**Parameters**

| Name      | Type           | Mandatory  | Description           |
| --------- | --------------- | ---- | --------------- |
| resources | Array\<string\> | Yes  | List of resources that can be requested by the web page with the permission to grant.|
