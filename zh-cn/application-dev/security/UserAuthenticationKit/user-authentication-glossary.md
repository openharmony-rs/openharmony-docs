## 模系统弹窗

模系统弹窗是指被应用拉起的弹窗在整个桌面居中显示，跟拉起弹窗的应用界面位置无关。模系统弹窗被拉起后，在和用户交互完成前，整个系统被“模住”，无法操作。

## 模应用弹窗

模应用弹窗是指被被应用拉起的弹窗在拉起弹窗的应用界面内居中显示，跟桌面无关。模应用弹窗被拉起后，在和用户交互完成前，拉起弹窗的应用界面被“模住”，无法操作；其余界面可正常操作。
