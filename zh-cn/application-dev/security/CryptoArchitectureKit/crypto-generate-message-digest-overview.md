消息摘要算法是一种能将任意长度的输入消息，通过特定运算生成固定长度摘要的算法。消息摘要算法也被称为哈希算法或单向散列算法。

在摘要算法相同时，生成的摘要值主要有下列特点：

- 当输入消息相同时，生成摘要序列相同。

- 当输入消息的长度不一致时，生成摘要序列长度固定（摘要长度由算法决定）。例如SHA256会生成256位，即32字节。

## 支持的算法与规格

当创建MD消息摘要时，需要使用表中“支持种类”一列，指定MD消息摘要算法规格。

| 摘要算法 | 支持种类 | 字节长度 |  API版本 |
| -------- | -------- | -------- | -------- |
| HASH | SHA1 | 20 | 9+ |
| HASH | SHA224 | 28 | 9+ |
| HASH | SHA256 | 32 | 9+ |
| HASH | SHA384 | 48 | 9+ |
| HASH | SHA512 | 64 | 9+ |
| HASH | MD5 | 16 | 9+ |
| HASH | SM3 | 32 | 10+ |
