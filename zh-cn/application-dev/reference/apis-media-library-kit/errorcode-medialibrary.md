# 媒体库错误码

> **说明：**
>
> 以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](../errorcode-universal.md)。

<!--Del-->
## 23800107 context为空或者无效

**错误信息**

Context is invalid.

**错误描述**

上下文对象不存在或者为空时，方法会返回错误码。

**可能原因**

上下文对象不存在。

**处理步骤**

请检查上下文对象是否可用。
<!--DelEnd-->

## 23800301 系统内部错误

**错误信息**

System inner fail.

**错误描述**

媒体库内部错误。

**可能原因**

1. 数据库异常。

2. 文件系统异常。

3. IPC消息超时。

**处理步骤**

清理后台或重启设备。

## 23800151 场景参数校验不通过

**错误信息**

Scene parameter validation failed.

**错误描述**

参数异常。

**可能原因**

1. 必选参数范围不满足要求。

2. 传入的记录已存在。

3. 传入的记录数量超过最大数量。

**处理步骤**

检查参数赋值或者参数长度。
