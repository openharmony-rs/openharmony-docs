# Orientation

显示设备当前显示的方向枚举。

**起始版本：** 10

<!--Device-display-enum Orientation--><!--Device-display-enum Orientation-End-->

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## PORTRAIT

```TypeScript
PORTRAIT = 0
```

表示设备当前以竖屏方式显示。

**起始版本：** 10

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
