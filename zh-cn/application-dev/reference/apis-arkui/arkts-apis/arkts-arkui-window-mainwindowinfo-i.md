# MainWindowInfo

主窗口信息。

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager

## displayId

```TypeScript
displayId: number
```

ID of the display to which the main window belongs.

**类型：** number

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager

## label

```TypeScript
label: string
```

主窗口的任务名称。

**类型：** string

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager

## showing

```TypeScript
showing: boolean
```

Foreground/Background status of the main window. **true** if the main window is in the foreground, **false**
otherwise.

**类型：** boolean

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager

## windowId

```TypeScript
windowId: number
```

ID of the main window.

**类型：** number

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager
