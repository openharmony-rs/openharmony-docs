# RectInVP

窗口矩形区域，单位为vp。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## height

```TypeScript
height: number
```

矩形区域的高度，单位为vp。

**类型：** number

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager

## left

```TypeScript
left: number
```
