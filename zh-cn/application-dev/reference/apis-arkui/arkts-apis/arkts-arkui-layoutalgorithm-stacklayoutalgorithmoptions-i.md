## alignContent

```TypeScript
alignContent?: LocalizedAlignment
```

设置子组件在堆叠布局算法中对齐格式。

默认值：LocalizedAlignment.CENTER

非法值：按默认值处理。

**类型：** LocalizedAlignment

**默认值：** LocalizedAlignment.Center

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本24开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本24开始，该接口支持在ArkTS卡片中使用。
