# FoldCreaseRegion

折叠折痕区域。

**起始版本：** 10

**系统能力：** SystemCapability.Window.SessionManager

## creaseRects

```TypeScript
readonly creaseRects: Array<Rect>
```

折痕区域。

**类型：** Array<Rect>

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## displayId

```TypeScript
readonly displayId: number
```

屏幕ID，用于识别折痕所在的屏幕。

**类型：** number

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
