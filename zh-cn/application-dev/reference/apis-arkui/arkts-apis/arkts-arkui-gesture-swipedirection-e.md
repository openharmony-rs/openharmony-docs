```TypeScript
None
```

## Horizontal

```TypeScript
Horizontal
```

水平方向，手指滑动方向与x轴夹角小于45度时触发。

**起始版本：** 8

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

```TypeScript
Vertical
```
