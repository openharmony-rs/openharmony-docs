# TabContentState

TabContent组件的状态。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ON_SHOW

```TypeScript
ON_SHOW = 0
```

TabContent组件显示。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ON_HIDE

```TypeScript
ON_HIDE = 1
```

TabContent组件隐藏。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
