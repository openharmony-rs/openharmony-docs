# borderRadiuses

## borderRadiuses

```TypeScript
export function borderRadiuses(all: number): BorderRadiuses
```

用于生成边框圆角均设置为传入值的边框圆角对象。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
