# ResponsiveFillType

```TypeScript
declare type ResponsiveFillType = PresetFillType
```

定义一个响应式item填充模式，可以在WaterFlow、Grid、List和Swiper组件中使用。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
