# TextPicker

定义文本选择器组件实例。


## TextPicker

```TypeScript
TextPicker(options?: TextPickerOptions)
```

根据指定的数据列表创建文本选择器。

**起始版本：** 8

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为8.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| options | TextPickerOptions | 否 |  |
