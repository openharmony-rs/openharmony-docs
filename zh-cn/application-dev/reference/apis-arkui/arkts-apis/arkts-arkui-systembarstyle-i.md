# SystemBarStyle

状态栏的属性。在设置页面级状态栏属性时使用。

**起始版本：** 12

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## statusBarContentColor

```TypeScript
statusBarContentColor?: string
```

状态栏文字颜色。默认值：`'#E5FFFFFF'`。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
