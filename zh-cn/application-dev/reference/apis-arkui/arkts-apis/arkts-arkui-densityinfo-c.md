# DensityInfo

屏幕像素密度变化回调包含的信息。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## context

```TypeScript
context: UIContext
```

屏幕像素密度变化时页面对应的上下文信息。

**类型：** UIContext

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## density

```TypeScript
density: number
```

变化后的屏幕像素密度。

取值范围：[0, +∞)

**类型：** number

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
