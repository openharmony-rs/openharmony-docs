# Swiper

Defines Swiper Component instance.


## Swiper

```TypeScript
Swiper(controller?: SwiperController)
```

创建滑块视图容器。

**起始版本：** 7

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为7.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| controller | SwiperController | 否 |  |
