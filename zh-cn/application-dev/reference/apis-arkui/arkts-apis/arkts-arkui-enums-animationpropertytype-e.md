## ROTATION

```TypeScript
ROTATION = 0
```

x、y、z方向的旋转角属性。该属性对应参数个数为3，属性的单位为度（°）。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本20开始，该接口支持在原子化服务API中使用。
