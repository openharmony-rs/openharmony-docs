# DialogSheet

Dialog sheet的配置项。

**起始版本：** 26.1.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## action

```TypeScript
action: VoidCallback
```

单击选项时执行的回调。

**类型：** VoidCallback

**起始版本：** 26.1.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.1.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
