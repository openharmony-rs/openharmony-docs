## matches

```TypeScript
readonly matches: boolean
```

是否符合匹配条件。true表示满足查询条件，false表示不满足查询条件。

**类型：** boolean

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。
