# TemplateProperty

切换悬浮窗模板并修改窗口尺寸时需要提供的参数配置。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Window.SessionManager

## size

```TypeScript
size: window.Size
```

更新模板类型时需要提供的窗口尺寸。

**类型：** window.Size

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager

## templateType

```TypeScript
templateType: FloatViewTemplateType
```

标准悬浮窗的模板类型。

**类型：** FloatViewTemplateType

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager
