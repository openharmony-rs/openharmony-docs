# OrientationResult

设置窗口显示方向的执行结果。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Window.SessionManager

## executionResult

```TypeScript
executionResult : OrientationExecutionResult
```

窗口显示方向的执行结果枚举。

**类型：** OrientationExecutionResult

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
