# EventFlag

事件类型枚举。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## EVENT_PAN_GESTURE_LEFT

```TypeScript
EVENT_PAN_GESTURE_LEFT = 0x00000001
```

左滑事件。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
