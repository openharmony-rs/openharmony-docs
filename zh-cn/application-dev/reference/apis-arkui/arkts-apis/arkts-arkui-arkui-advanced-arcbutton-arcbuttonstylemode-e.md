# ArcButtonStyleMode

定义ArcButton可设置弧形按钮样式模式。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## EMPHASIZED_LIGHT

```TypeScript
EMPHASIZED_LIGHT = 0
```

强调样式，亮色，表现为蓝色背景、白色文字。

**起始版本：** 18

**原子化服务API：** 从API版本18开始，该接口支持在原子化服务API中使用。
