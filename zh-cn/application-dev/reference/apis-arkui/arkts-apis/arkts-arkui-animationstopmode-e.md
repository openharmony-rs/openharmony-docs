# AnimationStopMode

动图停止模式。

**起始版本：** 24

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FIRST_FRAME

```TypeScript
FIRST_FRAME = 0
```

动图停止时回到首帧。

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## LAST_FRAME

```TypeScript
LAST_FRAME = 1
```

动图停止时停留在最后一帧。

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
