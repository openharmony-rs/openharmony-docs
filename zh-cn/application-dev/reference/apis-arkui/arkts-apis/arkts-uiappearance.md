# @ohos.uiAppearance

用户界面外观提供获取系统外观的一些基础能力，包括获取深浅色模式、字体大小缩放比例、字体粗细缩放比例。

> **说明：**

> 从API version 20开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.UiAppearance

**系统接口：** 此接口为系统接口。
