# ColorSpace

色域模式。

**起始版本：** 8

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## DEFAULT

```TypeScript
DEFAULT = 0
```

默认SRGB色域模式。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## WIDE_GAMUT

```TypeScript
WIDE_GAMUT = 1
```

广色域模式。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
