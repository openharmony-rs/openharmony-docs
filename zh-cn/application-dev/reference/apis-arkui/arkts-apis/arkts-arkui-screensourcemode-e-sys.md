# ScreenSourceMode（系统接口）

屏幕显示内容来源模式枚举。

**起始版本：** 10

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SCREEN_MAIN

```TypeScript
SCREEN_MAIN = 0
```

表示屏幕为默认主屏。

**起始版本：** 10

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SCREEN_MIRROR

```TypeScript
SCREEN_MIRROR = 1
```

表示屏幕内容来自镜像。

**起始版本：** 10

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SCREEN_EXTEND

```TypeScript
SCREEN_EXTEND = 2
```

表示屏幕内容来自扩展。

**起始版本：** 10

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SCREEN_ALONE
