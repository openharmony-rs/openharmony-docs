# Vector4

用于表示包含x、y、z、w四个值的向量。

**起始版本：** 26.0.0

## w

```TypeScript
w: number
```

向量w轴方向的值。取值范围：(-∞, +∞)。

**类型：** number

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本26.0.0开始，该接口支持在原子化服务API中使用。
