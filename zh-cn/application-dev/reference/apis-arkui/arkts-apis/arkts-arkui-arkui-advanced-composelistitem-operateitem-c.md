
## arrow

```TypeScript
arrow?: OperateIcon
```

右侧元素为箭头，大小为12*24vp。

默认不设置或设置为undefined，右侧箭头不显示。

**类型：** OperateIcon

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
