# VideoCallControlGroup

视频通话控件组枚举。仅当[PiPTemplateType](arkts-arkui-piptemplatetype-e.md) 为VIDEO_CALL时使用。

**起始版本：** 12

**系统能力：** SystemCapability.Window.SessionManager

## MICROPHONE_SWITCH

```TypeScript
MICROPHONE_SWITCH = 201
```

打开/关闭麦克风控件组。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
