# TimePicker

定义时间选择器组件实例。


## TimePicker

```TypeScript
TimePicker(options?: TimePickerOptions)
```

创建滑动选择器，默认使用24小时的时间区间。

**起始版本：** 8

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为8.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| options | TimePickerOptions | 否 |  |
