# ExpandOption（系统接口）

扩展屏幕的参数。

**起始版本：** 9

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## screenId

```TypeScript
screenId: number
```

屏幕的id，该参数应为整数。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## startX

```TypeScript
startX: number
```

屏幕的起始X轴坐标，该参数应为整数。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
