## AUTO

```TypeScript
AUTO = 0
```

两个或两个以下的按钮水平排列，且两个或两个以上的按钮垂直排列。

**起始版本：** 26.1.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本26.1.0开始，该接口支持在原子化服务API中使用。
