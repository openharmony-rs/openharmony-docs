# RectChangeReason

窗口矩形（窗口位置及窗口大小）变化的原因。

**起始版本：** 12

**系统能力：** SystemCapability.Window.SessionManager

## UNDEFINED

```TypeScript
UNDEFINED = 0
```

默认值。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
