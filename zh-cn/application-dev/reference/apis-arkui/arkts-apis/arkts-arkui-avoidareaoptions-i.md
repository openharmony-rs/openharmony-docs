# AvoidAreaOptions

系统避让区变化后返回当前避让区域以及避让区域类型。

**起始版本：** 12

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## area

```TypeScript
area: AvoidArea
```

系统避让区变化后返回的避让区域。

**类型：** AvoidArea

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## type

```TypeScript
type: AvoidAreaType
```

系统避让区变化后返回的避让区域类型。

**类型：** AvoidAreaType

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
