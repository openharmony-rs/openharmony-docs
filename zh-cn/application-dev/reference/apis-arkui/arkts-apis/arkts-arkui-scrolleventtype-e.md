# ScrollEventType

ScrollEvent type.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SCROLL_START

```TypeScript
SCROLL_START = 0
```

When the ScrollEvent start.

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SCROLL_STOP

```TypeScript
SCROLL_STOP = 1
```

When the ScrollEvent stop.

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
