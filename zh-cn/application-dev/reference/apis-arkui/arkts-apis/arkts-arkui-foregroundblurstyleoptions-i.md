# ForegroundBlurStyleOptions

继承自[BlurStyleOptions](arkts-arkui-common-blurstyleoptions-i.md#BlurStyleOptions)，设置内容模糊选项。

**继承/实现关系：** ForegroundBlurStyleOptions extends [BlurStyleOptions](arkts-arkui-common-blurstyleoptions-i.md#BlurStyleOptions)

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
