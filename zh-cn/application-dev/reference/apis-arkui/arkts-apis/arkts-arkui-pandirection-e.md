# PanDirection

```TypeScript
declare enum PanDirection
```

与SwipeDirection不同，PanDirection没有角度限制。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## None

```TypeScript
None = 0
```

任何方向都不可触发滑动手势事件。

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
