# StatusBarProperty

状态栏的属性。在获取状态栏属性信息时返回。

**起始版本：** 18

**系统能力：** SystemCapability.Window.SessionManager

## contentColor

```TypeScript
contentColor: string
```

状态栏文字颜色，固定为ARGB格式，如：`#E5FFFFFF`。

**类型：** string

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
