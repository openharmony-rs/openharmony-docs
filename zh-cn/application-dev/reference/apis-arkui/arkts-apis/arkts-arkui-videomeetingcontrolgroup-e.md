# VideoMeetingControlGroup

视频会议控件组枚举。仅当[PiPTemplateType](arkts-arkui-piptemplatetype-e.md) 为VIDEO_MEETING时使用。

**起始版本：** 12

**系统能力：** SystemCapability.Window.SessionManager

## HANG_UP_BUTTON

```TypeScript
HANG_UP_BUTTON = 301
```

挂断控件组。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
