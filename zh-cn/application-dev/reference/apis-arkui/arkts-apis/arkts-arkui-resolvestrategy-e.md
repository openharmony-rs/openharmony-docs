# ResolveStrategy

UIContext对象的解析策略。

**起始版本：** 22

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CALLING_SCOPE

```TypeScript
CALLING_SCOPE = 0
```

获取调用作用域的UIContext。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
