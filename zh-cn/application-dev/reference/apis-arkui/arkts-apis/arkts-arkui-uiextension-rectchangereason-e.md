# RectChangeReason

```TypeScript
enum RectChangeReason
```

组件（EmbeddedComponent或UIExtensionComponent）矩形（位置及尺寸）变化的原因。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
