# Dimension

```TypeScript
declare type Dimension = PX | VP | FP | LPX | Percentage | Resource
```

定义具有数值和单位（vp|px|fp|lpx|%）以及资源的尺寸属性。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本23开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本23开始，该接口支持在ArkTS卡片中使用。
