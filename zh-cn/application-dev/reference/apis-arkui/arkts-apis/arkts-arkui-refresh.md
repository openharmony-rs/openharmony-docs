# Refresh

可以进行页面下拉操作并显示刷新动效的容器组件。


## Refresh

```TypeScript
Refresh(value: RefreshOptions)
```

创建Refresh容器。

**起始版本：** 8

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为8.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| value | RefreshOptions | 是 |  |
