# WindowMode（系统接口）

窗口模式枚举。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## UNDEFINED

```TypeScript
UNDEFINED = 1
```

表示APP未定义窗口模式。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## FULLSCREEN

```TypeScript
FULLSCREEN = 2
```

表示APP全屏模式。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## PRIMARY

```TypeScript
PRIMARY = 3
```

表示APP分屏多窗口主要模式。上下分屏时，上分屏为主要模式。左右分屏时，左分屏为主要模式。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SECONDARY

```TypeScript
SECONDARY = 4
```

表示APP分屏多窗口次要模式。上下分屏时，下分屏为次要模式。左右分屏时，右分屏为次要模式。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## FLOATING

```TypeScript
FLOATING = 5
```

表示APP自由悬浮形式窗口模式。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
