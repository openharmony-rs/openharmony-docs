# DragState（系统接口）

拖拽状态。

**起始版本：** 10

**系统能力：** SystemCapability.Msdp.DeviceStatus.Drag

**系统接口：** 此接口为系统接口。

## MSG_DRAG_STATE_START

```TypeScript
MSG_DRAG_STATE_START = 1
```

表示开始拖拽。

**起始版本：** 10

**系统能力：** SystemCapability.Msdp.DeviceStatus.Drag

**系统接口：** 此接口为系统接口。

## MSG_DRAG_STATE_STOP

```TypeScript
MSG_DRAG_STATE_STOP = 2
```

表示结束拖拽。

**起始版本：** 10

**系统能力：** SystemCapability.Msdp.DeviceStatus.Drag

**系统接口：** 此接口为系统接口。

## MSG_DRAG_STATE_CANCEL

```TypeScript
MSG_DRAG_STATE_CANCEL = 3
```

表示取消拖拽。

**起始版本：** 10

**系统能力：** SystemCapability.Msdp.DeviceStatus.Drag

**系统接口：** 此接口为系统接口。
