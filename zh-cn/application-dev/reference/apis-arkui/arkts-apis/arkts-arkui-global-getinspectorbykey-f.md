# getInspectorByKey

## getInspectorByKey

```TypeScript
export declare function getInspectorByKey(id: string): string
```

根据id获取组件的所有属性。

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | string | 是 | ID为声明组件时通过id或者key方法传入的参数。 |

**返回值：**

| 类型 | 说明 |
| --- | --- |
| string | @syscap SystemCapability.ArkUI.ArkUI.Full@crossplatform@atomicservice |
