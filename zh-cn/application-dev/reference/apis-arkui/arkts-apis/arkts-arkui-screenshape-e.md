# ScreenShape

显示设备的屏幕形状枚举。

**起始版本：** 18

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## ROUND

```TypeScript
ROUND = 1
```

表示设备屏幕形状为圆形。

**起始版本：** 18

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## RECTANGLE

```TypeScript
RECTANGLE = 0
```

表示设备屏幕形状为矩形。

**起始版本：** 18

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
