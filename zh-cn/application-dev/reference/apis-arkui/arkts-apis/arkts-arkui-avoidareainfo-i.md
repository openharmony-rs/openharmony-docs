# AvoidAreaInfo

用于表示窗口避让区的信息。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## area

```TypeScript
area: window.AvoidArea
```

窗口内容避让区域。

**类型：** window.AvoidArea

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## type

```TypeScript
type: window.AvoidAreaType
```

窗口避让区类型。

**类型：** window.AvoidAreaType

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
