# MultiScreenMode（系统接口）

屏幕模式枚举。

**起始版本：** 13

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SCREEN_MIRROR

```TypeScript
SCREEN_MIRROR = 0
```

表示屏幕为镜像模式。

**起始版本：** 13

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## SCREEN_EXTEND

```TypeScript
SCREEN_EXTEND = 1
```

表示屏幕为扩展模式。

**起始版本：** 13

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
