# ThemeColorMode

```TypeScript
declare enum ThemeColorMode
```

设置颜色模式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SYSTEM

```TypeScript
SYSTEM = 0
```

跟随系统深浅色模式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
