# Size（系统接口）

表示截取图像的大小。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## height

```TypeScript
height: number
```

表示截取图像的高度，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## width

```TypeScript
width: number
```

表示截取图像的宽度，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
