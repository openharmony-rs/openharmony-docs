```TypeScript

```

创建文本时钟组件。

**起始版本：** 8

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为8.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |

| options | TextClockOptions | 否 |  |
