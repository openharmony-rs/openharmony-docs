# CommandPath

用于设置路径绘制的指令。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## commands

```TypeScript
commands: string
```

路径绘制的指令字符串。像素单位的转换方法请参考[像素单位](../../apis-ability-kit/arkts-apis/arkts-app-ability-common.md)。

单位：px

**类型：** string

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
