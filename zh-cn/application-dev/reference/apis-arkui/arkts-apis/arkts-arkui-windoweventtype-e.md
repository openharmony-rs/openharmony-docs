# WindowEventType

窗口生命周期。

**起始版本：** 10

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## WINDOW_SHOWN

```TypeScript
WINDOW_SHOWN = 1
```

切到前台。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
