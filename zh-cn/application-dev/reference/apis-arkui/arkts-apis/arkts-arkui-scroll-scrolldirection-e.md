# ScrollDirection

```TypeScript
declare enum ScrollDirection
```

滚动方向枚举。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Vertical

```TypeScript
Vertical
```

仅支持竖直方向滚动。

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Horizontal

```TypeScript
Horizontal
```

仅支持水平方向滚动。

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Free

```TypeScript
Free
```

支持水平和垂直方向滚动

**起始版本：** 7

**废弃版本：** 9

**替代接口：** [FREE](arkts-arkui-scrolldirection-e.md#FREE)

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## None

```TypeScript
None
```

不可滚动。

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FREE

```TypeScript
FREE
```

自由滚动。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
