# Popup

## Popup

```TypeScript
export declare function Popup(options: PopupOptions): void
```

**起始版本：** 11

**装饰器类型：** @Builder

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| options | PopupOptions | 是 | 定义Popup组件的类型。 |
