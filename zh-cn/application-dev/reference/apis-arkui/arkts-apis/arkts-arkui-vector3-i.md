# Vector3

用于表示包含x、y、z三个值的向量。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## x

```TypeScript
x: number
```

向量x轴方向的值。

取值范围：(-∞, +∞)

**类型：** number

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## y

```TypeScript
y: number
```

向量y轴方向的值。

取值范围：(-∞, +∞)

**类型：** number

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## z

```TypeScript
z: number
```

z轴方向的旋转角度。

取值范围：(-∞, +∞)

**类型：** number

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。
