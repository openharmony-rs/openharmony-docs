# Size

窗口大小，单位为px。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## height

```TypeScript
height: number
```

窗口高度，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## width

```TypeScript
width: number
```

窗口宽度，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
