# DarkMode（系统接口）

深色模式枚举。

| 名称 | 值 | 说明 |
| -- | -- | -- |
| ALWAYS_DARK | 0 | 系统始终为深色。 |
| ALWAYS_LIGHT | 1 | 系统始终为浅色。 |

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.UiAppearance

**系统接口：** 此接口为系统接口。

## ALWAYS_DARK

```TypeScript
ALWAYS_DARK = 0
```

Always display with dark mode.

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.UiAppearance

**系统接口：** 此接口为系统接口。

## ALWAYS_LIGHT

```TypeScript
ALWAYS_LIGHT = 1
```

Always display with light mode.

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.UiAppearance

**系统接口：** 此接口为系统接口。
