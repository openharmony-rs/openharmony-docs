# WindowLayoutMode（系统接口）

窗口布局模式枚举。

**起始版本：** 9

**废弃版本：** 26.0.0

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## WINDOW_LAYOUT_MODE_CASCADE

```TypeScript
WINDOW_LAYOUT_MODE_CASCADE = 0
```

表示使用层叠布局模式。层叠布局下，多个自由窗口层叠放置，以Z轴次序区分。

**起始版本：** 9

**废弃版本：** 26.0.0

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## WINDOW_LAYOUT_MODE_TILE

```TypeScript
WINDOW_LAYOUT_MODE_TILE = 1
```

表示使用平铺布局模式。平铺布局下，新打开的应用窗口出现在最右侧。

**起始版本：** 9

**废弃版本：** 26.0.0

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
