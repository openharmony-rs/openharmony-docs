# CustomKeyboardContinueFeature

自定义键盘接续特性的枚举。

**起始版本：** 23

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ENABLED

```TypeScript
ENABLED = 0
```

使能自定义键盘接续。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DISABLED

```TypeScript
DISABLED = 1
```

关闭自定义键盘接续。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
