# Degree

```TypeScript
declare type Degree = `${number}deg`
```

Defines the angle property with number in units of deg.

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
