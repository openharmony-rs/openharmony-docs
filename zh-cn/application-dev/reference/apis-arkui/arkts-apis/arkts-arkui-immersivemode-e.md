# ImmersiveMode

页面内弹窗蒙层显示区域模式。

**起始版本：** 15

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DEFAULT

```TypeScript
DEFAULT = 0
```

弹窗蒙层遵循父节点布局约束进行显示。

**起始版本：** 15

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## EXTEND

```TypeScript
EXTEND = 1
```

弹窗蒙层可扩展至覆盖状态栏和导航条。

**起始版本：** 15

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
