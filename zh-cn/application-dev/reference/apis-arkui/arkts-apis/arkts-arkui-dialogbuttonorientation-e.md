# DialogButtonOrientation

对话框中按钮的排列。

**起始版本：** 26.1.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## AUTO

```TypeScript
AUTO = 0
```

两个或两个以下的按钮水平排列，
且两个或两个以上的按钮垂直排列。

**起始版本：** 26.1.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.1.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

HORIZONTAL = 1

**元服务API：** 从API版本26.1.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
