# Progress

## Progress

```TypeScript

```

创建进度条组件。

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为7.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |

| options | ProgressOptions<Type> | 是 |  |
