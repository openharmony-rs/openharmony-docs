# ArcSliderTouchHandler

```TypeScript
declare type ArcSliderTouchHandler = (event: TouchEvent) => void
```

弧形Slider被触摸时，告知应用。

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| event | TouchEvent | 是 | 获得TouchEvent对象。 |
