# Rotation2D（系统接口）

描述二维空间中的旋转，可以通过旋转角度和旋转中心来定义。

**起始版本：** 23

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## angle

```TypeScript
angle: number
```

旋转角信息

**类型：** number

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
