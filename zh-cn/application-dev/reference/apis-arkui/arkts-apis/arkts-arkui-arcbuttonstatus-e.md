# ArcButtonStatus

定义ArcButton可设置的弧形按钮状态。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## NORMAL

```TypeScript
NORMAL = 0
```

正常状态。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## PRESSED

```TypeScript
PRESSED = 1
```

按压状态。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## DISABLED

```TypeScript
DISABLED = 2
```

禁用状态。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle
