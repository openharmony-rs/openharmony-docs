# PiPWindowInfo

画中画窗口信息。

**起始版本：** 15

**系统能力：** SystemCapability.Window.SessionManager

## size

```TypeScript
size: PiPWindowSize
```

画中画窗口大小。

**类型：** PiPWindowSize

**起始版本：** 15

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## windowId

```TypeScript
windowId: number
```

画中画窗口ID。

**类型：** number

**起始版本：** 15

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
