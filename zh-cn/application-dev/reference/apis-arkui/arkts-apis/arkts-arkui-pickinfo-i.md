# PickInfo

截取图像的信息。

**起始版本：** 12

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## pickRect

```TypeScript
pickRect: Rect
```

表示截取图像的区域。

**类型：** Rect

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## pixelMap

```TypeScript
pixelMap: image.PixelMap
```

表示截取的图像PixelMap对象。

**类型：** image.PixelMap

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
