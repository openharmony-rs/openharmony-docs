# LightEffectOptions

沉浸式材质的光感交互反馈配置。用于自定义反馈光感的颜色。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## color

```TypeScript
color?: ResourceColor
```

自定义交互反馈光感的颜色。

默认值：Color.White

**类型：** ResourceColor

**默认值：** Color.White

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
