# MediaQueryResult

用于执行媒体查询操作。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## matches

```TypeScript
readonly matches: boolean
```

是否符合匹配条件。true表示满足查询条件，false表示不满足查询条件。

**类型：** boolean

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## media

```TypeScript
readonly media: string
```

媒体事件的匹配条件。

**类型：** string

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
