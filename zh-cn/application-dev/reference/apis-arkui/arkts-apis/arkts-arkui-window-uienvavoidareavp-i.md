# UIEnvAvoidAreaVP

以vp为单位表示的窗口避让区域信息，在进行[沉浸式布局](../../../../windowmanager/window-terminology.md#沉浸式布局)适配时需关注。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## bottomRect

```TypeScript
bottomRect: RectInVP
```

中心位于窗口的两条对角线的底部的矩形区，单位为vp。

**类型：** RectInVP

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager
