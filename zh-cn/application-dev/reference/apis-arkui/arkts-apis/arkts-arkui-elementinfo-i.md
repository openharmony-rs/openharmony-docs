# ElementInfo

可被观察对象关联的组件信息，包含系统组件和自定义组件。

**起始版本：** 23

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## elementId

```TypeScript
elementId: number
```

组件的ID。

**类型：** number

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## elementName

```TypeScript
elementName: string
```

组件的名称。

**类型：** string

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
