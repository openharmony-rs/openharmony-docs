# @ohos.arkui.advanced.SelectionMenu

## 汇总

### 函数

| 名称 | 说明 |
| --- | --- |
| [SelectionMenu](arkts-arkui-arkui-advanced-selectionmenu-selectionmenu-f.md#selectionmenu-1) | 入参为空时，文本选择菜单组件SelectionMenu内容区大小及组件大小为零。表现例如，富文本组件[RichEditor](../arkts-components/arkts-arkui-richeditor.md)使用[bindSelectionMenu](RichEditorAttribute#bindSelectionMenu)接口绑定一个SelectionMenu的右键菜单，则右键富文本组件区域时无任何菜单弹出。 |
