# RotationChangeType

窗口旋转事件类型。

**起始版本：** 19

**系统能力：** SystemCapability.Window.SessionManager

## WINDOW_WILL_ROTATE

```TypeScript
WINDOW_WILL_ROTATE = 0
```

窗口即将旋转。

**起始版本：** 19

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## WINDOW_DID_ROTATE

```TypeScript
WINDOW_DID_ROTATE = 1
```

窗口旋转结束。

**起始版本：** 19

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
