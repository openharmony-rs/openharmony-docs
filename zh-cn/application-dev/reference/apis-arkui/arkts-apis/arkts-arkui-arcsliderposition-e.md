# ArcSliderPosition

配置弧形Slider的屏幕显示位置。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## LEFT

```TypeScript
LEFT = 0
```

弧形Slider的屏幕显示位置在左侧。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## RIGHT

```TypeScript
RIGHT = 1
```

弧形Slider的屏幕显示位置在右侧。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle
