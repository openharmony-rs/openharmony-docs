# ArcButtonStyleMode

定义ArcButton可设置弧形按钮样式模式。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## EMPHASIZED_LIGHT

```TypeScript
EMPHASIZED_LIGHT = 0
```

强调样式，亮色，表现为蓝色背景、白色文字。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## EMPHASIZED_DARK

```TypeScript
EMPHASIZED_DARK = 1
```

警告样式，暗色，表现为红色背景、白色文字。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## NORMAL_LIGHT

```TypeScript
NORMAL_LIGHT = 2
```

常规样式，亮色，表现为深蓝色背景、蓝色文字。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## NORMAL_DARK

```TypeScript
NORMAL_DARK = 3
```

常规样式，暗色，表现为深灰色背景、蓝色文字。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## CUSTOM

```TypeScript
CUSTOM = 4
```

自定义按钮颜色和字体颜色。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle
