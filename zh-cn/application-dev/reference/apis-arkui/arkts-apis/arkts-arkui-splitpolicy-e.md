# SplitPolicy

表示MultiNavigation中页面的类型。

**起始版本：** 14

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## HOME_PAGE

```TypeScript
HOME_PAGE = 0
```

主页页面类型。全屏模式显示。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DETAIL_PAGE

```TypeScript
DETAIL_PAGE = 1
```

详情页页面类型。分栏模式显示。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FULL_PAGE

```TypeScript
FULL_PAGE = 2
```

全屏页页面类型。全屏模式显示。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
