# ToastShowMode

设置Toast的显示模式，默认显示在应用内，支持显示在子窗。

**起始版本：** 11

## SYSTEM_TOP_MOST

```TypeScript
SYSTEM_TOP_MOST = 2
```

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。
