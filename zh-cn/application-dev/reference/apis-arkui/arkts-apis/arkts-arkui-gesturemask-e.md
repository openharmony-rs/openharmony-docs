# GestureMask

```TypeScript
declare enum GestureMask
```

定义是否屏蔽子组件手势。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Normal

```TypeScript
Normal = 0
```

不屏蔽子组件的手势，按照默认手势识别顺序进行识别。

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## IgnoreInternal

```TypeScript
IgnoreInternal = 1
```

Ignore internal gestures and recognize the current gesture first.

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
