## 导入模块

```TypeScript
import { uiMaterial } from '@kit.ArkUI';
```

## type

```TypeScript
type?: MaterialType
```

材质类型。

默认值：MaterialType.NONE

**默认值：** uiMaterial.MaterialType.NONE

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**卡片能力：** 从API版本23开始，该接口支持在ArkTS卡片中使用。
