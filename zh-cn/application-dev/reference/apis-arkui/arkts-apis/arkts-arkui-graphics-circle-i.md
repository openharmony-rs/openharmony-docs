# Circle

用于设置圆形的属性。

**起始版本：** 12

<!--Device-unnamed-export interface Circle--><!--Device-unnamed-export interface Circle-End-->

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## centerX

```TypeScript
centerX: number
```

圆心x轴的位置，单位为px。

**类型：** number

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
