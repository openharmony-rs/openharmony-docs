# PiPState

画中画生命周期状态枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Window.SessionManager

## ABOUT_TO_START

```TypeScript
ABOUT_TO_START = 1
```

表示画中画将要启动。

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## STARTED


**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。
