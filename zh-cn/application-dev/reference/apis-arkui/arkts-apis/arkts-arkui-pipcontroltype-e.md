# PiPControlType

控制面板控件类型枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Window.SessionManager

## VIDEO_PLAY_PAUSE

```TypeScript
VIDEO_PLAY_PAUSE = 0
```

播放/暂停直播控件组。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
