# DialogBaseAlignment

对话框的对齐方式。

**起始版本：** 26.1.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## TOP

```TypeScript
TOP = 0
```

垂直顶部对齐。

**起始版本：** 26.1.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.1.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
