# WindowSnapshotConfiguration

主窗口截图的配置项。

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager

## useCache

```TypeScript
useCache?: boolean
```

是否使用主窗口的已有截图。默认值为true。 true表示使用主窗口的已有截图，若主窗口无保存的截图，则使用主窗口的最新截图。false表示使用主窗口的最新截图。

**类型：** boolean

**起始版本：** 21

**系统能力：** SystemCapability.Window.SessionManager
