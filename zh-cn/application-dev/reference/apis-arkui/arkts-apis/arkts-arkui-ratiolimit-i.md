# RatioLimit

标准悬浮窗的宽高比限制范围。宽高比比值由窗口矩形区域的宽除以高获得。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Window.SessionManager

## maxRatio

```TypeScript
maxRatio: number
```

标准悬浮窗的宽高比最大值。

**类型：** number

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager

## minRatio

```TypeScript
minRatio: number
```

标准悬浮窗的宽高比最小值。

**类型：** number

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager
