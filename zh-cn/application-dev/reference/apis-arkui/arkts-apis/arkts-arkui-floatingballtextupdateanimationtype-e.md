# FloatingBallTextUpdateAnimationType

闪控球文本更新动画类型的枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Window.SessionManager

## ANIMATION_NONE

```TypeScript
ANIMATION_NONE = 0
```

无动画。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager

## ANIMATION_OPACITY

```TypeScript
ANIMATION_OPACITY = 1
```

淡入淡出动画。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager
