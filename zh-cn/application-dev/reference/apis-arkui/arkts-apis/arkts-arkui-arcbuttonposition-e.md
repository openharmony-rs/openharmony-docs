# ArcButtonPosition

定义ArcButton可设置的弧形按钮的类型。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## TOP_EDGE

```TypeScript
TOP_EDGE = 0
```

上弧形按钮，位于圆形屏幕上方。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

## BOTTOM_EDGE

```TypeScript
BOTTOM_EDGE = 1
```

底部弧形按钮，位于圆形屏幕底部。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle
