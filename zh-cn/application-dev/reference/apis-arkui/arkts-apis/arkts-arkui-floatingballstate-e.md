# FloatingBallState

闪控球生命周期状态的枚举。

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager

## STARTED

```TypeScript
STARTED = 1
```

表示闪控球启动。

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager

## STOPPED

```TypeScript
STOPPED = 2
```

表示闪控球停止。

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager
