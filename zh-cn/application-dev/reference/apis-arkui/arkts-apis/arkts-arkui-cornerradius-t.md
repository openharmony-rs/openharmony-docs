# CornerRadius

```TypeScript
export type CornerRadius = Corners<Vector2>
```

设置四个角的圆角x轴与y轴的半轴长。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
