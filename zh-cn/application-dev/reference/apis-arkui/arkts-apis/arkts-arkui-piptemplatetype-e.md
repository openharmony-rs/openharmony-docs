# PiPTemplateType

画中画模板类型枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Window.SessionManager

## VIDEO_PLAY

```TypeScript
VIDEO_PLAY = 0
```

表示将要切换为画中画播放的媒体类型是视频，系统依此加载视频播放模板，该模板默认存在播放/暂停控件。

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
