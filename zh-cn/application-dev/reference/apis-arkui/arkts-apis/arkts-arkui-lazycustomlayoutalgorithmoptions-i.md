# LazyCustomLayoutAlgorithmOptions

自定义懒加载布局算法的构造入参，设置布局算法的主轴方向。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## axis

```TypeScript
axis?: Axis
```

定义懒加载布局的主轴方向。

**类型：** Axis

**默认值：** Axis.Vertical

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
