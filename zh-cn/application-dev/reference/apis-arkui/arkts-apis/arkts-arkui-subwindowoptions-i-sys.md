# SubWindowOptions

子窗口创建参数。

**起始版本：** 11

**系统能力：** SystemCapability.Window.SessionManager

## isTopmost

```TypeScript
isTopmost?: boolean
```

子窗口是否启用置顶属性。true表示子窗口置顶，false表示子窗口不置顶。不设置，则默认为false。

**类型：** boolean

**起始版本：** 12

**系统能力：** SystemCapability.Window.SessionManager

**系统接口：** 此接口为系统接口。
