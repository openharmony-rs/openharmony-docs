# FadingEdgeOptions

Defines the fadingEdge options.

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

## fadingEdgeLength

```TypeScript
fadingEdgeLength?: LengthMetrics
```

The length of FadingEdge.

**类型：** LengthMetrics

**默认值：** 32vp

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务API中使用。
