# RoundedCorner

屏幕圆角定义。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## position

```TypeScript
readonly position: Position
```

圆角圆心的坐标点。

**类型：** Position

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## radius

```TypeScript
readonly radius: number
```

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
