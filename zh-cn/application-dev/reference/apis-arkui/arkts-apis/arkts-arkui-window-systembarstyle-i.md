## statusBarContentColor

```TypeScript
statusBarContentColor?: string
```

状态栏文字颜色。默认值：`'#E5FFFFFF'`。

**类型：** string

**起始版本：** 12

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
