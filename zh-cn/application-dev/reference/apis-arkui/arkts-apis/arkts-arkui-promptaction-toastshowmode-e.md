# ToastShowMode

设置Toast的显示模式，默认显示在应用内，支持显示在子窗。

**起始版本：** 11

```TypeScript

```

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
