# LayoutConstraint

描述组件的布局约束。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## maxSize

```TypeScript
maxSize: Size
```

最大尺寸。

**类型：** Size

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## minSize

minSize: Size

percentReference: Size
