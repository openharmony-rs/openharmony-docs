# clearInterval

## clearInterval

```TypeScript
export declare function clearInterval(intervalID?: number): void
```

Cancel the interval set by " setInterval()".

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
