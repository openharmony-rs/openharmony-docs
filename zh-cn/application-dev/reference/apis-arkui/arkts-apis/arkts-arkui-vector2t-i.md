# Vector2T

用于表示T类型的包含x和y两个值的向量。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## x

```TypeScript
x: T
```

向量x轴方向的值。

**类型：** T

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## y

```TypeScript
y: T
```

向量y轴方向的值。

**类型：** T

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
