# RRect（系统接口）

圆角矩形。

**起始版本：** 9

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## height

```TypeScript
height: number
```

动画目标窗口的高度大小。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## left

