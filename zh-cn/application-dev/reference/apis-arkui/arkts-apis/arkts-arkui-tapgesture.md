# TapGesture

## TapGesture

```TypeScript
TapGesture(tag: string)
```

设置手势的标志。

**起始版本：** 11

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为11.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tag | string | 是 |  |

## TapGesture

```TypeScript
TapGesture(types: Array<SourceTool>)
```

设置手势响应的输入类型。

**起始版本：** 14

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为14.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| types | Array<SourceTool> | 是 |  |
