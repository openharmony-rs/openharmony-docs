# ItemRestriction

```TypeScript
declare type ItemRestriction<T> = [T, T, T?, T?, T?]
```

保存按钮信息的元组类型。

> **说明：**  
>  
> 分段按钮组件仅支持2到5个按钮。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
