# DragStatus

拖拽开始和结束状态。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## STARTED

```TypeScript
STARTED = 0
```

拖拽已成功发起。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ENDED

```TypeScript
ENDED = 1
```

拖拽结束。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
