# TargetInfo

Defines the target info.

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## componentId

```TypeScript
componentId?: number
```

Unique ID that generated by framework. This ID used to constrain range of target.

**类型：** number

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## id

```TypeScript
id: string | number
```

ID of target node.

**类型：** string | number

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
