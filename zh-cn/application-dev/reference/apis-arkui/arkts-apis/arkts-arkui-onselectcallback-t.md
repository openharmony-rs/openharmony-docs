# OnSelectCallback

```TypeScript
declare type OnSelectCallback =  (index: number) => void
```

定义[onSelect](arkts-arkui-arcalphabetindexerattribute-c.md#onselect-1)中使用的回调类型。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Circle

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| index | number | 是 | 选中项序号。 |
