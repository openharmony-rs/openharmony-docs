# TextVerticalAlign

文本垂直对齐的方式。默认值BASELINE，沿基线对齐。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## BASELINE

```TypeScript
BASELINE = 0
```

对齐文本基线。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本20开始，该接口支持在原子化服务API中使用。
