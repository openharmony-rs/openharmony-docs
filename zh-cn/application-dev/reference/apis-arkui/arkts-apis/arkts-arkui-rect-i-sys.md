# Rect（系统接口）

矩形信息。

**起始版本：** 19

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## height

```TypeScript
height: number
```

矩形的高度，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 19

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## left

```TypeScript
left: number
```

矩形左上角顶点的X轴坐标，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 19

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## top

```TypeScript
top: number
```

矩形左上角顶点的Y轴坐标，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 19

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## width

```TypeScript
width: number
```

矩形的宽度，单位为px，该参数应为整数。

**类型：** number

**起始版本：** 19

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
