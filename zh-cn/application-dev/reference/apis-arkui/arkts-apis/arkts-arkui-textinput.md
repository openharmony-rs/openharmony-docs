# TextInput

## TextInput

```TypeScript
TextInput(value?: TextInputOptions)
```

定义单行文本输入组件构造函数。

**起始版本：** 7

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为7.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |

| value | TextInputOptions | 否 |  |
