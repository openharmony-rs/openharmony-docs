## centerX

```TypeScript
centerX?: number
```

变换中心点x轴坐标。

单位：px

默认值：组件中心点x轴坐标。

取值范围 (-∞, +∞)

**类型：** number

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

## x

```TypeScript
x?: number
```

x轴的缩放倍数。x>1时以x轴方向放大，0<x<1时以x轴方向缩小，x<0时沿x轴反向并缩放。

默认值：1

取值范围 (-∞, +∞)

**类型：** number

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
