# VideoLiveControlGroup

视频直播控件组枚举。仅当[PiPTemplateType](arkts-arkui-piptemplatetype-e.md) 为VIDEO_LIVE时使用。

**起始版本：** 12

**系统能力：** SystemCapability.Window.SessionManager

## VIDEO_PLAY_PAUSE

```TypeScript
VIDEO_PLAY_PAUSE = 401
```

播放/暂停直播控件组。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## MUTE_SWITCH

```TypeScript
MUTE_SWITCH = 402
```

静音控件组。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
