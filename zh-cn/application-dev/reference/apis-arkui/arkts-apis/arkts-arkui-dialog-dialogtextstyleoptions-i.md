## wordBreak

```TypeScript
wordBreak?: WordBreak
```

**类型：** WordBreak

**起始版本：** 26.1.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本26.1.0开始，该接口支持在原子化服务API中使用。
