# RectType

窗口矩形区域坐标系类型。

**起始版本：** 19

**系统能力：** SystemCapability.Window.SessionManager

## RELATIVE_TO_SCREEN

```TypeScript
RELATIVE_TO_SCREEN  = 0
```

窗口矩形区域相对于屏幕坐标系。

**起始版本：** 19

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager

## RELATIVE_TO_PARENT_WINDOW

```TypeScript
RELATIVE_TO_PARENT_WINDOW = 1
```

窗口矩形区域相对于父窗口坐标系。

**起始版本：** 19

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
