# UIEnvWindowAvoidAreaInfoPX

窗口不同类型避让区域信息组成的[环境变量](../../../../ui/arkts-env-system-property.md)数据类型，每种类型避让区域单位为px。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## cutout

```TypeScript
cutout: AvoidArea
```

表示[AvoidAreaType](arkts-arkui-avoidareatype-e.md)为TYPE_CUTOUT类型的避让区域，单位为px。

**类型：** AvoidArea

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## keyboard

```TypeScript
keyboard: AvoidArea
```

表示[AvoidAreaType](arkts-arkui-avoidareatype-e.md)为TYPE_KEYBOARD类型的避让区域，单位为px。

**类型：** AvoidArea

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager
