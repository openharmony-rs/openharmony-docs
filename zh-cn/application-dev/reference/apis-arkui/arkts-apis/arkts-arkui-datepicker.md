# DatePicker

定义日期滑动选择组件实例.


## DatePicker

```TypeScript
DatePicker(options?: DatePickerOptions)
```

根据指定日期范围创建日期选择器。

**起始版本：** 8

**ArkTS模式:** 仅支持ArkTS-Dyn，ArkTS-Dyn起始版本为8.

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| options | DatePickerOptions | 否 |  |
