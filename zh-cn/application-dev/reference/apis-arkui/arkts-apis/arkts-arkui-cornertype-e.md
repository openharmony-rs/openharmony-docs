# CornerType

屏幕圆角类型枚举。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## TOP_LEFT

```TypeScript
TOP_LEFT = 0
```

屏幕左上方的圆角。

**起始版本：** 23

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Window.SessionManager
