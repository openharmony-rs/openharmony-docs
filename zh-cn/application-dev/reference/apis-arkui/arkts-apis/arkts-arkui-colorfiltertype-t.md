# ColorFilterType

```TypeScript
declare type ColorFilterType = ColorFilter | DrawingColorFilter
```

图片颜色滤镜设置项类型。

**起始版本：** 15

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本15开始，该接口支持在原子化服务API中使用。
