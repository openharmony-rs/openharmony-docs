## 106206 节点已被接纳为附属节点

**错误信息**

The node has already been adopted.

**错误描述**

该节点已经被接纳为附属节点，不能再作为子节点挂载。<br>
接纳：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

该节点已经被接纳为附属节点，不能再作为子节点挂载。

**处理步骤**

先将该节点取消接纳，然后再进行子节点挂载操作。

## 106207 被接纳的附属节点已有父节点

**错误信息**

This node already has a parent node.

**错误描述**

该节点已经有父节点，不能再被其他节点接纳。<br>
接纳：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

该节点已经有父节点，不能再被其他节点接纳。

**处理步骤**

先将当前节点从其父节点上移除，然后再进行接纳操作。

## 106208 节点无法被接纳为附属节点

**错误信息**

The node cannot be adopted.

**错误描述**

该节点无法被接纳为附属节点。<br>
接纳：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

1. 节点为空指针。
2. 节点不符合被接纳为附属节点的标准，只有下面几种节点可被接纳：
    - TS侧的命令式节点。
    - CAPI侧的命令式节点。
    - BuilderNode的根节点。

**处理步骤**

传入符合标准的节点进行接纳操作。

## 106209 节点无法接纳其他节点

**错误信息**

The node cannot adopt children.

**错误描述**

该节点无法接纳其它附属节点。<br>
接纳：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

1. 节点为空指针。
2. 节点不符合接纳其他节点的标准，只有下面几种节点可接纳其他节点：
    - TS侧的命令式节点。
    - CAPI侧的命令式节点。

**处理步骤**

使用符合标准的节点进行接纳操作。

## 106210 节点不是被目标节点接纳的附属节点

**错误信息**

This child is not adopted by the parent node.

**错误描述**

子节点并不是被父节点接纳的节点。<br>
接纳：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

1. 子节点并未被接纳。
2. 子节点被接纳了，但不是被当前节点接纳的。

**处理步骤**

如果节点不是被目标节点接纳的附属节点，则无需执行移除操作。
