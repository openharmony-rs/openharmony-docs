## 106206 节点已被收养

**错误信息**

The node has already been adopted.

**错误描述**

该节点已经被收养，不能再作为子节点挂载。<br>
收养：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

该节点已经被收养，不能再作为子节点挂载。

**处理步骤**

先将该节点取消收养，然后再进行子节点挂载操作。

## 106207 被收养节点已有父节点

**错误信息**

This node already has a parent node.

**错误描述**

该节点已经有父节点，不能再被其他节点收养。<br>
收养：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

该节点已经有父节点，不能再被其他节点收养。

**处理步骤**

先将当前节点从其父节点上移除，然后再进行收养操作。

## 106208 节点无法被收养

**错误信息**

The node cannot be adopted.

**错误描述**

该节点无法被收养。<br>
收养：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

1. 节点为空指针。
2. 节点不符合被收养节点的标准，只有下面几种节点可被收养：
    - TS侧的命令式节点。
    - CAPI侧的命令式节点。
    - BuilderNode的根节点。

**处理步骤**

传入符合标准的节点进行收养操作。

## 106209 节点无法收养其他节点

**错误信息**

The node cannot adopt children.

**错误描述**

该节点无法收养其他节点。<br>
收养：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

1. 节点为空指针。
2. 节点不符合收养其他节点的标准，只有下面几种节点可收养其他节点：
    - TS侧的命令式节点。
    - CAPI侧的命令式节点。

**处理步骤**

使用符合标准的节点进行收养操作。

## 106210 子节点不是被父节点收养的节点

**错误信息**

This child is not adopted by the parent node.

**错误描述**

子节点并不是被父节点收养的节点。<br>
收养：类似于父子关系，但父节点并不会真正将子节点添加为正常的节点，而是仅允许它像子节点一样接收生命周期回调，但不会像正常子节点一样接收父节点的事件、响应事件等。

**可能原因**

1. 子节点并未被收养。
2. 子节点被收养了，但不是被当前节点收养的。

**处理步骤**

如果子节点不是被父节点收养的节点，则无需执行移除操作。
