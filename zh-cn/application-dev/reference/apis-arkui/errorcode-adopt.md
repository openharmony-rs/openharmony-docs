## 106206 节点已被接纳为附属节点

**错误信息**

The node has already been adopted.

**错误描述**

该节点已经被接纳为附属节点，不能再作为子节点挂载。<br>

**可能原因**

该节点已经被接纳为附属节点，不能再作为子节点挂载。

**处理步骤**

先将该节点取消接纳，然后再进行子节点挂载操作。

## 106207 被接纳的附属节点已有父节点

**错误信息**

This node already has a parent node.

**错误描述**

该节点已经有父节点，不能再被其他节点接纳。<br>

**可能原因**

该节点已经有父节点，不能再被其他节点接纳。

**处理步骤**

先将当前节点从其父节点上移除，然后再进行接纳操作。

## 106208 节点无法被接纳为附属节点

**错误信息**

The node cannot be adopted.

**错误描述**

该节点无法被接纳为附属节点。<br>

**可能原因**

1. 节点为空指针。
2. 节点不支持被接纳，只有下面几种节点可被接纳：
    - 使用ArkTS语言创建的命令式节点。
    - 使用C语言创建的命令式节点。
    - BuilderNode的根节点。

**处理步骤**

确保节点非空，且支持被接纳，再进行接纳操作。

## 106209 节点无法接纳其他节点

**错误信息**

The node cannot adopt children.

**错误描述**

该节点无法接纳其它附属节点。<br>

**可能原因**

1. 节点为空指针。
2. 节点不支持被接纳，只有下面几种节点可被接纳：
    - 使用ArkTS语言创建的命令式节点。
    - 使用C语言创建的命令式节点。

**处理步骤**

确保节点非空，且支持被接纳，再进行接纳操作。

## 106210 节点不是被目标节点接纳的附属节点

**错误信息**

This child is not adopted by the parent node.

**错误描述**

子节点并不是被父节点接纳的节点。<br>

**可能原因**

1. 子节点并未被接纳。
2. 子节点被接纳了，但不是被当前节点接纳的。

**处理步骤**

如果节点未被目标节点接纳为附属节点，无需移除。
