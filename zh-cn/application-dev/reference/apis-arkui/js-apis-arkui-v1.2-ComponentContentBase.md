# ComponentContentBase（ArkTS1.2）

ComponentContentBase表示[ComponentContent（ArkTS1.2）](./js-apis-arkui-v1.2-ComponentContent.md)的基础类型。

> **说明：**
>
> 从API version 20开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。
>
> 本模块仅适用于ArkTS1.2。

## 导入模块

```ts
import { ComponentContentBase } from '@ohos.arkui.node';
```
