# ArkUI_Rect

## 概述

定义遮罩屏蔽区域的范围结构体。

**起始版本：** 12

**相关模块：** [ArkUI_NativeModule](capi-arkui-nativemodule.md)

**所在头文件：** [native_type.h](capi-native-type-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| float x | 区域在x轴的位置。 |
| float y | 区域在y轴的位置。 |
| float width | 区域宽度。 |
| float height | 区域高度。 |
