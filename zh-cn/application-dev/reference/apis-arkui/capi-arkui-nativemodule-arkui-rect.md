## 概述

**起始版本：** 12

**相关模块：** [ArkUI_NativeModule](capi-arkui-nativemodule.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| float x | 区域在x轴的位置，单位：vp。 |
| float y | 区域在y轴的位置，单位：vp。 |
| float width | 区域宽度，单位：vp。 |
| float height | 区域高度，单位：vp。 |
