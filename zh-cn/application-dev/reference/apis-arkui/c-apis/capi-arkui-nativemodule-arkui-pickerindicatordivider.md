# ArkUI_PickerIndicatorDivider

```c
typedef struct ArkUI_PickerIndicatorDivider {...} ArkUI_PickerIndicatorDivider
```

## 概述

Style parameters of divider indicator.

**起始版本：** 23

**相关模块：** [ArkUI_NativeModule](capi-arkui-nativemodule.md)

**所在头文件：** [native_type.h](capi-native-type-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| float strokeWidth | stroke width |
| uint32_t dividerColor | divider color, 0xARGB format for example <b>0xFF1122FF</b> |
| float startMargin | the distance between the divider and the beginning of the side of the picker (unit: vp). |
| float endMargin | the distance between the divider and the end of the side of the picker (unit: vp). |
