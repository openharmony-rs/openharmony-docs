# OH_WindowManager_DensityInfo

```c
typedef struct OH_WindowManager_DensityInfo OH_WindowManager_DensityInfo
```

## 概述

Window density information, including the system display size scaling factor, system default display sizescaling factor, and custom display size scaling factor of the screen where the window is located.
