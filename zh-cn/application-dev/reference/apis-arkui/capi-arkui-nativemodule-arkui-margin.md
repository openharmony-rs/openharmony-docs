# ArkUI_Margin

## 概述

外边距属性，用于描述组件的外边距属性。

**起始版本：** 12

**相关模块：** [ArkUI_NativeModule](capi-arkui-nativemodule.md)

**所在头文件：** [native_type.h](capi-native-type-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| float top | 上外边距，单位为vp。 |
| float right | 右外边距，单位为vp。 |
| float bottom | 下外边距，单位为vp。 |
| float left | 左外边距，单位为vp。 |
