# AutoCapitalizationMode

自动大小写模式类型，只提供接口能力，具体实现以输入法应用为主。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## NONE

```TypeScript
NONE = 0
```

默认状态无效。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## WORDS

```TypeScript
WORDS = 1
```

按单词自动大小写，即输入单词的首个字符大写，其他字符小写。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
