# UndoStyle

撤销还原是否保留原样式选项。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CLEAR_STYLE

```TypeScript
CLEAR_STYLE = 0
```

撤销还原内容不保留原样式。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## KEEP_STYLE

```TypeScript
KEEP_STYLE = 1
```

撤销还原内容保留原样式。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
