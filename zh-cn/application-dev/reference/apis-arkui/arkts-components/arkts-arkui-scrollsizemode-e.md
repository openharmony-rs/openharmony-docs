# ScrollSizeMode

半模态面板上下滑动时的内容更新方式。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FOLLOW_DETENT

```TypeScript
FOLLOW_DETENT = 0
```

设置半模态面板跟手滑动结束后更新内容显示区域。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CONTINUOUS

```TypeScript
CONTINUOUS = 1
```

设置半模态面板在滑动过程中持续更新内容显示区域。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
