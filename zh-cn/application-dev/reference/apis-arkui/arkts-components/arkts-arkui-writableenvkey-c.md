# WritableEnvKey

定义系统环境键。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DIRECTION

```TypeScript
static readonly DIRECTION: WritableSystemEnvKey<Direction>
```

定义系统环境键方向。

**类型：** WritableSystemEnvKey<Direction>

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
