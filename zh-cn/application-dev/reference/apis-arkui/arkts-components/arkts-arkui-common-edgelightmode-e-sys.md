## EDGELIGHT_AUTO

```TypeScript
EDGELIGHT_AUTO = 0
```

自适应边缘光效动画。

根据设备运算能力，在低算力设备上关闭，在中高算力设备上开启。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。
