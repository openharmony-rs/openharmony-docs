# ColorSpace

定义了颜色空间的类型，用于指定颜色显示的模式。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SRGB

```TypeScript
SRGB = 0
```

SRGB颜色空间，适用于大多数显示设备。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DISPLAY_P3

```TypeScript
DISPLAY_P3 = 1
```

Display-P3颜色空间，具有更广的色域，适用于高端显示设备。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
