# ThemeColorMode

设置颜色模式。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SYSTEM

```TypeScript
SYSTEM = 0
```

跟随系统深浅色模式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## LIGHT

```TypeScript
LIGHT = 1
```

固定使用浅色模式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DARK

```TypeScript
DARK = 2
```

固定使用深色模式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
