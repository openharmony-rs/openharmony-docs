## closeSwipeAction

```TypeScript
closeSwipeAction?: boolean
```

系统返回键生效时是否收起ListItem的划出组件。

**类型：** boolean

**默认值：** true

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本26.0.0开始，该接口支持在原子化服务API中使用。
