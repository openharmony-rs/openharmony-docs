# DynamicRangeMode

期望展示的图像动态范围。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## HIGH

```TypeScript
HIGH = 0
```

不受限动态范围，最大限度进行图片提亮。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CONSTRAINT

```TypeScript
CONSTRAINT = 1
```

受限动态范围，受限进行图片提亮。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## STANDARD

```TypeScript
STANDARD = 2
```

标准动态范围，不进行图片提亮。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
