# SelectStatus

多选框群组的选中状态。

**起始版本：** 8

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## All

```TypeScript
All
```

群组多选择框全部选择。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Part

```TypeScript
Part
```

群组多选择框部分选择。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## None

```TypeScript
None
```

群组多选择框全部没有选择。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。
