# RichEditorSpanPosition

Span位置信息。

**起始版本：** 10

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## spanIndex

```TypeScript
spanIndex: number
```

Span索引值。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## spanRange

```TypeScript
spanRange: [number, number]
```

Span内容在RichEditor内的起始和结束位置。

**类型：** [number, number]

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
