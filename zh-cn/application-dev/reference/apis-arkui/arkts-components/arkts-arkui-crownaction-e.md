# CrownAction

旋转表冠动作。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## BEGIN

```TypeScript
BEGIN = 0
```

表冠开始转动。

**起始版本：** 18

**废弃版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## UPDATE

```TypeScript
UPDATE = 1
```

表冠转动中。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## END

```TypeScript
END = 2
```

表冠停止转动。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
