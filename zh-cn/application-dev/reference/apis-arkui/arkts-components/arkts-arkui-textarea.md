# TextArea

多行文本输入框组件，当输入的文本内容超过组件宽度时会自动换行显示。

高度未设置时，组件无默认高度，自适应内容高度。宽度未设置时，默认撑满最大宽度。

> **说明：**

## 子组件

无

```TypeScript
TextArea(value?: TextAreaOptions)
```

**起始版本：** 7
