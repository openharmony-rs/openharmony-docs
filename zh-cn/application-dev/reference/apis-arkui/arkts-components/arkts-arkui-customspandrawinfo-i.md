# CustomSpanDrawInfo

定义自定义绘制Span的绘制信息接口。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## baseline

```TypeScript
baseline: number
```

自定义绘制Span的所在行的基线偏移量。

单位：[px](../../../../reference/apis-arkui/arkui-ts/ts-pixel-units.md#基本像素单位)

**类型：** number

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## lineBottom

```TypeScript
lineBottom: number
```

自定义绘制Span相对于Text组件的下边距。
