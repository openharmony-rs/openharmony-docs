# LeadingMarginPlaceholder

前导边距占位符，用于表示文本段落左侧与组件边缘之间的距离。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## pixelMap

```TypeScript
pixelMap: PixelMap
```

图片内容。

**类型：** PixelMap

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## size

```TypeScript
size: [Dimension, Dimension]
```

图片大小，不支持设置百分比。

**类型：** [Dimension, Dimension]

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
