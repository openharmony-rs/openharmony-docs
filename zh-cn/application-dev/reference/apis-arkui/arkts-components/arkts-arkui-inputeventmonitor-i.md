# InputEventMonitor

输入事件监听器标识对象。

此对象由系统创建并返回，作为监听器的唯一标识。
> **说明：**  
>  
> - 对象为空对象，不包含任何可访问的成员。  
>  
> - 开发者无法主动构造此对象，只能通过[addLocalInputEventMonitor](../arkts-apis/arkts-arkui-arkui-uicontext-uicontext-c.md#addlocalinputeventmonitor)接口注册获取。  
>  
> - 用于后续解除注册时验证身份。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本26.0.0开始，该接口支持在原子化服务API中使用。
