# SpatialPosition（系统接口）

三维空间中的空间角位置。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## leftBottom

```TypeScript
leftBottom: DepthVector3
```

三维空间中的左下角角位置。

**类型：** DepthVector3

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
