# Vector2T

```TypeScript
declare type Vector2T<T> = import('../api/arkui/Graphics').Vector2T<T>
```

定义Vector2T类型。其中Vector2T类型包含x和y两个属性值。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
