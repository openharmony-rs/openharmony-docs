# BlurStyleActivePolicy

定义背景模糊激活策略。

**起始版本：** 14

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FOLLOWS_WINDOW_ACTIVE_STATE

```TypeScript
FOLLOWS_WINDOW_ACTIVE_STATE = 0
```

模糊效果跟随窗口焦点状态变化，非焦点不模糊，焦点模糊。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ALWAYS_ACTIVE

```TypeScript
ALWAYS_ACTIVE = 1
```

一直有模糊效果。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ALWAYS_INACTIVE

```TypeScript
ALWAYS_INACTIVE = 2
```

一直无模糊效果。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
