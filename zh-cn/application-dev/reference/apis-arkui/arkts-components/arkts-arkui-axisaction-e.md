# AxisAction

定义轴事件的轴动作类型。

**起始版本：** 17

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## NONE

```TypeScript
NONE = 0
```

无轴事件。

**起始版本：** 17

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本17开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
