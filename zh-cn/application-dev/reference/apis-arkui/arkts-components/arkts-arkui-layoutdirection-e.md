# LayoutDirection

系统的布局方向类型。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## LTR

```TypeScript
LTR
```

从左向右布局。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## RTL

```TypeScript
RTL
```

从右向左布局。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Auto

```TypeScript
Auto
```

自动布局，跟随系统。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
