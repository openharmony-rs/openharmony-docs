# PickerIndicatorType

设置选中项指示器的类型。

**起始版本：** 22

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## BACKGROUND

```TypeScript
BACKGROUND = 0
```

通过给选中项添加背景，标识选中项。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DIVIDER

```TypeScript
DIVIDER = 1
```

通过在选中项的上下边缘添加分割线，标识选中项。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
