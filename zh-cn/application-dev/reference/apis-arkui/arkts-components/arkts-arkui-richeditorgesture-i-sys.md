## onDoubleClick

```TypeScript
onDoubleClick?: Callback<GestureEvent>
```

双击事件回调函数，在用户双击操作完成时触发。回调参数为[GestureEvent](docroot://reference/apis-arkui/arkui-ts/ts-gesture-common.md#gestureevent对象说明)对象，包含手势事件信息。

**类型：** Callback&lt;GestureEvent&gt;

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。
