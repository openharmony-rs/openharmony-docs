# BreakpointsReference

设置栅格容器组件的断点参照物。

**起始版本：** 9

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## WindowSize

```TypeScript
WindowSize = 0
```

以窗口为参照。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ComponentSize

```TypeScript
ComponentSize = 1
```

以容器为参照。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
