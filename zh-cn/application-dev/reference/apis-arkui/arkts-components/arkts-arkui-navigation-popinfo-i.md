# PopInfo

**起始版本：** 11

## info

```TypeScript
info: NavPathInfo
```

页面触发返回时的当前页面信息，系统自动获取填入，无需开发者传入。

**类型：** NavPathInfo

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
