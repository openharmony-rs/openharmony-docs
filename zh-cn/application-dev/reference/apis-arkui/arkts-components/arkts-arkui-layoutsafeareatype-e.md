# LayoutSafeAreaType

扩展布局安全区域的枚举类型。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SYSTEM

```TypeScript
SYSTEM = 0
```

设置后，组件的布局范围可扩展至组件级安全区([safeAreaPadding](arkts-arkui-commonmethod-c.md#safeareapadding-1))和页面级安全区（状态栏、导航栏、挖孔区）。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
