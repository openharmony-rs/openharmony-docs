# DragAnimationType（系统接口）

拖拽动画类型。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## DEFAULT

```TypeScript
DEFAULT = 0
```

使用默认拖拽动画。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## FOLLOW_HAND_MORPH

```TypeScript
FOLLOW_HAND_MORPH = 1
```

使用跟手变形拖拽动画。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
