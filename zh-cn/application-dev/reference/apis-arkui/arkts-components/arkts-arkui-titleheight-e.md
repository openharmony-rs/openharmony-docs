# TitleHeight

Title height.

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## MainOnly

```TypeScript
MainOnly
```

Title height when only main title is available.

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## MainWithSub

```TypeScript
MainWithSub
```

Title height when main title and subtitle are both available.

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
