# GutterOption

栅格布局间距类型，用于描述栅格子组件不同方向的间距。

**起始版本：** 9

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## x

```TypeScript
x?: Length | GridRowSizeOption
```

栅格子组件水平方向间距。

**类型：** Length | GridRowSizeOption

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。
