# SheetDismiss

控制半模态的关闭。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## dismiss

```TypeScript
dismiss: () => void
```

半模态面板关闭回调函数。开发者需要退出时调用，不需要退出时无需调用。

**类型：** () => void

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
