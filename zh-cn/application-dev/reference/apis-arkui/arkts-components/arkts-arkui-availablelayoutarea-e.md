# AvailableLayoutArea

预览图宽高设置为百分比时的参考可布局区域大小。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SAFE_AREA

```TypeScript
SAFE_AREA = 0
```

参考可布局区域大小为窗口大小减去上下左右安全边距。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
