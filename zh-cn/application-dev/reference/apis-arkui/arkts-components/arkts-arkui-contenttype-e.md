# ContentType

自动填充类型。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## USER_NAME

```TypeScript
USER_NAME = 0
```

【用户名】在已启用密码保险箱的情况下，支持用户名的自动保存和自动填充。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
