# TextLayoutOptions

文本布局选项。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## constraintWidth

```TypeScript
constraintWidth?: LengthMetrics
```

设置被计算文本布局宽度。若不设置则宽度为单行布局所占最大宽度值。

**类型：** LengthMetrics

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
