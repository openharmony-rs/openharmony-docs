# VoiceButtonOptions（系统接口）

语音按钮选项。

**起始版本：** 23

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## enabled

```TypeScript
enabled?: boolean
```

输入框启用或禁用语音按钮。

true表示启用语音按钮，false表示禁用语音按钮。

默认值：false

**类型：** boolean

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
