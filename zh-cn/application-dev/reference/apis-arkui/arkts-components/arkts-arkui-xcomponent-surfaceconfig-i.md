## isOpaque

```TypeScript
isOpaque?: boolean
```

是否需要将XComponent所持有的surface视为不透明，即使该surface具有半透明像素。true表示需要视为不透明，false表示其他情况。

**类型：** boolean

**默认值：** false

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
