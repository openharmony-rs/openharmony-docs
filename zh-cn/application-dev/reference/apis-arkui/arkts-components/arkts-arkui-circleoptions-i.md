# CircleOptions

用于描述Circle组件绘制属性。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## height

```TypeScript
height?: Length
```

高度，取值范围≥0。

默认值：0

默认单位：vp

异常值undefined、null、NaN和Infinity按照默认值处理。

**类型：** Length

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## width

```TypeScript
width?: Length
```

默认值：0

默认单位：vp

异常值undefined、null、NaN和Infinity按照默认值处理。

**类型：** Length

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
