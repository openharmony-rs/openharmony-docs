# FadingEdgeOptions

Defines the fadingEdge options.

**起始版本：** 14

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## fadingEdgeLength

```TypeScript
fadingEdgeLength?: LengthMetrics
```

The length of FadingEdge.

**类型：** LengthMetrics

**默认值：** 32vp

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
