# NavigationMenuOptions

页面右上角菜单选项。

**起始版本：** 19

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## moreButtonOptions

```TypeScript
moreButtonOptions?: MoreButtonOptions
```

工具栏更多图标的菜单选项。

**类型：** MoreButtonOptions

**起始版本：** 19

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
