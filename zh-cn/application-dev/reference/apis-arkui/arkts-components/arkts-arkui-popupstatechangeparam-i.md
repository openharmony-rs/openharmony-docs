# PopupStateChangeParam

气泡的显示状态。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## isVisible

```TypeScript
isVisible: boolean
```

气泡的显示状态。返回true时，表示气泡从关闭到打开，返回false时，表示气泡从打开到关闭。

**类型：** boolean

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
