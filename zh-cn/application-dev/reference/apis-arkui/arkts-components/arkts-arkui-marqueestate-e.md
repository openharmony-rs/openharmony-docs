# MarqueeState

Marquee状态回调的返回值。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## START

```TypeScript
START = 0
```

跑马灯滚动开始。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## BOUNCE

```TypeScript
BOUNCE = 1
```

完成一次跑马灯滚动，如果循环次数不是1，将会多次返回。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FINISH

```TypeScript
FINISH = 2
```

跑马灯全部循环次数完成。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
