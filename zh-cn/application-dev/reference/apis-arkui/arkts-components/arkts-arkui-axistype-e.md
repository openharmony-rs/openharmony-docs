# AxisType

定义轴事件的轴类型。

**起始版本：** 22

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## VERTICAL_AXIS

```TypeScript
VERTICAL_AXIS = 0
```

垂直滚动轴。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
