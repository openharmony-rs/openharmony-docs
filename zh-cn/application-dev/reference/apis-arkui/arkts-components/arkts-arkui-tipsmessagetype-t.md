# TipsMessageType

```TypeScript
declare type TipsMessageType = ResourceStr | StyledString
```

悬浮气泡弹窗信息。

**起始版本：** 19

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

| 类型 | 说明 |
| --- | --- |
| ResourceStr | 字符串类型，用于描述字符串入参可以使用的类型。 |
| StyledString | 属性字符串。 |
