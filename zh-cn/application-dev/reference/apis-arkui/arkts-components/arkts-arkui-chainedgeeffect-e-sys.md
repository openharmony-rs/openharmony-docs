# ChainEdgeEffect（系统接口）

设置链式动效边缘效果。

**起始版本：** 10

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## DEFAULT

```TypeScript
DEFAULT
```

Default edge effect. Compress the space in the drag direction
and stretch the space in the opposite drag direction.

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## STRETCH

```TypeScript
STRETCH
```

Stretch all space.

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
