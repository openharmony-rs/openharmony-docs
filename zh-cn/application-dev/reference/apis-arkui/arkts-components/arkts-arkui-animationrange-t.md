# AnimationRange

```TypeScript
declare type AnimationRange<T> = [from: T, to: T]
```

动画开始和结束时相对预览原图缩放比例。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
