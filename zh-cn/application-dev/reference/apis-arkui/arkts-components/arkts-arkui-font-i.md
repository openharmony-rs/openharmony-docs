# Font

Defines the font used for text.

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## family

```TypeScript
family?: string | Resource
```

font family.

**类型：** string | Resource

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
