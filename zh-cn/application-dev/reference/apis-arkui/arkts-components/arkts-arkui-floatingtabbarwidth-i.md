# FloatingTabBarWidth

提供了一个接口，用于设置不同断点处的tab宽度的浮动栏宽度。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## largeBarWidth

```TypeScript
largeBarWidth?: Length
```

大型设备的条形宽度。不能以百分比形式设置。

**类型：** Length

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
