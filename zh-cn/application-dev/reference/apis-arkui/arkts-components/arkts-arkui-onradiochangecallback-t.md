# OnRadioChangeCallback

```TypeScript
declare type OnRadioChangeCallback = (isChecked: boolean) => void
```

单选框选中状态改变时触发的回调函数类型定义。

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| isChecked | boolean | 是 |  |
