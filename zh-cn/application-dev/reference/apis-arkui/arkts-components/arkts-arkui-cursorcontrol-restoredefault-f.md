# restoreDefault

<a id="restoredefault"></a>
## restoreDefault

```TypeScript
function restoreDefault(): void
```

方法语句中可使用的全局接口，调用此接口可将鼠标光标恢复成默认箭头样式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
