## distortion

```TypeScript
distortion?: DistortionParam
```

空间扭曲形变参数。通过指定四个角的位置关系和四条边的桶形变程度产生空间扭曲效果。

**类型：** DistortionParam

**模型约束：** 此接口仅可在Stage模型下使用。
