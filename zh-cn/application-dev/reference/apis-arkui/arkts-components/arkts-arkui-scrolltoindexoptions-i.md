# ScrollToIndexOptions

滚动到指定索引的参数。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## extraOffset

```TypeScript
extraOffset?: LengthMetrics
```

滚动到指定索引的额外偏移量。

**类型：** LengthMetrics

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
