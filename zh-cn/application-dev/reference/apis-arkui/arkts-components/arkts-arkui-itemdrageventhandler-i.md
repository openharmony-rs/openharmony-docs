# ItemDragEventHandler

定义拖拽事件

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## onDragStart

```TypeScript
onDragStart?: Callback<number>
```

当项目被拖动时，会触发此回调。

**类型：** Callback<number>

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
