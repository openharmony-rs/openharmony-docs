# MaxLinesMode

TextArea组件在文本超长时显示效果。默认值为CLIP，按最大行截断显示。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CLIP

```TypeScript
CLIP = 0
```

文本超长时按最大行截断显示。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SCROLL

```TypeScript
SCROLL = 1
```

文本超长时可滚动显示。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
