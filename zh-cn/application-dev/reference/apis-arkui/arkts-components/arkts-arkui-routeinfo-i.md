# RouteInfo

路由信息。

**起始版本：** 10

**废弃版本：** 13

**替代接口：** NavPathStack

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## name

```TypeScript
name: string
```

点击NavRouter跳转到的NavDestination页面的名称。

**类型：** string

**起始版本：** 10

**废弃版本：** 13

**替代接口：** name

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## param

```TypeScript
param?: unknown
```

点击NavRouter跳转到NavDestination页面时，传递的参数。

**类型：** unknown

**起始版本：** 10

**废弃版本：** 13

**替代接口：** param

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
