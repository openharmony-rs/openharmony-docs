# AvoidanceMode

下拉菜单避让模式的枚举选项。

**起始版本：** 19

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## COVER_TARGET

```TypeScript
COVER_TARGET
```

目标组件下方无足够空间时，覆盖目标组件。

**起始版本：** 19

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## AVOID_AROUND_TARGET

```TypeScript
AVOID_AROUND_TARGET
```

目标组件四周无足够空间时，在最大空间处压缩显示（可滚动）。

**起始版本：** 19

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
