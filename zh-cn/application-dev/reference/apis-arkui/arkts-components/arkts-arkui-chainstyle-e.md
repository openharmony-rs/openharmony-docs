# ChainStyle

定义链的风格，支持attributeModifier动态设置属性方法。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SPREAD

```TypeScript
SPREAD = 0
```

组件在约束锚点间均匀分布。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SPREAD_INSIDE

```TypeScript
SPREAD_INSIDE = 1
```

除首尾2个子组件的其他组件在约束锚点间均匀分布。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## PACKED

```TypeScript
PACKED = 2
```

链内子组件无间隙。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
