# Counter

计数器组件，提供相应的增加或者减少的计数操作。

> **说明：**

> - 该组件从API版本26.0.0开始支持[WithTheme]{@link with_theme}。

## 子组件

可以包含子组件。

## Counter

```TypeScript
Counter()
```

创建计数器组件。

**起始版本：** 7

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。
