# BarrierDirection

定义屏障线的方向。

**起始版本：** 12

**系统能力：** SystemCapability.Test.UiTest

## LEFT

```TypeScript
LEFT = 0
```

屏障在其所有[referencedId](arkts-arkui-barrierstyle-i.md)的最左侧。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
