# setCursor

## setCursor

```TypeScript
function setCursor(value: PointerStyle): void
```

方法语句中可使用的全局接口，调用该接口可更改当前的鼠标光标样式。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| value | PointerStyle | 是 | 设置的鼠标样式。 |
