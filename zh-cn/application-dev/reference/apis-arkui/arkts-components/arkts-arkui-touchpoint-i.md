# TouchPoint

配置跟手点坐标，不配置时，默认居中。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## x

```TypeScript
x: Dimension
```

跟手点X轴坐标。

**类型：** Dimension

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## y

```TypeScript
y: Dimension
```

跟手点Y轴坐标。

**类型：** Dimension

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
