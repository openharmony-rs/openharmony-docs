# MarqueeUpdatePolicy

跑马灯组件属性更新后，跑马灯的滚动策略。

**起始版本：** 23

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DEFAULT

```TypeScript
DEFAULT = 0
```

跑马灯组件属性更新后，从开始位置，运行跑马灯效果。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## PRESERVE_POSITION

```TypeScript
PRESERVE_POSITION = 1
```

跑马灯组件属性更新后，保持当前位置，运行跑马灯效果。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本23开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
