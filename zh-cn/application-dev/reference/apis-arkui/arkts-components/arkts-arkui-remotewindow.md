# RemoteWindow

远程控制窗口组件，可以通过此组件控制应用窗口，提供启动退出过程中控件动画和应用窗口联动动画的能力。


## RemoteWindow

```TypeScript
RemoteWindow(target: WindowAnimationTarget)
```

通过窗口动画对象创建组件。

**起始版本：** 9

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

**参数:**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| target | WindowAnimationTarget | 是 | 需要控制的动画窗口的描述。 |
