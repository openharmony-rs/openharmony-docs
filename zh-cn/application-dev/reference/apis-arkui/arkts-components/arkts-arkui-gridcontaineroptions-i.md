# GridContainerOptions

Defines the options of GridContainer.

**起始版本：** 7

**废弃版本：** 9

**替代接口：** grid_col/GridColOptions

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## columns

```TypeScript
columns?: number | "auto"
```

当前布局总列数。

默认值：'auto'

**类型：** number | "auto"

**起始版本：** 7

**废弃版本：** 9

**替代接口：** grid_col/GridColOptions

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## gutter

```TypeScript
gutter?: number | string
```

栅格布局列间距，不支持百分比。

**类型：** number | string

**起始版本：** 7

**废弃版本：** 9

**替代接口：** grid_col/GridColOptions

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## margin

```TypeScript
margin?: number | string
```

栅格布局两侧间距，不支持百分比。

**类型：** number | string

**起始版本：** 7

**废弃版本：** 9

**替代接口：** grid_col/GridColOptions

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## sizeType

```TypeScript
sizeType?: SizeType
```

选用设备宽度类型。

默认值：SizeType.Auto

**类型：** SizeType

**起始版本：** 7

**废弃版本：** 9

**替代接口：** grid_col/GridColOptions

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
