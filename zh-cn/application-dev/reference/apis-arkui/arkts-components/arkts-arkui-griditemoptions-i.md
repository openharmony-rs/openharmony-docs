# GridItemOptions

GridItem样式对象。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## style

```TypeScript
style?: GridItemStyle
```

设置GridItem样式。

默认值：GridItemStyle.NONE

设置为GridItemStyle.NONE时无样式。

设置为GridItemStyle.PLAIN时，显示Hover、Press态样式。

**类型：** GridItemStyle

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
