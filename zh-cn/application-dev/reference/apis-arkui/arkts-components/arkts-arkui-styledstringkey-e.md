# StyledStringKey

范围属性字符串样式。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## FONT

```TypeScript
FONT = 0
```

字体样式键。[TextStyle](arkts-arkui-textstyle-c.md)所属键。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
