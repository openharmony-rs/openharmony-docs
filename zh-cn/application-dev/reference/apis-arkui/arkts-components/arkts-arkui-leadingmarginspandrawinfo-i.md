# LeadingMarginSpanDrawInfo

自定义绘制信息。

**起始版本：** 22

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## baseline

```TypeScript
baseline: number
```

当前行的基线与组件上边缘的距离。

单位：[px](../../../../reference/apis-arkui/arkui-ts/ts-pixel-units.md#基本像素单位)

取值范围：大于等于0。

**类型：** number

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
