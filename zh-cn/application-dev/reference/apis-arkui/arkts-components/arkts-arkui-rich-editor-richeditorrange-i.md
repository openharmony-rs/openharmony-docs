# RichEditorRange

定义RichEditor的范围。

**起始版本：** 10

## end

```TypeScript
end?: number
```

文本的结束位置，省略或者超出文本范围时表示无穷大。

**类型：** number

**默认值：** text length

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

## start

```TypeScript
start?: number
```

文本的起始位置，省略或者设置负值时表示从0开始。

**类型：** number

**默认值：** 0

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
