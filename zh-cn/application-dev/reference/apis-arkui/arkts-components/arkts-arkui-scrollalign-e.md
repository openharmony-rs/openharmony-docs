# ScrollAlign

滚动对齐模式枚举。

**起始版本：** 10

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## START

```TypeScript
START
```

列表项起始边与列表起始边对齐。

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CENTER

```TypeScript
CENTER
```

列表项在主轴上居中对齐。

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
