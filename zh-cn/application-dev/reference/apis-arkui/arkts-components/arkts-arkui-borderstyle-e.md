# BorderStyle

Border Style

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Dotted

```TypeScript
Dotted
```

Displays as a series of dots with a radius of half the borderWidth.

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本11开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
