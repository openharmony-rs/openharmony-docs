# FontWeight

The font weight of the text

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Lighter

```TypeScript
Lighter = 0
```

Defines a lighter value than [Inherited Value]..

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
