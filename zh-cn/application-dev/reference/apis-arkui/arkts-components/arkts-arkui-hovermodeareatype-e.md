# HoverModeAreaType

悬停态显示区域类型。

**起始版本：** 14

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## TOP_SCREEN

```TypeScript
TOP_SCREEN = 0
```

上半屏。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
