# PointerStyle

```TypeScript
declare type PointerStyle = import('../api/@ohos.multimodalInput.pointer').default.PointerStyle
```

光标样式。

> **说明：**  
>  
> 直接使用cursorControl可能导致[UI上下文不明确](../../../../ui/arkts-global-interface.md#ui上下文不明确)的问题，建议使用getUIContext()获取  
> [UIContext](../arkts-apis/arkts-arkui-uicontext.md)实例，并使用  
> [getCursorController](@ohos.arkui.UIContext:UIContext#getcursorcontroller)获取绑定实例  
> 的cursorControl。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
