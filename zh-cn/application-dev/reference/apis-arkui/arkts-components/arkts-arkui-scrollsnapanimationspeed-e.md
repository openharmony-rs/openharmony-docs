# ScrollSnapAnimationSpeed

设置列表项滚动限位动画速度。

**起始版本：** 22

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## NORMAL

```TypeScript
NORMAL = 0
```

默认列表限位动画速度，通常用于列表项尺寸较大，划一下滚动一个列表项场景。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SLOW

```TypeScript
SLOW = 1
```

列表限位动画速度较慢，通常用于列表项尺寸较小，划一下滚动多个列表项场景。

**起始版本：** 22

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
