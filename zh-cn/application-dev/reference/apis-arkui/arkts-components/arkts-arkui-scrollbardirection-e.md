# ScrollBarDirection

滚动条方向枚举。

**起始版本：** 8

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Vertical

```TypeScript
Vertical = 0
```

纵向滚动条。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Horizontal

```TypeScript
Horizontal = 1
```

横向滚动条。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
