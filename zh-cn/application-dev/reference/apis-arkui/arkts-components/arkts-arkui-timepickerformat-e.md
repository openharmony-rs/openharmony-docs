# TimePickerFormat

时间选择器的数据格式。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## HOUR_MINUTE

```TypeScript
HOUR_MINUTE = 0
```

按照小时和分钟进行显示。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## HOUR_MINUTE_SECOND

```TypeScript
HOUR_MINUTE_SECOND = 1
```

Hour and minute and second

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
