# MenuType

菜单类型。

**起始版本：** 13

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SELECTION_MENU

```TypeScript
SELECTION_MENU = 0
```

文本选择菜单。

**起始版本：** 13

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本13开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## PREVIEW_MENU

```TypeScript
PREVIEW_MENU = 1
```

图片预览菜单。

**起始版本：** 13

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本13开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
