# SwipeEdgeEffect

滑动效果枚举

**起始版本：** 9

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Spring

```TypeScript
Spring
```

弹性物理动效，滑动到边缘后按初始速度或触摸事件继续滑动一段距离，释放后回弹。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
