# KeySource

定义触发按键事件的设备类型。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Unknown

```TypeScript
Unknown
```

输入设备类型未知。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Keyboard

```TypeScript
Keyboard
```

输入设备类型为键盘。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## JOYSTICK

```TypeScript
JOYSTICK
```

输入设备类型为游戏手柄。

**起始版本：** 15

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
