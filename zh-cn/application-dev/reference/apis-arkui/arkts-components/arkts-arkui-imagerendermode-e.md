# ImageRenderMode

图片的插值效果。

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Original

```TypeScript
Original = 0
```

原色渲染模式。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Template

```TypeScript
Template = 1
```

Render the image as a template image, ignoring the color information of the image.

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
