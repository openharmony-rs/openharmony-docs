# focusControl

焦点控制模块。

**起始版本：** 9

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 汇总

### 函数

| 名称 | 说明 |
| --- | --- |
| [requestFocus](arkts-arkui-requestfocus-f.md#requestfocus-1) | 方法语句中可使用的全局接口，调用此接口可以主动让焦点在下一帧渲染时转移至参数指定的组件上。如果需要指定组件立刻获焦，推荐使用FocusController中的焦点同步转移接口[requestFocus](../arkts-apis/arkts-arkui-focuscontroller-c.md#requestfocus-1)。 |
