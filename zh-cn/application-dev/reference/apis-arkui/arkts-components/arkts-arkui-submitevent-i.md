# SubmitEvent

定义用户提交事件。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## keepEditableState

```TypeScript
keepEditableState(): void
```

用户自定义输入框编辑状态，调用时保持编辑态。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
