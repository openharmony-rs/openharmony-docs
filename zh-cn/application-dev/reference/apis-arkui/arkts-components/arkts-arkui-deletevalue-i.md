# DeleteValue

提供从文本中删除值的接口。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## deleteOffset

```TypeScript
deleteOffset: number
```

删除的值的位置信息。

**类型：** number

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
