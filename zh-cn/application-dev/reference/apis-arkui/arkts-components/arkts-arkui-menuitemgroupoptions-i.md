# MenuItemGroupOptions

## footer

```TypeScript
footer?: ResourceStr | CustomBuilder
```

设置对应group的尾部显示信息。

未设置时，不显示尾部信息。

**类型：** ResourceStr \| CustomBuilder

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
