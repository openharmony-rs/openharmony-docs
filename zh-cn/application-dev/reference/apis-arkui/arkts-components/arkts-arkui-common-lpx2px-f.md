# lpx2px

## lpx2px

```TypeScript
declare function lpx2px(value: number): number
```

Converts a number in units of lpx to a number in units of px.

**起始版本：** 11

**废弃版本：** 18

**替代接口：** lpx2px

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本11开始，该接口支持在ArkTS卡片中使用。
