# ScrollEdgeOptions

滚动到容器边缘的参数。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## velocity

```TypeScript
velocity?: number
```

滚动到边缘位置的固定速度。设置为小于等于0的值时该参数不生效。

**类型：** number

**默认值：** 0vp/s

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
