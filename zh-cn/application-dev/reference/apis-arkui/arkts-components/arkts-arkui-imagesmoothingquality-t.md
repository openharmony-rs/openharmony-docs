# ImageSmoothingQuality

```TypeScript
declare type ImageSmoothingQuality = "high" | "low" | "medium"
```

定义图片平滑度类型。取值类型为下表类型中的并集。

**起始版本：** 8

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。
