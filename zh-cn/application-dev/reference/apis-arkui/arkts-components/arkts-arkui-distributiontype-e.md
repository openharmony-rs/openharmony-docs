# DistributionType

初始颜色随机值分布类型。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## UNIFORM

```TypeScript
UNIFORM = 0
```

初始颜色随机值分布为均匀分布。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## GAUSSIAN

```TypeScript
GAUSSIAN = 1
```

初始颜色随机值分布为高斯分布。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
