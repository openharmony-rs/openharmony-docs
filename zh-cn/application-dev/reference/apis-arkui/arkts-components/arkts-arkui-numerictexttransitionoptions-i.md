# NumericTextTransitionOptions

数字翻牌的参数。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## enableBlur

```TypeScript
enableBlur?: boolean
```

是否开启翻牌模糊效果。

默认值：false

true：开启翻牌模糊效果。

false：不开启翻牌模糊效果。

**类型：** boolean

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## flipDirection

```TypeScript
flipDirection?: FlipDirection
```

翻牌方向。

默认值：FlipDirection.DOWN

**类型：** FlipDirection

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
