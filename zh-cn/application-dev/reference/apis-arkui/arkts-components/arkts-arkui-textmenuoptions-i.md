# TextMenuOptions

菜单选项。

**起始版本：** 16

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## showMode

```TypeScript
showMode?: TextMenuShowMode
```

菜单的显示模式。

默认值：TextMenuShowMode.DEFAULT

**类型：** TextMenuShowMode

**起始版本：** 16

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本16开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
