# EffectComponentOptions（系统接口）

设置当前EffectComponent构造参数，包含EffectComponent的渲染层级。

**起始版本：** 20

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## effectLayer

```TypeScript
effectLayer?: EffectLayer
```

EffectComponent的渲染层级。

默认值：EffectLayer.NONE

**类型：** EffectLayer

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
