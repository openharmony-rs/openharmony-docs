# ModelType

模型类型枚举

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## TEXTURE

```TypeScript
TEXTURE = 0
```

渲染到纹理，GPU将此纹理合成到屏幕.

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D

## SURFACE

```TypeScript
SURFACE = 1
```

渲染到表面，专有硬件将此表面合成到屏幕.

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D
