# StrokeJoinStyle

定义线条拐角的样式，即在绘制折线时线段拐角处的画笔样式。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## MITER_JOIN

```TypeScript
MITER_JOIN = 0
```

拐角类型为锐角。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
