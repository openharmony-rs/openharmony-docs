# ListItemAlign

设置子组件在List交叉轴方向的对齐方式。

**起始版本：** 9

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Start

```TypeScript
Start
```

ListItem在List中，交叉轴方向首部对齐。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Center

```TypeScript
Center
```

ListItem在List中，交叉轴方向居中对齐。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## End

```TypeScript
End
```

ListItem在List中，交叉轴方向尾部对齐。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本9开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
