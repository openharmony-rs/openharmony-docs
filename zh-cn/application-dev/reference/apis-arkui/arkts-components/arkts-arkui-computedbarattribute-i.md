# ComputedBarAttribute

滚动条位置和长度对象。

**起始版本：** 10

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## totalLength

```TypeScript
totalLength: number
```

Grid内容总长度，单位px。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## totalOffset

```TypeScript
totalOffset: number
```

Grid内容相对显示区域的总偏移，单位px。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
