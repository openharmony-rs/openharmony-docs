# NavigationOptions

路由栈操作选项。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## animated

```TypeScript
animated?: boolean
```

是否支持转场动画。

true：支持转场动画；false：不支持转场动画。

默认值：true

**类型：** boolean

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## launchMode

```TypeScript
launchMode?: LaunchMode
```

路由栈的操作模式。

默认值：LaunchMode.STANDARD

**类型：** LaunchMode

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
