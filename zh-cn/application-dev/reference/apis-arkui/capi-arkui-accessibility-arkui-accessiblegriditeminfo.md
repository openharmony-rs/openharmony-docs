## 概述

用于特定组件设置组件的属性值，如List、Flex、Select、Swiper组件。

**起始版本：** 13

**相关模块：** [ArkUI_Accessibility](capi-arkui-accessibility.md)

## 汇总

### 成员变量

| 名称 | 描述     |
| -- |--------|
| bool heading | 是否是标题。 |
| bool selected | 是否被选中。 |
| int32_t columnIndex | 列下标。   |
| int32_t rowIndex | 行下标。   |
| int32_t columnSpan | 列跨度。   |
| int32_t rowSpan | 行跨度。   |
