&lt;[tabs](js-components-container-tabs.md)&gt;的子组件，用来展示tab的标签区，子组件排列方式为横向排列。

## 权限列表

无


## 子组件

支持。


## 属性

除支持[通用属性](js-components-common-attributes.md)外，还支持如下属性：

| 名称 | 类型 | 默认值 | 必填 | 描述 |
| -------- | -------- | -------- | -------- | -------- |
| mode | string | scrollable | 否 | 设置组件宽度的可延展性。可选值为：<br/>-&nbsp;scrollable：子组件宽度为实际设置的宽度，当宽度之和（包括margin边距）大于tab-bar的宽度时，子组件可以横向滑动。<br/>-&nbsp;fixed：子组件宽度均分tab-bar的宽度。 |
