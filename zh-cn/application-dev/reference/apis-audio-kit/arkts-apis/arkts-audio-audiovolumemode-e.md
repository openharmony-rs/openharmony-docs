# AudioVolumeMode

表示音量模式的枚举。

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Audio.Volume

## SYSTEM_GLOBAL

```TypeScript
SYSTEM_GLOBAL = 0
```

系统级音量（默认模式）。

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Audio.Volume

## APP_INDIVIDUAL

```TypeScript
APP_INDIVIDUAL = 1
```

应用级音量。

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Audio.Volume
