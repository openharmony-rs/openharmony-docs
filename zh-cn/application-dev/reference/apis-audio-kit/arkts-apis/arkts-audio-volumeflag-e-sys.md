# VolumeFlag（系统接口）

Enumerates volume related operations.
Flags should be powers of 2!

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Audio.Volume

**系统接口：** 此接口为系统接口。

## FLAG_SHOW_SYSTEM_UI

```TypeScript
FLAG_SHOW_SYSTEM_UI = 1
```

Show system volume bar.

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Audio.Volume

**系统接口：** 此接口为系统接口。
