## loop

```TypeScript
loop: boolean
```

是否开启循环播放，true表示开启循环播放，false表示不开启循环播放。

**类型：** boolean

**起始版本：** 10

## volume

```TypeScript
volume: number
```

指定的相对音量大小，取值范围为[0.00, 1.00]，1表示最大音量，即100%。

**类型：** number

**起始版本：** 10
