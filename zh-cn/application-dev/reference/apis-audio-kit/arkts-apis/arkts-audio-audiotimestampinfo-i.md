# AudioTimestampInfo

音频流时间戳和当前数据帧位置信息。

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Audio.Core

## framePos

```TypeScript
readonly framePos: number
```

当前播放或者录制的数据帧位置。

**类型：** number

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Audio.Core

## timestamp

```TypeScript
readonly timestamp: number
```

播放或者录制到当前数据帧位置时对应的时间戳，单位为纳秒。

**类型：** number

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Audio.Core
