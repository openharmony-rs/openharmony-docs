# ChannelBlendMode

表示声道混合模式类型的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Audio.Core

## MODE_DEFAULT

```TypeScript
MODE_DEFAULT = 0
```

无声道混合。

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Audio.Core

## MODE_BLEND_LR

```TypeScript
MODE_BLEND_LR = 1
```

混合左右声道。

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Audio.Core

## MODE_ALL_LEFT

```TypeScript
MODE_ALL_LEFT = 2
```

从左声道覆盖到右声道混合。

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Audio.Core

## MODE_ALL_RIGHT

```TypeScript
MODE_ALL_RIGHT = 3
```

从右声道覆盖到左声道混合。

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Audio.Core
