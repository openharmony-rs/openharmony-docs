# SystemToneOptions（系统接口）

提示音参数选项。

**起始版本：** 11

## muteAudio

```TypeScript
muteAudio?: boolean
```

提示音是否静音，true表示静音，false表示不静音。

**类型：** boolean

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.SystemSound.Core

## muteHaptics

```TypeScript
muteHaptics?: boolean
```

提示音时是否振动，true表示关闭振动，false表示开启振动。

**类型：** boolean

**起始版本：** 11
