# DeviceFlag

表示音频设备类型的枚举。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device

## OUTPUT_DEVICES_FLAG

```TypeScript
OUTPUT_DEVICES_FLAG = 1
```

输出设备。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device

## INPUT_DEVICES_FLAG

```TypeScript
INPUT_DEVICES_FLAG = 2
```

输入设备。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device

## ALL_DEVICES_FLAG

```TypeScript
ALL_DEVICES_FLAG = 3
```

所有设备。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device
