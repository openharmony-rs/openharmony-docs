# ToneHapticsMode（系统接口）

枚举，系统铃音场景的振动模式。

| 名称 | 值 | 说明 |
| ----------------------------- | -- | -------------------- |
| NONE | 0 | 无振动模式。 |
| SYNC | 1 | 与铃音同步模式。 |
| NON_SYNC | 2 | 非同步模式。 |

**起始版本：** 14

**系统能力：** SystemCapability.Multimedia.SystemSound.Core

**系统接口：** 此接口为系统接口。

## NONE

```TypeScript
NONE = 0
```

None haptics mode.

**起始版本：** 14

**系统能力：** SystemCapability.Multimedia.SystemSound.Core

**系统接口：** 此接口为系统接口。

## SYNC

```TypeScript
SYNC = 1
```

Haptics is synchronized with tone.

**起始版本：** 14

**系统能力：** SystemCapability.Multimedia.SystemSound.Core

**系统接口：** 此接口为系统接口。

## NON_SYNC

```TypeScript
NON_SYNC = 2
```

Haptics is out of synchronize with tone.

**起始版本：** 14

**系统能力：** SystemCapability.Multimedia.SystemSound.Core

**系统接口：** 此接口为系统接口。
