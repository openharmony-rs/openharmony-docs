# InterruptType

表示中断类型的枚举。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## INTERRUPT_TYPE_BEGIN

```TypeScript
INTERRUPT_TYPE_BEGIN = 1
```

音频播放中断事件开始。

**起始版本：** 7

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## INTERRUPT_TYPE_END

```TypeScript
INTERRUPT_TYPE_END = 2
```

音频播放中断事件结束。

**起始版本：** 7

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Renderer
