# AudioEncodingType

表示音频编码类型的枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## ENCODING_TYPE_INVALID

```TypeScript
ENCODING_TYPE_INVALID = -1
```

无效。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Core

## ENCODING_TYPE_RAW

```TypeScript
ENCODING_TYPE_RAW = 0
```

PCM编码。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Core
