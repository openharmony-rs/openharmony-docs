# AudioLoopbackEqualizerPreset

表示返听均衡器类型的枚举。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer

## FLAT

```TypeScript
FLAT = 1
```

保持原始声音，不进行均衡调节。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer

## FULL

```TypeScript
FULL = 2
```

使人声更饱满（默认的均衡器类型）。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer

## BRIGHT

```TypeScript
BRIGHT = 3
```

使人声更明亮。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer
