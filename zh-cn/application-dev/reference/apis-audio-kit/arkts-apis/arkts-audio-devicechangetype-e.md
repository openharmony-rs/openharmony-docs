# DeviceChangeType

表示设备连接状态变化的枚举。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device

## CONNECT

```TypeScript
CONNECT = 0
```

设备连接。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device

## DISCONNECT

```TypeScript
DISCONNECT = 1
```

断开设备连接。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device
