# AudioSamplingRate

表示音频采样率的枚举（具体设备支持的采样率规格会存在差异）。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## SAMPLE_RATE_8000

```TypeScript
SAMPLE_RATE_8000 = 8000
```

采样率为8000。单位为赫兹（Hz）。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core
