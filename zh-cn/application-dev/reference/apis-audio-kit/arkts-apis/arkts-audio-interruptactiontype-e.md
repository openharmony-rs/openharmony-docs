# InterruptActionType

表示中断事件返回类型的枚举。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** InterruptType

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## TYPE_ACTIVATED

```TypeScript
TYPE_ACTIVATED = 0
```

表示触发焦点事件。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** INTERRUPT_TYPE_BEGIN

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## TYPE_INTERRUPT

```TypeScript
TYPE_INTERRUPT = 1
```

表示音频打断事件。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** INTERRUPT_TYPE_END

**系统能力：** SystemCapability.Multimedia.Audio.Renderer
