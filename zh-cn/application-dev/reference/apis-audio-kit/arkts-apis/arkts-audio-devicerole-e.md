# DeviceRole

表示设备角色的枚举。

**起始版本：** 7

**系统能力：** SystemCapability.Multimedia.Audio.Device

## INPUT_DEVICE

```TypeScript
INPUT_DEVICE = 1
```

输入设备角色。

**起始版本：** 7

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Device

## OUTPUT_DEVICE

```TypeScript
OUTPUT_DEVICE = 2
```

输出设备角色。

**起始版本：** 7

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Device
