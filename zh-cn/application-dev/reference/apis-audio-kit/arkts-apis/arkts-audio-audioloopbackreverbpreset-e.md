# AudioLoopbackReverbPreset

表示返听混响模式的枚举。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer

## ORIGINAL

```TypeScript
ORIGINAL = 1
```

保持原始混响，不进行任何增强。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer

## KTV

```TypeScript
KTV = 2
```

提供类似KTV的混响效果。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer

## THEATER

```TypeScript
THEATER = 3
```

提供类似剧场的混响效果（默认的混响模式）。

**起始版本：** 21

**系统能力：** SystemCapability.Multimedia.Audio.Capturer
