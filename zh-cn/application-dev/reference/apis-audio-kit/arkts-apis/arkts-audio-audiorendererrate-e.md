# AudioRendererRate

表示音频渲染速度的枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## RENDER_RATE_NORMAL

```TypeScript
RENDER_RATE_NORMAL = 0
```

正常速度。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## RENDER_RATE_DOUBLE

```TypeScript
RENDER_RATE_DOUBLE = 1
```

2倍速。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Renderer

## RENDER_RATE_HALF

```TypeScript
RENDER_RATE_HALF = 2
```

0.5倍速。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Renderer
