# ToneHapticsMode（系统接口）

枚举，系统铃音场景的振动模式。

| 名称 | 值 | 说明 |  
| ----------------------------- | -- | -------------------- |  
| NONE | 0 | 无振动模式。 |  
| SYNC | 1 | 与铃音同步模式。 |  
| NON_SYNC | 2 | 非同步模式。 |

**起始版本：** 14
