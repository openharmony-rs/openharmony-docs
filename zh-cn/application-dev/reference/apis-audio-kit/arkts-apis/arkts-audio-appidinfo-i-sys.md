# AppIdInfo（系统接口）

描述app id信息。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Multimedia.Audio.Core

**系统接口：** 此接口为系统接口。

## appFullTokenId

```TypeScript
appFullTokenId: number
```

**类型：** number

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统接口：** 此接口为系统接口。
