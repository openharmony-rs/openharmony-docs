# DeviceBlockStatus

表示音频设备是否被堵塞的枚举。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Audio.Device

## UNBLOCKED

```TypeScript
UNBLOCKED = 0
```

音频设备正常。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Audio.Device

## BLOCKED

```TypeScript
BLOCKED = 1
```

音频设备被堵塞。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Audio.Device
