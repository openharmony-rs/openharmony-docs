# VolumeAdjustType（系统接口）

Enumerates volume adjustment types.

**起始版本：** 10

**系统能力：** SystemCapability.Multimedia.Audio.Volume

**系统接口：** 此接口为系统接口。

## VOLUME_UP

```TypeScript
VOLUME_UP = 0
```

Adjust volume up.

**起始版本：** 10

**系统能力：** SystemCapability.Multimedia.Audio.Volume

**系统接口：** 此接口为系统接口。

## VOLUME_DOWN

```TypeScript
VOLUME_DOWN = 1
```

Adjust volume down.

**起始版本：** 10

**系统能力：** SystemCapability.Multimedia.Audio.Volume

**系统接口：** 此接口为系统接口。
