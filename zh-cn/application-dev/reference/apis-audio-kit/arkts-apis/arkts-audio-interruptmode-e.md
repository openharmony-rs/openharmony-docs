# InterruptMode

表示焦点模型的枚举。

**起始版本：** 9

**系统能力：** SystemCapability.Multimedia.Audio.Interrupt

## SHARE_MODE

```TypeScript
SHARE_MODE = 0
```

共享焦点模式。

**起始版本：** 9

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Interrupt

## INDEPENDENT_MODE

```TypeScript
INDEPENDENT_MODE = 1
```

独立焦点模式。

**起始版本：** 9

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Audio.Interrupt
