# ContentType

表示音频内容类型的枚举。

**起始版本：** 7

**废弃版本：** 10

**替代接口：** StreamUsage

**系统能力：** SystemCapability.Multimedia.Audio.Core

## CONTENT_TYPE_UNKNOWN

```TypeScript
CONTENT_TYPE_UNKNOWN = 0
```

未知类型。

**起始版本：** 7

**废弃版本：** 10

**替代接口：** STREAM_USAGE_UNKNOWN

**系统能力：** SystemCapability.Multimedia.Audio.Core
