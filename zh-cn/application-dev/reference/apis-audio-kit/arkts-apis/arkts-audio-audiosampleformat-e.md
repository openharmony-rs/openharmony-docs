# AudioSampleFormat

表示音频采样格式的枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## SAMPLE_FORMAT_INVALID

```TypeScript
SAMPLE_FORMAT_INVALID = -1
```

无效格式。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## SAMPLE_FORMAT_U8

```TypeScript
SAMPLE_FORMAT_U8 = 0
```

无符号8位整数。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## SAMPLE_FORMAT_S16LE

```TypeScript
SAMPLE_FORMAT_S16LE = 1
```

带符号的16位整数，小尾数。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## SAMPLE_FORMAT_S24LE

```TypeScript
SAMPLE_FORMAT_S24LE = 2
```

带符号的24位整数，小尾数。

由于系统限制，该采样格式仅部分设备支持，请根据实际情况使用。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

```TypeScript
SAMPLE_FORMAT_S32LE = 3
```

由于系统限制，该采样格式仅部分设备支持，请根据实际情况使用。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

