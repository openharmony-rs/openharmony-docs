# DisplayRotation

设备显示器的显示方向。

**起始版本：** 9

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_0

```TypeScript
ROTATION_0 = 0
```

设备显示器不旋转，初始形态垂直显示。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_90

```TypeScript
ROTATION_90 = 1
```

设备显示器顺时针旋转90°，水平显示。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_180

```TypeScript
ROTATION_180 = 2
```

设备显示器顺时针旋转180°，逆向垂直显示。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_270

```TypeScript
ROTATION_270 = 3
```

设备显示器顺时针旋转270°，逆向水平显示。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
