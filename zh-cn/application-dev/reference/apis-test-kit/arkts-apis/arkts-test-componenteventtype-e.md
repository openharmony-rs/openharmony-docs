# ComponentEventType

支持监听的控件操作事件类型。

**起始版本：** 22

**系统能力：** SystemCapability.Test.UiTest

## COMPONENT_UNDEFINED

```TypeScript
COMPONENT_UNDEFINED = 0
```

非窗口变化事件。

**说明：** 该枚举值仅支持作为返回值，如果作为接口入参会抛出异常。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
