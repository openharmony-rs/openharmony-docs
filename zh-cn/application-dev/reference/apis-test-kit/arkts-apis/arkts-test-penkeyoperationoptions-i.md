# PenKeyOperationOptions

笔按键操作选项。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Test.UiTest

## point

```TypeScript
point?: Point
```

空鼠模式操作的坐标点。当按键为 AIR_MOUSE 并处于空鼠模式时，该坐标点是必需的。

**类型：** Point

**起始版本：** 26.0.0

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
