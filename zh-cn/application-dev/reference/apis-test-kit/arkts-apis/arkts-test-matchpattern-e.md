# MatchPattern

控件属性支持的匹配模式。

**起始版本：** 8

**系统能力：** SystemCapability.Test.UiTest

## EQUALS

```TypeScript
EQUALS = 0
```

等于给定值。

从API version 11开始，该接口支持在原子化服务中使用。

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
