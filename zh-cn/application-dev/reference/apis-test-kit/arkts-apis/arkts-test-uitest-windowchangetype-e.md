## WINDOW_UNDEFINED

```TypeScript
WINDOW_UNDEFINED = 0
```

非窗口变化事件。

**说明：** 该枚举值仅支持作为返回值，如果作为接口入参会抛出异常。

**起始版本：** 22

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
