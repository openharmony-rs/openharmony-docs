# WindowChangeType

支持监听的窗口变化事件类型。

**起始版本：** 22

**系统能力：** SystemCapability.Test.UiTest

## WINDOW_UNDEFINED

```TypeScript
WINDOW_UNDEFINED = 0
```

非窗口变化事件。

**说明：** 该枚举值仅支持作为返回值，如果作为接口入参会抛出异常。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## WINDOW_ADDED

```TypeScript
WINDOW_ADDED = 1
```

窗口出现事件。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## WINDOW_REMOVED

```TypeScript
WINDOW_REMOVED = 2
```

窗口消失事件。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

