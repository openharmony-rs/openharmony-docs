## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| uint32_t baudRate | 波特率。 |
| uint8_t nDataBits | 数据传输位数。 |
| uint8_t nStopBits | 数据停止位数。 |
| uint8_t parity | 校验参数设置。 |
