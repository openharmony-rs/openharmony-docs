# DeviceCapability

表示设备支持的能力。

**起始版本：** 6

**系统能力：** SystemCapability.Global.ResourceManager

## deviceType

```TypeScript
deviceType: DeviceType
```

当前设备类型。

**类型：** DeviceType

**起始版本：** 6

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## screenDensity

```TypeScript
screenDensity: ScreenDensity
```

当前设备屏幕密度。

**类型：** ScreenDensity

**起始版本：** 6

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager
