# Resource

```TypeScript
export type Resource = _Resource
```

表示资源信息，包含资源ID值、应用包名、模块名称等信息，一般可使用$r方式获取。

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
