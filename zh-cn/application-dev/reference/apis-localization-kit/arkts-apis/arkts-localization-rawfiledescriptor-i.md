# RawFileDescriptor

本模块提供rawfile文件所在hap的descriptor信息。

**起始版本：** 8

**系统能力：** SystemCapability.Global.ResourceManager

## fd

```TypeScript
fd: number
```

文件描述符。

**类型：** number

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## length

```TypeScript
length: number
```

文件长度。

**类型：** number

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## offset

```TypeScript
offset: number
```

起始偏移量。

**类型：** number

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager
