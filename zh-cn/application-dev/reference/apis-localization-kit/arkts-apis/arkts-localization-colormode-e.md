# ColorMode

用于表示当前设备颜色模式。

**起始版本：** 12

**系统能力：** SystemCapability.Global.ResourceManager

## DARK

```TypeScript
DARK = 0
```

深色模式。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## LIGHT

```TypeScript
LIGHT = 1
```

浅色模式。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager
