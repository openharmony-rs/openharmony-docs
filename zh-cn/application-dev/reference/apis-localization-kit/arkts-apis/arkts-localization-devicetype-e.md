# DeviceType

用于表示当前设备类型。

**起始版本：** 6

**系统能力：** SystemCapability.Global.ResourceManager

## DEVICE_TYPE_PHONE

```TypeScript
DEVICE_TYPE_PHONE = 0x00
```

手机。

**起始版本：** 6

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager
