# RawFileDescriptor

```TypeScript
export type RawFileDescriptor = _RawFileDescriptor
```

表示rawfile文件所在HAP的文件描述符（fd）。

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
