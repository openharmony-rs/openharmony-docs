# DataMigrationProgress（系统接口）

描述数据迁移的进度信息。

**起始版本：** 23

**系统能力：** SystemCapability.Global.FontManager

**系统接口：** 此接口为系统接口。

## progressPercentage

```TypeScript
progressPercentage: number
```

表示数据迁移百分比进展，取值：0-100。
取值范围为全体整数。

**类型：** number

**起始版本：** 23

**系统能力：** SystemCapability.Global.FontManager

**系统接口：** 此接口为系统接口。

## timeRemaining

```TypeScript
timeRemaining: number
```

表示预计剩余时间，单位：秒。
取值限定为整数。

**类型：** number

**起始版本：** 23

**系统能力：** SystemCapability.Global.FontManager

**系统接口：** 此接口为系统接口。
