# Direction

用于表示设备屏幕方向。

**起始版本：** 6

**系统能力：** SystemCapability.Global.ResourceManager

## DIRECTION_VERTICAL

```TypeScript
DIRECTION_VERTICAL = 0
```

竖屏。

**起始版本：** 6

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## DIRECTION_HORIZONTAL

```TypeScript
DIRECTION_HORIZONTAL = 1
```

横屏。

**起始版本：** 6

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.ResourceManager
