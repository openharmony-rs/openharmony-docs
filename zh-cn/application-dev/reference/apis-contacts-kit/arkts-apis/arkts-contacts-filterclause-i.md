# FilterClause

联系人过滤条件。多个筛选条件之间是“或者”的关系，如果参数是数组类型，数组最多只能包含3个元素。

**起始版本：** 15

**系统能力：** SystemCapability.Applications.Contacts

## dataItem

```TypeScript
dataItem?: DataFilter
```

联系人数据过滤项。

**类型：** DataFilter

**起始版本：** 15

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.Contacts

## focusModeList

```TypeScript
focusModeList?: Array<FilterOptions>
```

专注模式。

**类型：** Array<FilterOptions>

**起始版本：** 15

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.Contacts
