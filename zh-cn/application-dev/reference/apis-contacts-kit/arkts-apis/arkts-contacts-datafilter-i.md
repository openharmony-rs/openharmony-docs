# DataFilter

联系人数据过滤项。

**起始版本：** 15

**系统能力：** SystemCapability.Applications.Contacts

## field

```TypeScript
field: DataField
```

联系人数据字段。

**类型：** DataField

**起始版本：** 15

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.Contacts

## options

```TypeScript
options: Array<FilterOptions>
```

联系人过滤参数，数组中多个FilterOptions之间是“或”的关系，数组的最大长度为3。

**类型：** Array<FilterOptions>

**起始版本：** 15

**元服务API：** 从API版本15开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.Contacts
