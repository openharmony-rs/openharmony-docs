# Attribute

枚举，类型为number。联系人属性列表。
通过JSON格式创建数据。

**起始版本：** 7

**系统能力：** SystemCapability.Applications.ContactsData

## ATTR_CONTACT_EVENT

```TypeScript
ATTR_CONTACT_EVENT = 0
```

联系人的生日、周年纪念等重要日期。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.ContactsData
