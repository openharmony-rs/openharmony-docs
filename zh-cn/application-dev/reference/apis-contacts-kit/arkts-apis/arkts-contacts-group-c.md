# Group

联系人的群组类。

**起始版本：** 7

**系统能力：** SystemCapability.Applications.ContactsData

## groupId

```TypeScript
groupId?: number
```

联系人群组的Id。

**类型：** number

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.ContactsData

## title

```TypeScript
title: string
```

联系人群组的名称。

**类型：** string

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。
