# PrintPageSize

定义打印页面尺寸的接口。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## height

```TypeScript
height: number
```

表示页面高度，单位：毫米。

**类型：** number

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## id

```TypeScript
id: string
```

表示纸张类型ID。

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## name

```TypeScript
name: string
```

## width
