# PrinterInformation

定义打印机信息的接口。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework


**类型：** string

**起始版本：** 18

**系统能力：** SystemCapability.Print.PrintFramework
