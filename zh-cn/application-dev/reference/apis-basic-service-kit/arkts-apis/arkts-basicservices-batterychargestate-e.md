# BatteryChargeState

表示电池充电状态的枚举。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## NONE

```TypeScript
NONE
```

表示电池充电状态为未充电。

**起始版本：** 6

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## ENABLE

```TypeScript
ENABLE
```

表示电池充电状态为使能状态。

**起始版本：** 6

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## DISABLE

```TypeScript
DISABLE
```

表示电池充电状态为停止状态。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core
