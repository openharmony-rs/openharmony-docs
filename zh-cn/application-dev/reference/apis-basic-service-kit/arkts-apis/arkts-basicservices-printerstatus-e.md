# PrinterStatus

打印机状态的枚举。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINTER_IDLE

```TypeScript
PRINTER_IDLE = 0
```

表示打印机空闲状态。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINTER_BUSY

```TypeScript
PRINTER_BUSY = 1
```

表示打印机忙碌状态。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINTER_UNAVAILABLE

```TypeScript
PRINTER_UNAVAILABLE = 2
```

表示打印机脱机状态。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework
