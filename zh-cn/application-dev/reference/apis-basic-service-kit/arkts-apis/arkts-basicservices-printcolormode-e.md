# PrintColorMode

打印色彩模式的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## COLOR_MODE_MONOCHROME

```TypeScript
COLOR_MODE_MONOCHROME = 0
```

表示黑白打印。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## COLOR_MODE_COLOR

```TypeScript
COLOR_MODE_COLOR = 1
```

表示彩色打印。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework
