# Notification

通知栏自定义信息。

**起始版本：** 15

**系统能力：** SystemCapability.Request.FileTransferAgent

## disable

```TypeScript
disable?: boolean
```

Disables the notification.
If the value is false, a notification will be displayed, otherwise nothing will be displayed.
If not specified, the value is false.

**类型：** boolean

**起始版本：** 20

**系统能力：** SystemCapability.Request.FileTransferAgent

**系统接口：** 此接口为系统接口。
