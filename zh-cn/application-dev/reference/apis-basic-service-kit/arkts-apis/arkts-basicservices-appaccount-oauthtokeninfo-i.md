# OAuthTokenInfo

表示OAuth令牌信息。
> **说明：**  
>  
> 从API version 8开始支持，从API version 9开始废弃。建议使用[AuthTokenInfo](arkts-basicservices-appaccount-authtokeninfo-i.md)替代。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [AuthTokenInfo](arkts-basicservices-appaccount-authtokeninfo-i.md)
