## MENU_PANEL

```TypeScript
MENU_PANEL = 1
```

菜单面板可以作为一级面板，显示当前应用程序可以提供的功能，如翻译、搜索等。

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

## MAIN_PANEL

```TypeScript
MAIN_PANEL = 2
```

主面板可以作为二级面板，当用户点击菜单面板中的功能按钮时弹出，显示特定的翻译或搜索结果等。

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。
