# CompressStrategy

CompressStrategy作为[Options](arkts-basicservices-options-i.md)的一个属性，用于指定压缩时的压缩策略。

**起始版本：** 7

**系统能力：** SystemCapability.BundleManager.Zlib

## COMPRESS_STRATEGY_DEFAULT_STRATEGY

```TypeScript
COMPRESS_STRATEGY_DEFAULT_STRATEGY = 0
```

常规数据策略。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib

## COMPRESS_STRATEGY_FILTERED

```TypeScript
COMPRESS_STRATEGY_FILTERED = 1
```

过滤器产生的数据压缩策略。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib

COMPRESS_STRATEGY_HUFFMAN_ONLY = 2

**系统能力：** SystemCapability.BundleManager.Zlib
