# GetDomainAccountInfoOptions（系统接口）

表示查询域账号信息的选项。

**起始版本：** 10

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## accountName

```TypeScript
accountName: string
```

域账号名。

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## domain

```TypeScript
domain?: string
```

域名。默认为undefined。

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## serverConfigId

```TypeScript
serverConfigId?: string
```

域账号所属服务器标识。默认为undefined。

**类型：** string

**起始版本：** 12

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
