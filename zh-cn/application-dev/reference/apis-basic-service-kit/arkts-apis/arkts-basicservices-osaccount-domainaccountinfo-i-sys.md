# DomainAccountInfo

表示域账号信息。

**起始版本：** 8

**系统能力：** SystemCapability.Account.OsAccount

## accountId

```TypeScript
accountId?: string
```

域账号标识。

此接口为系统接口，默认为undefined。

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## isAuthenticated

```TypeScript
isAuthenticated?: boolean
```

指示域账号是否已认证。true表示指定的域账号已认证；false表示指定的域账号未认证。

此接口为系统接口，默认为false。

**类型：** boolean

**起始版本：** 11

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
