# @ohos.systemTime

本模块主要由系统时间和系统时区功能组成。开发者可以设置、获取系统时间及系统时区。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** [systemDateTime:systemDateTime](arkts-systemdatetime.md#systemDateTime)

**系统能力：** SystemCapability.MiscServices.Time
