# PauseDownloadOptions（系统接口）

暂停下载选项。

**起始版本：** 9

**系统能力：** SystemCapability.Update.UpdateService

**系统接口：** 此接口为系统接口。

## isAllowAutoResume

```TypeScript
isAllowAutoResume: boolean
```

是否允许自动恢复。

true表示允许自动恢复，false表示不允许。

**类型：** boolean

**起始版本：** 9

**系统能力：** SystemCapability.Update.UpdateService

**系统接口：** 此接口为系统接口。
