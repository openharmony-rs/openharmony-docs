# PrintDuplexMode

打印单双面模式的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## DUPLEX_MODE_NONE

```TypeScript
DUPLEX_MODE_NONE = 0
```

表示单面打印。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## DUPLEX_MODE_LONG_EDGE

```TypeScript
DUPLEX_MODE_LONG_EDGE = 1
```

表示双面打印沿长边翻转。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## DUPLEX_MODE_SHORT_EDGE

```TypeScript
DUPLEX_MODE_SHORT_EDGE = 2
```

表示双面打印沿短边翻转。

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework
