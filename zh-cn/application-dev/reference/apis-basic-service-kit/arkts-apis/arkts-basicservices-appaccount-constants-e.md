## ACTION_ADD_ACCOUNT_IMPLICITLY

```TypeScript

```

表示操作，隐式添加账号。

**废弃版本：** 9

**替代接口：** [ACTION_CREATE_ACCOUNT_IMPLICITLY](arkts-basicservices-appaccount-constants-e.md#action_create_account_implicitly)
