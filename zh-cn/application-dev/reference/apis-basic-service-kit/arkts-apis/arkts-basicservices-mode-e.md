# Mode

定义模式选项。

当应用的前台任务切换到后台一段时间后会显示运行失败或暂停，而后台任务不受此操作影响。

**起始版本：** 10

**系统能力：** SystemCapability.Request.FileTransferAgent

## BACKGROUND

```TypeScript
BACKGROUND
```

表示后台任务。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Request.FileTransferAgent

## FOREGROUND

```TypeScript
FOREGROUND
```

表示前台任务。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Request.FileTransferAgent
