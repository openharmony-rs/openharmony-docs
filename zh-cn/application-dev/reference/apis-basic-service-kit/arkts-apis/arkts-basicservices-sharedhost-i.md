# SharedHost

定义共享设备信息的接口。

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

## ip

```TypeScript
ip: string
```

表示共享设备的IP地址。

**类型：** string

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Print.PrintFramework

## shareName

```TypeScript
shareName: string
```

表示共享设备的主机名称。

**类型：** string

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Print.PrintFramework

## workgroupName

```TypeScript
workgroupName: string
```

表示共享设备的工作组名称。

**类型：** string

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Print.PrintFramework
