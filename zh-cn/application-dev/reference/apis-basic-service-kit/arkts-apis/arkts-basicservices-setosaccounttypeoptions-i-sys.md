# SetOsAccountTypeOptions（系统接口）

设置系统账号类型的选项。

**起始版本：** 24

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## token

```TypeScript
token?: Uint8Array
```

表示从认证管理接口获取的token，默认为空。

**类型：** Uint8Array

**起始版本：** 24

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
