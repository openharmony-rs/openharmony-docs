# PrintResolution（系统接口）

定义打印分辨率的接口。

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## horizontalDpi

```TypeScript
horizontalDpi: number
```

表示水平DPI。单位：DPI。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## id

```TypeScript
id: string
```

表示分辨率ID。

**类型：** string

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## verticalDpi

```TypeScript
verticalDpi: number
```

表示垂直DPI。单位：DPI。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。
