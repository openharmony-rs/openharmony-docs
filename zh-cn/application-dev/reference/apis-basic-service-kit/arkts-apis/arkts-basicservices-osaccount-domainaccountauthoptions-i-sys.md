# DomainAccountAuthOptions（系统接口）

表示域账号认证的选项。

**起始版本：** 24

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## serverParams

```TypeScript
serverParams?: Record<string, Object>
```

域账号认证服务器配置参数。默认为undefined。

**类型：** Record<string, Object>

**起始版本：** 24

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
