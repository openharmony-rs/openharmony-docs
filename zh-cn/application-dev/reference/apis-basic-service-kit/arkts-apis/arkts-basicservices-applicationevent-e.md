# ApplicationEvent

打印应用事件的枚举。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## APPLICATION_CREATED

```TypeScript
APPLICATION_CREATED = 0
```

表示打印应用被拉起的事件。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## APPLICATION_CLOSED_FOR_STARTED

```TypeScript
APPLICATION_CLOSED_FOR_STARTED = 1
```

表示由于点击打印而关闭打印应用的事件。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## APPLICATION_CLOSED_FOR_CANCELED

```TypeScript
APPLICATION_CLOSED_FOR_CANCELED = 2
```

表示由于点击取消而关闭打印应用的事件。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework
