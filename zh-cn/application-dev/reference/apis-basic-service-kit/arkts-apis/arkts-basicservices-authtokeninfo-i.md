# AuthTokenInfo

表示Auth令牌信息。

**起始版本：** 9

**系统能力：** SystemCapability.Account.AppAccount

## account

```TypeScript
account?: AppAccountInfo
```

令牌所属的账号信息，默认为空。

**类型：** AppAccountInfo

**起始版本：** 9

**系统能力：** SystemCapability.Account.AppAccount

## authType

```TypeScript
authType: string
```

令牌的鉴权类型。

**类型：** string

**起始版本：** 9

**系统能力：** SystemCapability.Account.AppAccount

## token

```TypeScript
token: string
```

令牌的取值。

**类型：** string

**起始版本：** 9

**系统能力：** SystemCapability.Account.AppAccount
