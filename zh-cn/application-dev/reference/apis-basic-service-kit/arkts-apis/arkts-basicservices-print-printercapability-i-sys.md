# PrinterCapability（系统接口）

定义打印能力的接口。

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## colorMode

```TypeScript
colorMode: number
```

表示色彩模式。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## duplexMode

```TypeScript
duplexMode: number
```

表示单双面打印模式。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## minMargin

```TypeScript
minMargin?: PrintMargin
```

表示打印机最小边距。

**类型：** PrintMargin

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## options

```TypeScript
options?: Object
```

**系统接口：** 此接口为系统接口。
