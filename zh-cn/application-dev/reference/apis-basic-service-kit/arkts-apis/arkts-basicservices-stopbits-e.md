# StopBits

Enumerates of the number of stop bits.

**起始版本：** 19

**系统能力：** SystemCapability.USB.USBManager.Serial

## STOPBIT_1

```TypeScript
STOPBIT_1 = 0
```

报文的有效停止位宽为1比特。

**起始版本：** 19

**系统能力：** SystemCapability.USB.USBManager.Serial

## STOPBIT_2

```TypeScript
STOPBIT_2 = 1
```

报文的有效停止位宽为2比特。

**起始版本：** 19

**系统能力：** SystemCapability.USB.USBManager.Serial
