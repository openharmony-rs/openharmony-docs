# OAuthTokenInfo

表示OAuth令牌信息。

> **说明：**
>
> 从API version 8开始支持，从API version 9开始废弃。建议使用[AuthTokenInfo](arkts-basicservices-authtokeninfo-i.md)替代。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [AuthTokenInfo](arkts-basicservices-authtokeninfo-i.md)

**系统能力：** SystemCapability.Account.AppAccount

## authType

```TypeScript
authType: string
```

令牌的鉴权类型。

**类型：** string

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [authType](arkts-basicservices-authtokeninfo-i.md#authtype)

**系统能力：** SystemCapability.Account.AppAccount

## token

```TypeScript
token: string
```

令牌的取值。

**类型：** string

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [token](arkts-basicservices-authtokeninfo-i.md#token)

**系统能力：** SystemCapability.Account.AppAccount
