# PrintDocumentFormat

打印数据格式的枚举。

**起始版本：** 23

**系统能力：** SystemCapability.Print.PrintFramework

## DOCUMENT_FORMAT_AUTO

```TypeScript
DOCUMENT_FORMAT_AUTO = 0
```

表示自动检测格式。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Print.PrintFramework
