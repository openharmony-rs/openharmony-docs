# FingerprintTips（系统接口）

表示指纹身份验证过程中提示的枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## FINGERPRINT_TIP_GOOD

```TypeScript
FINGERPRINT_TIP_GOOD = 0
```

表示采集的图像良好。

**起始版本：** 8

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## FINGERPRINT_TIP_IMAGER_DIRTY

```TypeScript
FINGERPRINT_TIP_IMAGER_DIRTY = 1
```

表示由于传感器上可疑或检测到污垢，指纹图像噪声过大。

**起始版本：** 8

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## FINGERPRINT_TIP_INSUFFICIENT

```TypeScript
FINGERPRINT_TIP_INSUFFICIENT = 2
```

表示由于检测到的情况，指纹图像噪声太大，无法处理。

**起始版本：** 8

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
