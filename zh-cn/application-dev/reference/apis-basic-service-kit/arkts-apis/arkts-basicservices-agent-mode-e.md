# Mode

定义模式选项。

当应用的前台任务切换到后台一段时间后会显示运行失败或暂停，而后台任务不受此操作影响。

**起始版本：** 10

## BACKGROUND

```TypeScript
BACKGROUND
```

表示后台任务。

**起始版本：** 10

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
