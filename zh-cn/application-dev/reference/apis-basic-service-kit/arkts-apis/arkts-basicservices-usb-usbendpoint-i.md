# USBEndpoint

通过USB发送和接收数据的端口。通过[USBInterface](arkts-basicservices-usb-usbinterface-i.md)获取。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [USBEndpoint](arkts-basicservices-usbmanager-usbendpoint-i.md)
