# Constants

表示常量的枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Account.AppAccount

## ACTION_ADD_ACCOUNT_IMPLICITLY

```TypeScript
ACTION_ADD_ACCOUNT_IMPLICITLY = 'addAccountImplicitly'
```

表示操作，隐式添加账号。

**说明：**从API version 8开始支持，从API version 9开始废弃，建议使用ACTION_CREATE_ACCOUNT_IMPLICITLY替代。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [ACTION_CREATE_ACCOUNT_IMPLICITLY](arkts-basicservices-constants-e.md#action_create_account_implicitly)

**系统能力：** SystemCapability.Account.AppAccount
