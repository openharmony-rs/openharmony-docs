# DocFlavor

打印数据来源形式的枚举。

**起始版本：** 23

**系统能力：** SystemCapability.Print.PrintFramework

## FILE_DESCRIPTOR

```TypeScript
FILE_DESCRIPTOR = 0
```

表示文件数据。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Print.PrintFramework
