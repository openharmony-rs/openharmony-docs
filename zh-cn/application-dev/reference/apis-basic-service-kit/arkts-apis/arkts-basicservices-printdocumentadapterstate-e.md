# PrintDocumentAdapterState

**起始版本：** 11

**系统能力：** SystemCapability.Print.PrintFramework

## PREVIEW_DESTROY

```TypeScript
PREVIEW_DESTROY = 0
```

表示预览失败。

**系统能力：** SystemCapability.Print.PrintFramework
