# PrinterRange（系统接口）

定义打印范围的接口。

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## endPage

```TypeScript
endPage?: number
```

表示结束页。默认值为待打印文件的最大页数。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## pages

```TypeScript
pages?: Array<number>
```

表示待打印的页面范围的集合。默认值为空。

**类型：** Array<number>

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## startPage

```TypeScript
startPage?: number
```

表示起始页。默认值为1。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。
