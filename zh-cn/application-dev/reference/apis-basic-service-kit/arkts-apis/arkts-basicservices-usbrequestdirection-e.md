# USBRequestDirection

Enumerates request directions.

**起始版本：** 9

**系统能力：** SystemCapability.USB.USBManager

## USB_REQUEST_DIR_TO_DEVICE

```TypeScript
USB_REQUEST_DIR_TO_DEVICE = 0
```

写数据，主设备往从设备。

**起始版本：** 9

**系统能力：** SystemCapability.USB.USBManager

## USB_REQUEST_DIR_FROM_DEVICE

```TypeScript
USB_REQUEST_DIR_FROM_DEVICE = 0x80
```

读数据，从设备往主设备。

**起始版本：** 9

**系统能力：** SystemCapability.USB.USBManager
