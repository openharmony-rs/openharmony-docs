# PrintMargin（系统接口）

定义打印页边距的接口。

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## bottom

```TypeScript
bottom?: number
```

表示页面下边距。默认值为0。单位：毫米。

**类型：** number

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。
