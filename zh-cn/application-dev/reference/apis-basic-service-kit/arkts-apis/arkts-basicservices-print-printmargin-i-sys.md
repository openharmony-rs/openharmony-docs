# PrintMargin（系统接口）

定义打印页边距的接口。

**起始版本：** 24

<!--Device-print-interface PrintMargin--><!--Device-print-interface PrintMargin-End-->

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## bottom

```TypeScript
bottom?: number
```

表示页面下边距。默认值为0。单位：毫米。

**类型：** number

**起始版本：** 24
