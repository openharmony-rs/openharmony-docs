# CompressLevel

压缩等级。

**起始版本：** 7

**系统能力：** SystemCapability.BundleManager.Zlib

## COMPRESS_LEVEL_NO_COMPRESSION

```TypeScript
COMPRESS_LEVEL_NO_COMPRESSION = 0
```

压缩率为0压缩等级。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib
