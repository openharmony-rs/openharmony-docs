# PanelInfo（系统接口）

划词面板属性信息。

**起始版本：** 24

**系统能力：** SystemCapability.SelectionInput.Selection

**系统接口：** 此接口为系统接口。

## height

```TypeScript
height: number
```

划词面板高度，单位为px。

**类型：** number

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.SelectionInput.Selection

**系统接口：** 此接口为系统接口。

## panelType

```TypeScript
panelType: PanelType
```

划词面板类型。

**类型：** PanelType

**默认值：** MENU_PANEL

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.SelectionInput.Selection

**系统接口：** 此接口为系统接口。
