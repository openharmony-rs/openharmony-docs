# GzErrorOutputInfo

GzError返回信息。

**起始版本：** 12

**系统能力：** SystemCapability.BundleManager.Zlib

## status

```TypeScript
status: ReturnStatus
```

返回zlib文件状态码，参考ReturnStatus的定义。

**类型：** ReturnStatus

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib

## statusMsg

```TypeScript
statusMsg: string
```

zlib文件上发生的最后一个状态的状态消息。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib
