# RemoveOsAccountOptions（系统接口）

表示用于删除系统账号的可选参数。

**起始版本：** 24

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## token

```TypeScript
token?: Uint8Array
```

表示从认证管理接口获取的token，默认为空。

**类型：** Uint8Array

**起始版本：** 24

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
