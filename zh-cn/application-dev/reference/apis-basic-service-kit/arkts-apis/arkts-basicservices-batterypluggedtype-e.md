# BatteryPluggedType

表示连接的充电器类型的枚举。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## NONE

```TypeScript
NONE
```

表示未获取到连接充电器类型。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## AC

```TypeScript
AC
```

表示连接的充电器类型为交流充电器。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## USB

```TypeScript
USB
```

表示连接的充电器类型为USB。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

## WIRELESS

```TypeScript
WIRELESS
```

表示连接的充电器类型为无线充电器。

**起始版本：** 6

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core
