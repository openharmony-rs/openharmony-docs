# Network

定义网络选项。

网络不满足设置条件时，未执行的任务会等待执行，执行中的任务将失败或暂停。

**起始版本：** 10

**系统能力：** SystemCapability.Request.FileTransferAgent

## ANY

```TypeScript
ANY
```

表示不限网络类型。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Request.FileTransferAgent

## WIFI

```TypeScript
WIFI
```

表示无线网络。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Request.FileTransferAgent

## CELLULAR

```TypeScript
CELLULAR
```

表示蜂窝数据网络。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Request.FileTransferAgent
