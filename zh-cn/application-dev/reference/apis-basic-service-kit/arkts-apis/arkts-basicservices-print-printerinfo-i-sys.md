# PrinterInfo（系统接口）

定义打印信息的接口。

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## capability

```TypeScript
capability?: PrinterCapability
```

表示打印机功能。

**类型：** PrinterCapability

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## description

```TypeScript
description?: string
```

表示打印机说明。

**类型：** string

**起始版本：** 24

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

