# PrintOrientationMode

打印方向的枚举。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## ORIENTATION_MODE_PORTRAIT

```TypeScript
ORIENTATION_MODE_PORTRAIT = 0
```

表示纵向打印。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## ORIENTATION_MODE_LANDSCAPE

```TypeScript
ORIENTATION_MODE_LANDSCAPE= 1
```

表示横向打印。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## ORIENTATION_MODE_REVERSE_LANDSCAPE

```TypeScript
ORIENTATION_MODE_REVERSE_LANDSCAPE = 2
```

表示横向翻转打印。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## ORIENTATION_MODE_REVERSE_PORTRAIT

```TypeScript
ORIENTATION_MODE_REVERSE_PORTRAIT = 3
```

表示纵向翻转打印。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## ORIENTATION_MODE_NONE

```TypeScript
ORIENTATION_MODE_NONE = 4
```

表示自适应方向打印。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework
