# GzHeader

传递从zlib例程中获取的Gzip头部信息。

**起始版本：** 12

**系统能力：** SystemCapability.BundleManager.Zlib

## comment

```TypeScript
comment?: ArrayBuffer
```

**类型：** ArrayBuffer

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib
