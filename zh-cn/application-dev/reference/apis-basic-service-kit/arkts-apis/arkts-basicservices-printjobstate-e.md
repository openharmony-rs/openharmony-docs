# PrintJobState

打印任务状态的枚举。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINT_JOB_PREPARE

```TypeScript
PRINT_JOB_PREPARE = 0
```

表示打印任务的初始状态。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINT_JOB_QUEUED

```TypeScript
PRINT_JOB_QUEUED = 1
```

表示打印任务传送到打印机。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINT_JOB_RUNNING

```TypeScript
PRINT_JOB_RUNNING = 2
```

表示执行打印任务。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINT_JOB_BLOCKED

```TypeScript
PRINT_JOB_BLOCKED = 3
```

表示打印任务已被阻止。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework

## PRINT_JOB_COMPLETED

```TypeScript
PRINT_JOB_COMPLETED = 4
```

表示打印任务完成。

**起始版本：** 14

**系统能力：** SystemCapability.Print.PrintFramework
