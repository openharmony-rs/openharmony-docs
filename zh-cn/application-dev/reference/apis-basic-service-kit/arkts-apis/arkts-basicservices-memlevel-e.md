# MemLevel

内存等级。

**起始版本：** 7

**系统能力：** SystemCapability.BundleManager.Zlib

## MEM_LEVEL_MIN

```TypeScript
MEM_LEVEL_MIN = 1
```

zlib接口在压缩过程中最小使用内存。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib

## MEM_LEVEL_MAX

```TypeScript
MEM_LEVEL_MAX = 9
```

zlib接口在压缩过程中最大使用内存。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib

## MEM_LEVEL_DEFAULT

```TypeScript
MEM_LEVEL_DEFAULT = 8
```

zlib接口在压缩过程中默认使用内存。

**起始版本：** 7

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.Zlib
