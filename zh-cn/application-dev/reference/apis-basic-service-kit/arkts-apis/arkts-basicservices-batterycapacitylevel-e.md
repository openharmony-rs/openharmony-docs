# BatteryCapacityLevel

表示电池电量等级的枚举。

**起始版本：** 9

**系统能力：** SystemCapability.PowerManager.BatteryManager.Core

```TypeScript
LEVEL_NONE
```

表示电池电量等级为未知电量。
