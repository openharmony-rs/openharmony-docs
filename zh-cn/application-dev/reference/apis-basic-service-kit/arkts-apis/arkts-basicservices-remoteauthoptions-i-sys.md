# RemoteAuthOptions（系统接口）

表示远程认证的可选参数集合。

**起始版本：** 12

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## collectorNetworkId

```TypeScript
collectorNetworkId?: string
```

凭据收集者的网络标识，默认为空。

**类型：** string

**起始版本：** 12

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## collectorTokenId

```TypeScript
collectorTokenId?: number
```

凭据收集者的令牌标识，默认为undefined。

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## verifierNetworkId

```TypeScript
verifierNetworkId?: string
```

凭据验证者的网络标识，默认为空。

**类型：** string

**起始版本：** 12

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。
