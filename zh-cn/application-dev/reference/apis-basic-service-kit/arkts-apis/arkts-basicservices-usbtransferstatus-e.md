# UsbTransferStatus

Enumerates the status code returned after data processing is complete.

**起始版本：** 18

**系统能力：** SystemCapability.USB.USBManager

## TRANSFER_COMPLETED

```TypeScript
TRANSFER_COMPLETED = 0
```

传输完成。

**起始版本：** 18

**系统能力：** SystemCapability.USB.USBManager

## TRANSFER_ERROR

```TypeScript
TRANSFER_ERROR = 1
```

传输失败。

**起始版本：** 18

**系统能力：** SystemCapability.USB.USBManager

## TRANSFER_TIMED_OUT

