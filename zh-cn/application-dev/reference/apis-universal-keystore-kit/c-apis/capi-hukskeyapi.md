# HuksKeyApi

## 概述

Describes the OpenHarmony Universal KeyStore (HUKS) capabilities, including key management andcryptography operations, provided for applications.The keys managed by HUKS can be imported by applications or generated by calling the HUKS APIs.

**起始版本：** 9
