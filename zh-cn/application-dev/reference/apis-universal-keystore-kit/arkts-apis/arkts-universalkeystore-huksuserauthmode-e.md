# HuksUserAuthMode

表示用户认证模式。

**起始版本：** 12

**系统能力：** SystemCapability.Security.Huks.Extension

## HUKS_USER_AUTH_MODE_LOCAL

```TypeScript
HUKS_USER_AUTH_MODE_LOCAL = 0
```

本地认证模式。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Huks.Extension

## HUKS_USER_AUTH_MODE_COAUTH

```TypeScript
HUKS_USER_AUTH_MODE_COAUTH = 1
```

跨端协同认证模式。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Huks.Extension
