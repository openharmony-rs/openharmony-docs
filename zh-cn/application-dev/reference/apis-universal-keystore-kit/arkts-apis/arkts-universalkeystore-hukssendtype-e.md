# HuksSendType

表示发送TAG的方式。

**起始版本：** 8

**系统能力：** 
- API版本12+：SystemCapability.Security.Huks.Core
- API版本8-11：SystemCapability.Security.Huks.Extension

## HUKS_SEND_TYPE_ASYNC

```TypeScript
HUKS_SEND_TYPE_ASYNC = 0
```

表示异步发送TAG。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** 
- API版本12+：SystemCapability.Security.Huks.Core
- API版本8-11：SystemCapability.Security.Huks.Extension

## HUKS_SEND_TYPE_SYNC

```TypeScript
HUKS_SEND_TYPE_SYNC = 1
```

表示同步发送TAG。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** 
- API版本12+：SystemCapability.Security.Huks.Core
- API版本8-11：SystemCapability.Security.Huks.Extension
