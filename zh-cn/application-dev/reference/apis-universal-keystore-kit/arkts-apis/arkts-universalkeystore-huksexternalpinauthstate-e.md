# HuksExternalPinAuthState

枚举PIN认证的状态

**起始版本：** 22

**系统能力：** SystemCapability.Security.Huks.CryptoExtension

## HUKS_EXT_CRYPTO_PIN_NO_AUTH

```TypeScript
HUKS_EXT_CRYPTO_PIN_NO_AUTH = 0
```

Ukey PIN未认证。

**起始版本：** 22

**系统能力：** SystemCapability.Security.Huks.CryptoExtension

## HUKS_EXT_CRYPTO_PIN_AUTH_SUCCEEDED

```TypeScript
HUKS_EXT_CRYPTO_PIN_AUTH_SUCCEEDED = 1
```

PIN认证成功

**起始版本：** 22

**系统能力：** SystemCapability.Security.Huks.CryptoExtension

## HUKS_EXT_CRYPTO_PIN_LOCKED

```TypeScript
HUKS_EXT_CRYPTO_PIN_LOCKED = 2
```

UKey PIN码已锁定

**起始版本：** 22

**系统能力：** SystemCapability.Security.Huks.CryptoExtension
