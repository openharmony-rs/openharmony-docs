# HuksKeyAlg

表示密钥使用的算法。

**起始版本：** 8

**系统能力：** SystemCapability.Security.Huks.Core

## HUKS_ALG_RSA

```TypeScript
HUKS_ALG_RSA = 1
```

表示使用RSA算法。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** 
- API版本12+：SystemCapability.Security.Huks.Core
- API版本8-11：SystemCapability.Security.Huks.Extension
