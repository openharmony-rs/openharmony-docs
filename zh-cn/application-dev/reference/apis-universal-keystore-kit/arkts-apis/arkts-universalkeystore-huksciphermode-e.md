# HuksCipherMode

表示加密模式。

**起始版本：** 8

**系统能力：** SystemCapability.Security.Huks.Core

## HUKS_MODE_ECB

```TypeScript
HUKS_MODE_ECB = 1
```

表示使用ECB加密模式。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Huks.Core
