# SourceType

表示媒体源的类型。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## URL

```TypeScript
URL = 0
```

媒体源的类型是URL。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## MSE

```TypeScript
MSE = 1
```

媒体源的类型是blob。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
