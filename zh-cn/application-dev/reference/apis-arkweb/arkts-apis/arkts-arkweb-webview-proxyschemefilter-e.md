# ProxySchemeFilter

Enum type supplied to {@link insertProxyRule} for indicating the scheme filter for proxy.

**起始版本：** 15

## MATCH_ALL_SCHEMES

```TypeScript
MATCH_ALL_SCHEMES = 0
```

所有的scheme都会使用代理。

**起始版本：** 15
