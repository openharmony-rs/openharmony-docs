# PdfConfiguration

Defines the configuration of creating pdf, related to {@Link createPdf} method.

**起始版本：** 14

**系统能力：** SystemCapability.Web.Webview.Core


**类型：** number

**起始版本：** 14

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。
