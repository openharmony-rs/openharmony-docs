# LargestContentfulPaint

提供网页绘制页面最大内容的详细信息。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## imageBPP

```TypeScript
imageBPP?: number
```

最大图片像素位数。

**类型：** number

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## largestImagePaintTime

```TypeScript
largestImagePaintTime?: number
```

最大图片加载的时间，单位是以毫秒表示。

**类型：** number

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## largestTextPaintTime

```TypeScript
largestTextPaintTime?: number
```

最大文本加载时间，单位是以毫秒表示。

**类型：** number

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## navigationStartTime

```TypeScript
navigationStartTime?: number
```

导航条加载时间，单位以微秒表示。

**类型：** number

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
