# OnFaviconReceivedEvent

定义应用为当前页面接收到新的网站图标（favicon）时的回调函数。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## favicon

```TypeScript
favicon: PixelMap
```

接收到的favicon图标的PixelMap对象。

**类型：** PixelMap

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
