# OnFullScreenEnterCallback

```TypeScript
type OnFullScreenEnterCallback = (event: FullScreenEnterEvent) => void
```

Web组件进入全屏时触发的回调。

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| event | FullScreenEnterEvent | 是 | Web组件进入全屏的回调事件详情。 |
