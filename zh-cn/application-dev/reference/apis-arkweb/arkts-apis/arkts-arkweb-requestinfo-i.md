# RequestInfo

Web组件发送的资源请求信息。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## formData

```TypeScript
formData: string
```

请求的表单数据。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## method

```TypeScript
method: string
```

请求的方法。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## url

```TypeScript
url: string
```

请求的链接。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
