# WebHeader

Web组件返回的请求/响应头对象。

**起始版本：** 9

**系统能力：** SystemCapability.Web.Webview.Core

## headerKey

```TypeScript
headerKey: string
```

请求/响应头的key。

**类型：** string

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## headerValue

```TypeScript
headerValue: string
```

请求/响应头的value。

**类型：** string

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
