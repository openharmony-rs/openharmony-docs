# SuspendType

表示播放器的挂起类型。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## ENTER_BACK_FORWARD_CACHE

```TypeScript
ENTER_BACK_FORWARD_CACHE = 0
```

页面进入BFCache。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## ENTER_BACKGROUND

```TypeScript
ENTER_BACKGROUND = 1
```

页面进入后台。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## AUTO_CLEANUP

```TypeScript
AUTO_CLEANUP = 2
```

系统自动清理。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core
