# Preload

播放器预加载媒体数据。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## NONE

```TypeScript
NONE = 0
```

不预加载。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## METADATA

```TypeScript
METADATA = 1
```

只预加载媒体的元数据。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## AUTO

```TypeScript
AUTO = 2
```

预加载足够多的媒体数据，以保证能流畅地播放。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
