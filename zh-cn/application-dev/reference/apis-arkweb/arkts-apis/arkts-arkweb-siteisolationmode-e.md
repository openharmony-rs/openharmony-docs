# SiteIsolationMode

Indicates the site isolation mode of the application, default value depends on different devices type.

**起始版本：** 21

**系统能力：** SystemCapability.Web.Webview.Core

## PARTIAL

```TypeScript
PARTIAL = 0
```

The partial site isolation mode

**起始版本：** 21

**系统能力：** SystemCapability.Web.Webview.Core

## STRICT

```TypeScript
STRICT = 1
```

The strict site isolation mode

**起始版本：** 21

**系统能力：** SystemCapability.Web.Webview.Core
