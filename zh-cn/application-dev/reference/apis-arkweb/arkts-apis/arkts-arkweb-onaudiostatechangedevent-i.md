# OnAudioStateChangedEvent

定义网页上的音频播放状态发生改变时的回调函数。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## playing

```TypeScript
playing: boolean
```

当前页面的音频播放状态，true表示正在播放，false表示未播放。

**类型：** boolean

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
