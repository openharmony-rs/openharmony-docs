# NativeMediaPlayerSurfaceInfo

[应用接管网页媒体播放功能](../../../../reference/apis-arkweb/arkts-basic-components-web-attributes.md#enablenativemediaplayer12)中
用于同层渲染的 surface 信息。

> **说明：**
>
> - 本Class首批接口从API version 12开始支持。
>
> - 示例效果请以真机运行为准。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## id

```TypeScript
id: string
```

surface的id，用于同层渲染的NativeImage的surfaceId。

详见[NativeEmbedDataInfo](arkts-arkweb-web-nativeembeddatainfo-i.md#NativeEmbedDataInfo)。

**类型：** string

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## rect

```TypeScript
rect: RectEvent
```

surface的位置信息。

**类型：** RectEvent

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core
