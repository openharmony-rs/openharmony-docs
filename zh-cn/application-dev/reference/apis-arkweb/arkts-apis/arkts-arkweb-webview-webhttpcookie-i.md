# WebHttpCookie

Defines the Web's HTTPCookie.
<p><strong>API Note</strong>:<br>
The maximum length allowed for each attribute value in a cookie string is 1024.
</p>

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## domain

```TypeScript
domain: string
```

获取cookie的域名

**类型：** string

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core
