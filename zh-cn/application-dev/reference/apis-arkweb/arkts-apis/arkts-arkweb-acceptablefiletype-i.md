# AcceptableFileType

定义文件选择器拉取文件时网页推荐的文件类型信息。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## acceptableType

```TypeScript
acceptableType: Array<string>
```

文件类型数组，包含若干可供选择的文件类型。

**类型：** Array<string>

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## mimeType

```TypeScript
mimeType: string
```

文件MIME类型。

**类型：** string

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core
