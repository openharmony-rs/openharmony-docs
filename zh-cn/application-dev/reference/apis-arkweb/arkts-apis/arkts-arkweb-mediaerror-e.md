# MediaError

播放器的错误类型。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## NETWORK_ERROR

```TypeScript
NETWORK_ERROR = 1
```

网络错误。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
