# ControllerAttachState

表示controller的绑定状态枚举

**起始版本：** 20

**系统能力：** SystemCapability.Web.Webview.Core

## UNATTACHED

```TypeScript
UNATTACHED = 0
```

表示controller当前并未绑定一个web组件

**起始版本：** 20

**系统能力：** SystemCapability.Web.Webview.Core

## ATTACHED

```TypeScript
ATTACHED = 1
```

Indicates webviewController is attached a web component.

**起始版本：** 20

**系统能力：** SystemCapability.Web.Webview.Core
