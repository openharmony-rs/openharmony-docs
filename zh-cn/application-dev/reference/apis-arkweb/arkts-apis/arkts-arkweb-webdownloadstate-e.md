# WebDownloadState

Defines the state for download.

**起始版本：** 11

**系统能力：** SystemCapability.Web.Webview.Core

## IN_PROGRESS

```TypeScript
IN_PROGRESS = 0
```

The web download is in progress.

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
