# PlaybackStatus

[handleStatusChanged](arkts-arkweb-nativemediaplayerhandler-i.md#handlestatuschanged-1) 接口参数， 用于表示播放器的播
放状态。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## PAUSED

```TypeScript
PAUSED = 0
```

播放状态为暂停状态，表示媒体已暂停。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## PLAYING

```TypeScript
PLAYING = 1
```

播放状态为播放状态，表示媒体正在播放。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
