# NativeEmbedVisibilityInfo

提供同层标签的可见性信息。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## embedId

```TypeScript
embedId: string
```

同层渲染标签的唯一id。

**类型：** string

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## visibility

```TypeScript
visibility: boolean
```

可见性。

true表示可见，false表示不可见。

**类型：** boolean

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core
