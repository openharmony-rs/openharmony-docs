# FileSelectorMode

文件选择器的模式。

**起始版本：** 9

**系统能力：** SystemCapability.Web.Webview.Core

## FileOpenMode

```TypeScript
FileOpenMode = 0
```

打开上传单个文件。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## FileOpenMultipleMode


**系统能力：** SystemCapability.Web.Webview.Core
