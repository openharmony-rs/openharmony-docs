# WebResponseType

菜单的响应类型。

**起始版本：** 13

**系统能力：** SystemCapability.Web.Webview.Core

## LONG_PRESS

```TypeScript
LONG_PRESS = 1
```

通过长按触发菜单弹出。

**起始版本：** 13

**系统能力：** SystemCapability.Web.Webview.Core

## RIGHT_CLICK

```TypeScript
RIGHT_CLICK = 2
```

通过鼠标右键触发菜单弹出。

**起始版本：** 21

**系统能力：** SystemCapability.Web.Webview.Core
