# OnFirstScreenPaintCallback

```TypeScript
type OnFirstScreenPaintCallback = (firstScreenPaint: FirstScreenPaint) => void
```

检测到首屏渲染结束时会触发此回调。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| firstScreenPaint | FirstScreenPaint | 是 | 检测到首屏渲染时的详细信息。 |
