# OverScrollMode

设置Web的过滚动模式为关闭或开启。

**起始版本：** 11

**系统能力：** SystemCapability.Web.Webview.Core

## NEVER

```TypeScript
NEVER = 0
```

Web过滚动模式关闭。

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## ALWAYS

```TypeScript
ALWAYS = 1
```

Web过滚动模式开启。

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
