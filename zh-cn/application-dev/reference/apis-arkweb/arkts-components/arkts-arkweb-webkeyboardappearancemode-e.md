# WebKeyboardAppearanceMode

提供给{@链接键盘外观}的枚举类型，用于设置Web键盘外观模式。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Web.Webview.Core

## NONE_IMMERSIVE

```TypeScript
NONE_IMMERSIVE = 0
```

默认皮肤模式，没有沉浸式风格。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Web.Webview.Core

## IMMERSIVE

```TypeScript
IMMERSIVE = 1
```

没有沉浸式风格

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Web.Webview.Core

LIGHT_IMMERSIVE = 2
