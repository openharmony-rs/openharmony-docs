# MicrophoneCaptureState

定义麦克风使用状态的值。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## NONE

```TypeScript
NONE = 0
```

麦克风未工作。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## PAUSED

```TypeScript
PAUSED = 1
```

麦克风暂停中。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## ACTIVE

```TypeScript
ACTIVE = 2
```

麦克风捕获中。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core
