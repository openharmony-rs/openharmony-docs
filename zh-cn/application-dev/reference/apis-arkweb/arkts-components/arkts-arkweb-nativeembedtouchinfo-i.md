# NativeEmbedTouchInfo

提供手指触摸到同层标签的详细信息。

**起始版本：** 11

**系统能力：** SystemCapability.Web.Webview.Core

## embedId

```TypeScript
embedId?: string
```

同层标签的唯一id。

**类型：** string

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## result

```TypeScript
result?: EventResult
```

通知Web组件手势事件的消费结果。

**类型：** EventResult

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## touchEvent

```TypeScript
touchEvent?: TouchEvent
```

手指触摸动作信息。

**类型：** TouchEvent

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
