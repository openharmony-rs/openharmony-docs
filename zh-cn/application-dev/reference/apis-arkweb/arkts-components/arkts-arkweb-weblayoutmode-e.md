# WebLayoutMode

**起始版本：** 11

**系统能力：** SystemCapability.Web.Webview.Core

## NONE

```TypeScript
NONE = 0
```

**起始版本：** 11

**系统能力：** SystemCapability.Web.Webview.Core

**起始版本：** 11
