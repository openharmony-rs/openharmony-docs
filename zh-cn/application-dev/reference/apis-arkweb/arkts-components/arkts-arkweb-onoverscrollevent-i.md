# OnOverScrollEvent

定义网页过度滚动时触发的回调。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## xOffset

```TypeScript
xOffset: number
```

以网页最左端为基准，水平过度滚动的偏移量。

单位：vp。

**类型：** number

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## yOffset

```TypeScript
yOffset: number
```

以网页最上端为基准，竖直过度滚动的偏移量。

单位：vp。

**类型：** number

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
