# FirstScreenPaint

检测到首屏渲染时的事件信息。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## firstScreenPaintTime

```TypeScript
firstScreenPaintTime: number
```

url所指页面首屏绘制完成的时刻。

单位：毫秒。

**类型：** number

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## navigationStartTime

```TypeScript
navigationStartTime: number
```

url所指页面开始导航的时刻。

单位：毫秒。

**类型：** number

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## url

```TypeScript
url: string
```

本次首屏渲染统计所对应的url。

**类型：** string

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core
