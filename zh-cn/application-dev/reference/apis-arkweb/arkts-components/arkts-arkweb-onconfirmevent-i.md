# OnConfirmEvent

定义网页触发 `confirm()` 弹窗时的回调函数。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## message

```TypeScript
message: string
```

显示在弹窗中的信息。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## result

```TypeScript
result: JsResult
```

通知Web组件用户的操作结果。

**类型：** JsResult

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## url

```TypeScript
url: string
```

当前显示弹窗的网页的URL。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
