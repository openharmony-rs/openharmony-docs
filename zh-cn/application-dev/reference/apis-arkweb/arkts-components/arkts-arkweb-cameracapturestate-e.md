# CameraCaptureState

定义摄像头使用状态的值。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## NONE

```TypeScript
NONE = 0
```

摄像头未工作。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## PAUSED

```TypeScript
PAUSED = 1
```

摄像头暂停中。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core

## ACTIVE

```TypeScript
ACTIVE = 2
```

摄像头捕获中。

**起始版本：** 23

**系统能力：** SystemCapability.Web.Webview.Core
