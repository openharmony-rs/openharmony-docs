# ThreatType

Enum type supplied to {@link threatType} for the website's threat type.

**起始版本：** 11

**系统能力：** SystemCapability.Web.Webview.Core

## THREAT_ILLEGAL

```TypeScript
THREAT_ILLEGAL = 0
```

非法网站。

**起始版本：** 11

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

```TypeScript

```
