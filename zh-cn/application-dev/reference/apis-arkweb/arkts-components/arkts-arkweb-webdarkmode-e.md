# WebDarkMode

Web深色模式的配置

**起始版本：** 9

**系统能力：** SystemCapability.Web.Webview.Core

## Off

```TypeScript
Off = 0
```

Web深色模式关闭。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## On

```TypeScript
On = 1
```

Web深色模式开启。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## Auto

```TypeScript
Auto = 2
```

Web深色模式跟随系统。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
