# ScrollbarLayoutPolicy

定义滚动条布局模式控制参数的枚举类型。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Web.Webview.Core

## CONTENT

```TypeScript
CONTENT = 0
```

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Web.Webview.Core

**起始版本：** 26.0.0
