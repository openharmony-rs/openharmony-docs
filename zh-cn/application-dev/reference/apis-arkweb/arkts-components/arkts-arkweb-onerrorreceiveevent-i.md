# OnErrorReceiveEvent

定义网页加载遇到错误时触发该回调。

**起始版本：** 12

**系统能力：** SystemCapability.Web.Webview.Core

## error

```TypeScript
error: WebResourceError
```

网页加载资源错误的封装信息。

**类型：** WebResourceError

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core

## request

```TypeScript
request: WebResourceRequest
```

网页请求的封装信息。

**类型：** WebResourceRequest

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Web.Webview.Core
