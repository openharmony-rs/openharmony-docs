## 概述

ArkWeb_JavaScriptValue是用于在Native代码中封装JavaScript值的结构体，提供了JavaScript值的基本创建和操作能力。该结构体支持将Native数据转换为JavaScript可识别的格式，是JavaScript桥接通信中的数据传递基础类型。
