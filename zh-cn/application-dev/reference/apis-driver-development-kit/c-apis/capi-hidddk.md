# HidDdk

## 概述

Provides HID DDK interfaces, including creating a device, sending an event, and destroying a device.

**系统能力：** SystemCapability.Driver.HID.Extension

**起始版本：** 11

| [hid_ddk_api.h](capi-hid-ddk-api-h.md) | 声明主机侧访问输入设备的HID DDK接口。 |
