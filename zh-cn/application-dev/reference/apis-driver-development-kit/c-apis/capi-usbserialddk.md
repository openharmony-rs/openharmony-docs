# USBSerialDDK

## 概述

Provides USB SERIAL DDK types and declares the macros, enumerated variables, anddata structures required by the USB SERIAL DDK APIs.

**起始版本：** 18
## 文件汇总

| 名称 | 描述 |
| -- | -- |
| [usb_serial_api.h](capi-usb-serial-api-h.md) | 声明用于主机侧通过USB接口访问串口设备的USB Serial DDK接口。 |
| [usb_serial_types.h](capi-usb-serial-types-h.md) | 提供USB Serial DDK中的枚举变量、结构体定义与宏定义。 |
