**起始版本：** 11

## TYPE_LOCAL

```TypeScript
TYPE_LOCAL = 0
```

本地设备，包括设备本身的内置扬声器或音频插孔、A2DP 设备。

**起始版本：** 11

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
