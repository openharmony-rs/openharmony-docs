**起始版本：** 23

## PLAYBACK_STATE_PREPARE

```TypeScript
PLAYBACK_STATE_PREPARE = 0
```

准备中状态。

表示媒体文件尚未准备好播放，媒体正在加载或缓冲中。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。
