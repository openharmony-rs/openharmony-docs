# RenderParameters

## alwaysRender

```TypeScript
alwaysRender?: boolean
```

如果为true，即使场景没有变化也始终渲染自上一帧以来. 如果为false，则场景没有变化时可以省略渲染.

**类型：** boolean

**起始版本：** 15
