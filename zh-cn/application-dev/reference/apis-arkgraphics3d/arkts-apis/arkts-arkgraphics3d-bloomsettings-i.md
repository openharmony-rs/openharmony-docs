# BloomSettings

定义泛光参数.

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## scaleFactor

```TypeScript
scaleFactor?: number
```

缩放因子. 控制缩放和泛光扩散量.
减少降缩放和上缩放步骤.
取值范围0-1. 值为0.5时减半缩放步骤.

**类型：** number

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## scatter

```TypeScript
scatter?: number
```

扩散（泛光扩散量）. (1.0 full spread / default).

**类型：** number

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## thresholdHard

```TypeScript
thresholdHard?: number
```

泛光硬阈值.

**类型：** number

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## thresholdSoft

```TypeScript
thresholdSoft?: number
```

泛光软阈值.

**类型：** number

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D
