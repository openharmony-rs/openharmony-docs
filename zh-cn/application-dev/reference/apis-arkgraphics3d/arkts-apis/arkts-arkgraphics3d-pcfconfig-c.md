# PCFConfig

PCF软阴影配置参数

**继承/实现关系：** PCFConfig extends [SoftShadowConfig](arkts-arkgraphics3d-softshadowconfig-c.md)

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUi.Graphics3D

## shadowSampleCount

```TypeScript
set shadowSampleCount(value: number | undefined)
```

设置用于渲染阴影像素的阴影图采样数量.

**类型：** number

**默认值：** 16

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D

## shadowSampleRadius

```TypeScript
set shadowSampleRadius(value: number | undefined)
```

设置像素级阴影边缘周围的采样半径.

**类型：** number

**默认值：** 5.0

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D
