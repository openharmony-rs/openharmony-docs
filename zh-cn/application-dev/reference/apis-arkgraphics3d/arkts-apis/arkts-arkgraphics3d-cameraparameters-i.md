# CameraParameters

相机创建参数. 可用于定义相机创建的额外选项.

**起始版本：** 21

**系统能力：** SystemCapability.ArkUi.Graphics3D

## msaa

```TypeScript
msaa?: boolean
```

选择是否启用MSAA.

**类型：** boolean

**默认值：** false

**起始版本：** 22

**系统能力：** SystemCapability.ArkUi.Graphics3D

## renderingPipeline

```TypeScript
renderingPipeline?: RenderingPipelineType
```

选择初始渲染管线类型.

**类型：** RenderingPipelineType

**默认值：** RenderingPipelineType.FORWARD_LIGHTWEIGHT 前向轻量级渲染管线

**起始版本：** 21

**系统能力：** SystemCapability.ArkUi.Graphics3D
