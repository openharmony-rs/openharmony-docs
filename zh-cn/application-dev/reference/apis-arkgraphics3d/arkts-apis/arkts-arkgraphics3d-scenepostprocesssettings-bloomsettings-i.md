## scaleFactor

```TypeScript
scaleFactor?: number
```

缩放因子. 控制缩放和泛光扩散量.减少降缩放和上缩放步骤.取值范围0-1. 值为0.5时减半缩放步骤.

**类型：** number

**起始版本：** 18
