# SpotLight

定义聚光灯.

**继承/实现关系：** SpotLight extends [Light](arkts-arkgraphics3d-light-i.md)

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## innerAngle

```TypeScript
innerAngle?: number
```

聚光灯的内角, 单位为弧度.

**类型：** number

**默认值：** 0

**起始版本：** 23

**系统能力：** SystemCapability.ArkUi.Graphics3D

## outerAngle

```TypeScript
outerAngle?: number
```

聚光灯的外角, 单位为弧度.

**类型：** number

**默认值：** PI / 4.0 π/4 弧度

**起始版本：** 23

**系统能力：** SystemCapability.ArkUi.Graphics3D
