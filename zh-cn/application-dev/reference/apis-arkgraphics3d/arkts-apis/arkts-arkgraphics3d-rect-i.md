# Rect

定义矩形.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## height

```TypeScript
height: number
```

矩形的高度, 单位为世界坐标系下的场景单位（例如cm、m、km等）.

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## width

```TypeScript
width: number
```

矩形的宽度, 单位为世界坐标系下的场景单位（例如cm、m、km等）.

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## x

```TypeScript
x: number
```

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D
