# Aabb

轴对齐边界盒.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## aabbMax

```TypeScript
aabbMax: Vec3
```

AABB最大角点的坐标, 单位为世界坐标系下的场景单位（例如cm、m、km等）.

**类型：** Vec3

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

aabbMin: Vec3

**类型：** Vec3
