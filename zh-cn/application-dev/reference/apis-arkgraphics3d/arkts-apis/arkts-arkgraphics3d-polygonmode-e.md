# PolygonMode

多边形模式枚举.

**起始版本：** 23

**系统能力：** SystemCapability.ArkUi.Graphics3D

## FILL

```TypeScript
FILL = 0
```

渲染整个多边形

**起始版本：** 23

**系统能力：** SystemCapability.ArkUi.Graphics3D

## LINE

```TypeScript
LINE = 1
```

仅渲染多边形的边（线框）

**起始版本：** 23

**系统能力：** SystemCapability.ArkUi.Graphics3D

## POINT

```TypeScript
POINT = 2
```

仅渲染多边形的顶点

**起始版本：** 23

**系统能力：** SystemCapability.ArkUi.Graphics3D
