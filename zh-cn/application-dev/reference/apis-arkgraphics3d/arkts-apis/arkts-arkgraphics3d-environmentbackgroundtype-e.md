# EnvironmentBackgroundType

环境背景类型枚举.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## BACKGROUND_NONE

```TypeScript
BACKGROUND_NONE = 0
```

背景为空.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## BACKGROUND_IMAGE

```TypeScript
BACKGROUND_IMAGE = 1
```

背景为图像.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## BACKGROUND_CUBEMAP

```TypeScript
BACKGROUND_CUBEMAP = 2
```

背景为立方体贴图.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## BACKGROUND_EQUIRECTANGULAR

```TypeScript
BACKGROUND_EQUIRECTANGULAR = 3
```

背景为等距柱状投影.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D
