# SphereGeometry

定义球体.

**继承/实现关系：** SphereGeometry extends [GeometryDefinition](arkts-arkgraphics3d-geometrydefinition-c.md)

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## radius

```TypeScript
set radius(value: number)
```

球体的半径, 单位为世界坐标系下的场景单位（例如cm、m、km等）.

**类型：** number

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## segmentCount

```TypeScript
set segmentCount(value: number)
```

将球体按经纬度分割成若干圈和段.

**类型：** number

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D
