# ASON

**起始版本：** 12

**系统能力：** SystemCapability.Utils.Lang

| 名称 | 说明 |
| --- | --- |
| [parse](arkts-arkts-ason-parse-f.md#parse-1) | 将JavaScript对象表示法（JSON）字符串转换为ArkTS值。<br/> |
| [stringify](arkts-arkts-ason-stringify-f.md#stringify-1) | 将ArkTS值转换为JavaScript对象表示法（JSON）字符串。<br/>额外支持Map和Set。<br/> |
