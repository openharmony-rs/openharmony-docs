import { TreeMap } from '@kit.ArkTS';

## 汇总

### 类

| 名称 | 说明 |
| --- | --- |
| [TreeMap](arkts-arkts-util-treemap-treemap-c.md) | TreeMap可用于存储具有关联关系的key-value键值对集合，存储元素中key值唯一，每个key对应一个value。TreeMap底层使用红黑树实现，可以利用二叉树特性快速查找键值对。key值有序存储，可以实现快速的插入和删除。 |
