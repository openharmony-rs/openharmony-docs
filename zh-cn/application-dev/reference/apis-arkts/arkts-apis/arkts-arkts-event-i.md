# Event

事件类。

**起始版本：** 7

**系统能力：** SystemCapability.Utils.Lang

## timeStamp

```TypeScript
readonly timeStamp: number
```

事件创建时的时间戳（精度为毫秒），目前不支持。

**类型：** number

**起始版本：** 7

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Utils.Lang

## type

```TypeScript
readonly type: string
```

指定事件的类型。

**类型：** string

**起始版本：** 7

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Utils.Lang
