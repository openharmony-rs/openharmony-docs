# CryptoMode

```TypeScript
enum CryptoMode
```

表示加解密操作的枚举。

**起始版本：** 9

**系统能力：** SystemCapability.Security.CryptoFramework.Cipher

## ENCRYPT_MODE

```TypeScript
ENCRYPT_MODE = 0
```

表示进行加密操作。

**起始版本：** 9

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.CryptoFramework.Cipher

## DECRYPT_MODE

```TypeScript
DECRYPT_MODE = 1
```

表示进行解密操作。

**起始版本：** 9

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.CryptoFramework.Cipher
