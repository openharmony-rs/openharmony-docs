# ECField

指定椭圆曲线的域类型。当前只支持Fp域。

**起始版本：** 10

**系统能力：** 
- API版本12+：SystemCapability.Security.CryptoFramework.Key.AsymKey
- API版本10-11：SystemCapability.Security.CryptoFramework

## fieldType

```TypeScript
fieldType: string
```

指定椭圆曲线域的类型，当前只支持"Fp"。

**类型：** string

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** 
- API版本12+：SystemCapability.Security.CryptoFramework.Key.AsymKey
- API版本10-11：SystemCapability.Security.CryptoFramework
