# DataBlob

二进制数据的封装接口，核心字段data为Uint8Array类型。

> **说明：**
>
> Uint8Array类型数据表示8位无符号整数的数组。

**起始版本：** 9

**系统能力：** SystemCapability.Security.CryptoFramework

## data

```TypeScript
data: Uint8Array
```

数据。

**类型：** Uint8Array

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.CryptoFramework
