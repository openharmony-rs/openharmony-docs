# MacSpec

消息认证码参数，计算HMAC、CMAC消息认证码时，需要构建子类对象并作为输入参数。

> **说明：**
>
> algName是必选参数，表示消息验证码算法。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CryptoFramework.Mac

## algName

```TypeScript
algName: string
```

消息验证码算法名。

**类型：** string

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.CryptoFramework.Mac
