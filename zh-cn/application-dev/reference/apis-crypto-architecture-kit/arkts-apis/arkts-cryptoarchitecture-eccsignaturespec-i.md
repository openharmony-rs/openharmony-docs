# EccSignatureSpec

包含（r、s）的ECC/SM2签名数据的对象。

> **说明：**
>
> r和s的长度各为256位。

**起始版本：** 20

**系统能力：** SystemCapability.Security.CryptoFramework.Signature

```TypeScript
r: bigint
```

r分量。
