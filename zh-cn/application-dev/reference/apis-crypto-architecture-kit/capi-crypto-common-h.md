```

```

**描述**

**起始版本：** 12

| 枚举项 | 描述 |
| -- | -- |
| CRYPTO_SUCCESS = 0 | 表示操作成功。 |
| CRYPTO_INVALID_PARAMS = 401 | 输入参数不合法。 |
| CRYPTO_NOT_SUPPORTED = 801 | 不支持的函数或算法。 |
| CRYPTO_MEMORY_ERROR = 17620001 | 内存错误。 |
