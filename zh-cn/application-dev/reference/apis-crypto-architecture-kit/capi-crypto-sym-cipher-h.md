## 枚举类型说明

### CryptoSymCipher_ParamsType

```
enum CryptoSymCipher_ParamsType
```

**描述**

定义对称加解密参数类型。

**起始版本：** 12

| 枚举项 | 描述 |
| -- | -- |
| CRYPTO_IV_DATABLOB = 100 | 表示iv等参数。 |
| CRYPTO_AAD_DATABLOB = 101 | 表示GCM模式下的附加认证数据。 |
| CRYPTO_TAG_DATABLOB = 102 | 表示加密操作的输出标签,用于完整性检查。 |
