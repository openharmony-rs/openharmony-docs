# input

## 概述

Provides the C interface in the multi-modal input domain.

**起始版本：** 22
## 文件汇总

| 名称 | 描述 |
| -- | -- |
| [oh_pointer_style.h](capi-oh-pointer-style-h.md) | Defines the mouse pointer styles. |
| [oh_key_code.h](capi-oh-key-code-h.md) | Defines the key event structure and related enumeration values. |
| [oh_input_manager.h](capi-oh-input-manager-h.md) | Provides functions such as input event injection, key state query, device hot swapping listener, eventinterception, hotkey management, mouse cursor management, input device information query, and injection permissionmanagement. |
| [oh_axis_type.h](capi-oh-axis-type-h.md) | Defines the device axis event struct and enumerates device axis events. The axis type defines the physicalbehavior characteristics of an input device in different interaction scenarios. The system uses the axis type todistinguish and transmit different gesture interaction information. |
