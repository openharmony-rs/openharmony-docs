# @ohos.multimodalInput.shortKey

通过本模块接口，可以设置快捷键拉起Ability的延迟时间，如设置长按快捷键3s后再截屏等。
> **说明：**  
>  
> - 本模块接口为系统接口。

## 导入模块

```TypeScript
import { shortKey } from '@kit.InputKit';
```
