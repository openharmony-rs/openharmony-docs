# AxisType

```TypeScript
type AxisType =
    'touchmajor'
    | 'touchminor'
    | 'orientation'
    | 'x'
    | 'y'
    | 'pressure'
    | 'toolminor'
    | 'toolmajor'
    | 'null'
```

输入设备的轴类型。

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

| 类型 | 说明 |
| --- | --- |
| 'touchmajor' | 椭圆触摸区域长轴。 |
| 'touchminor' | 椭圆触摸区域短轴。 |
| 'orientation' | 方向轴。 |
| 'x' | 横坐标轴。 |
| 'y' | 纵坐标轴。 |
| 'pressure' | 压力轴。 |
| 'toolminor' | 工具区域短轴。 |
| 'toolmajor' | 工具区域长轴。 |
| 'null' | 无类型。 |
