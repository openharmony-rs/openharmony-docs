# AxisRange

输入设备的轴信息。

**起始版本：** 8

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## axis

```TypeScript
axis: AxisType
```

输入设备的轴类型。

**类型：** AxisType

**起始版本：** 8

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## flat

```TypeScript
flat: number
```

轴的基准值。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## fuzz

```TypeScript
fuzz: number
```

轴的模糊值。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## max

```TypeScript
max: number
```

轴的最大值。

**类型：** number

**起始版本：** 8

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## min

```TypeScript
min: number
```

轴的最小值。

**类型：** number

**起始版本：** 8

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## resolution

```TypeScript
resolution: number
```

轴的分辨率。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## source

```TypeScript
source: SourceType
```

输入设备的输入能力。包括键盘、鼠标、触摸屏、轨迹球、触控板、操纵杆等。

**类型：** SourceType

**起始版本：** 8

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice
