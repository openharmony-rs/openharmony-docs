# DeviceListener

描述输入设备热插拔的信息。

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## deviceId

```TypeScript
deviceId: number
```

输入设备的唯一标识，同一个物理设备反复插拔或重启，设备ID可能会发生变化。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice

## type

```TypeScript
type: ChangedType
```

输入设备插入或者移除。

**类型：** ChangedType

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.InputDevice
