# KeyEvent

按键事件。

**继承/实现关系：** KeyEvent extends [InputEvent](arkts-input-inputevent-i.md)

**起始版本：** 9

**系统能力：** SystemCapability.MultimodalInput.Input.Core

## action

```TypeScript
action: Action
```

按键事件类型。

**类型：** Action

**起始版本：** 9

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.MultimodalInput.Input.Core

## altKey

```TypeScript
altKey: boolean
```

当前altKey是否处于按下状态。

true表示处于按下状态，false表示处于抬起状态。

**类型：** boolean

**起始版本：** 9

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.MultimodalInput.Input.Core
