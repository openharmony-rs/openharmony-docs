## 1000000 模型路径错误

**错误信息**

Model Path error.

**ArkTS模式：** 该错误码仅适用于ArkTS-Sta。

**错误描述**

模型路径错误。

**可能原因**

1. 模型路径为空。
2. 模型路径不存在。

**处理步骤**

1. 请确保模型路径不为空。
2. 请确保模型路径存在。


## 1000001 无效的上下文

**错误信息**

Invalid context.

**ArkTS模式：** 该错误码仅适用于ArkTS-Sta。

**错误描述**

无效的上下文。

**可能原因**

1. 上下文目标错误。
2. 设备信息错误。

**处理步骤**

1. 请确保上下文目标正确。
2. 请确保设备信息正确。


## 1000002 从路径创建模型失败

**错误信息**

Failed to create native model.

**ArkTS模式：** 该错误码仅适用于ArkTS-Sta。

**错误描述**

从路径创建模型失败。

**可能原因**

1. 访问模型路径的权限不足。
2. 模型文件已损坏。

**处理步骤**

1. 请确保有权限访问模型路径。
2. 请确保模型文件未损坏。


## 1000003 加载模型方式错误

**错误信息**

Error in model loading method.

**ArkTS模式：** 该错误码仅适用于ArkTS-Sta。

**错误描述**

加载模型方式错误。

**可能原因**

加载模型方式不是路径、缓冲区或者fd。

**处理步骤**

请确保加载模型的方式只能是路径、缓冲区或者fd。


## 1000004 模型缓冲区错误

**ArkTS模式：** 该错误码仅适用于ArkTS-Sta。

**错误描述**

模型缓冲区错误。

**可能原因**

1. 缓冲区大小为0。

**处理步骤**
