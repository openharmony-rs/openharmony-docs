## 概述

数据防泄漏（DLP）是系统级的数据防泄漏解决方案，提供跨设备文件权限管理、加密存储、授权访问等能力。

 **使用场景**：
 - 企业敏感文档保护
 - 跨设备安全文件共享
 - 数据合规性管理

 **解决的问题**： 
 - 防止敏感数据泄露
 - 避免未授权访问
 - 跨设备数据传输安全

 **带来的收益**： 
 - 保障企业数据安全
 - 满足数据合规要求
 - 提升数据管理效率

**支持设备类型**：手机、平板和2in1。

本模块首批接口从API version 14开始支持。
