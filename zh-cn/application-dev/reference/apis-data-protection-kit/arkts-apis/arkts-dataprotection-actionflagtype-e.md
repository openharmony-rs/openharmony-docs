# ActionFlagType

可以对DLP文件进行的操作类型枚举。例如：DLP沙箱应用可以根据是否具有操作权限，对其按钮进行置灰。

**起始版本：** 10

**系统能力：** SystemCapability.Security.DataLossPrevention

## ACTION_VIEW

```TypeScript
ACTION_VIEW = 0x00000001
```

表示文件的查看权限。

**起始版本：** 10

**系统能力：** SystemCapability.Security.DataLossPrevention
