# StatusInfoResult（系统接口）

DLP特性开关状态设置的结果信息。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。

## isSuccess

```TypeScript
isSuccess: boolean
```

DLP特性开关状态是否设置成功。true表示设置成功；false表示设置失败。

**类型：** boolean

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。
