# DLPFeatureInfo（系统接口）

DLP特性开关的状态信息。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。

## dlpFeatureStatus

```TypeScript
dlpFeatureStatus: DlpFeatureStatus
```

DLP特性开关的状态，包含NOT_ENABLED_FEATURE和ENABLED_FEATURE。

**类型：** DlpFeatureStatus

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。
