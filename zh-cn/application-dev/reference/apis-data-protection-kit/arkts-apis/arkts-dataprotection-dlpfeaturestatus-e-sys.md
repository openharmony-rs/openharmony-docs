# DlpFeatureStatus（系统接口）

DLP特性开关状态的枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。

## NOT_ENABLED_FEATURE

```TypeScript
NOT_ENABLED_FEATURE = 0
```

表示关闭DLP特性开关。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。

## ENABLED_FEATURE

```TypeScript
ENABLED_FEATURE = 1
```

表示开启DLP特性开关。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。
