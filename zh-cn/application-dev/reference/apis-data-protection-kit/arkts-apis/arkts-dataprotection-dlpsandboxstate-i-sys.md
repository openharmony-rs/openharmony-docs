# DLPSandboxState（系统接口）

DLP沙箱的状态信息。

**起始版本：** 10

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。

## appIndex

```TypeScript
appIndex: number
```

表示DLP沙箱应用索引。

**类型：** number

**起始版本：** 10

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。

## bundleName

```TypeScript
bundleName: string
```

表示应用包名。最小7字节，最大128字节。超出此范围抛出错误码19100001。

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Security.DataLossPrevention

**系统接口：** 此接口为系统接口。
