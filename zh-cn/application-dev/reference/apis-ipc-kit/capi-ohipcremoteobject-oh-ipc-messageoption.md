# OH_IPC_MessageOption

IPC消息选项定义。

**起始版本：** 12

**相关模块：** [OHIPCRemoteOhject](capi-ohipcremoteobject.md)

**所在头文件：** [ipc_cremote_object.h](capi-ipc-cremote-object-h.md)

## 汇总

| 名称 | 描述 |
| ---- | ---- |
| OH_IPC_RequestMode mode | 消息请求模式。 |
