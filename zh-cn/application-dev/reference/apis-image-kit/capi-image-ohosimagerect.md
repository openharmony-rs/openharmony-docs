## 概述

定义图像矩形信息。

**起始版本：** 10

**相关模块：** [Image](capi-image.md)

**所在头文件：** [image_mdk.h](capi-image-mdk-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| int32_t x | 矩形x坐标值。 |
| int32_t y | 矩形y坐标值。 |
| int32_t width | 矩形宽度值，用pixels表示。 |
| int32_t height | 矩形高度值，用pixels表示。 |
