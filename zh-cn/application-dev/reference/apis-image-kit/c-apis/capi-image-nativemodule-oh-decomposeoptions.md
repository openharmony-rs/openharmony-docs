# OH_DecomposeOptions

```c
typedef struct OH_DecomposeOptions OH_DecomposeOptions
```

## 概述

**OH_DecomposeOptions** is the HDR decomposition option struct encapsulated at the native layer. It isused to specify parameters used for HDR decomposition, such as the target pixel format.

**起始版本：** 26.0.0

**相关模块：** [Image_NativeModule](capi-image-nativemodule.md)
