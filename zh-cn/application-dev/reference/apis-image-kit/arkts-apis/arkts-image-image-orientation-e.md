# Orientation

表示图像方向类型的枚举。

**起始版本：** 23

<!--Device-image-enum Orientation--><!--Device-image-enum Orientation-End-->

**系统能力：** SystemCapability.Multimedia.Image.Core

## TOP_LEFT

```TypeScript
TOP_LEFT = 1
```

图像未旋转。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

<!--Device-Orientation-TOP_LEFT = 1--><!--Device-Orientation-TOP_LEFT = 1-End-->

**系统能力：** SystemCapability.Multimedia.Image.Core
