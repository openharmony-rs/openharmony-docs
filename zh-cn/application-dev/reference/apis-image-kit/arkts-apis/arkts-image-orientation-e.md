# Orientation

表示图像方向类型的枚举。

**起始版本：** 23

**系统能力：** SystemCapability.Multimedia.Image.Core

## TOP_LEFT

```TypeScript
TOP_LEFT = 1
```

图像未旋转。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Multimedia.Image.Core

## TOP_RIGHT

```TypeScript
TOP_RIGHT = 2
```

图像是水平镜像的。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Multimedia.Image.Core

## BOTTOM_RIGHT

```TypeScript
BOTTOM_RIGHT = 3
```

图像旋转180度。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Multimedia.Image.Core
