## FIT_TARGET_SIZE

```TypeScript
FIT_TARGET_SIZE = 0
```

图像适合目标尺寸的效果。

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。

## CENTER_CROP

```TypeScript
CENTER_CROP = 1
```

**起始版本：** 9

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。
