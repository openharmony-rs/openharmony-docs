# ImagePropertyOptions

**系统能力：** SystemCapability.Multimedia.Image.ImageSource

## defaultValue

```TypeScript
defaultValue?: string
```

默认属性值。默认值为空。

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Image.ImageSource

## index

```TypeScript
index?: number
```

**类型：** number

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Image.ImageSource
