# FragmentMapPropertyKey

表示水印裁剪图图片信息的枚举。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Image.Core

## X_IN_ORIGINAL

```TypeScript
X_IN_ORIGINAL = 'XInOriginal'
```

水印裁剪图左上角在原始图中的X坐标。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Image.Core

## Y_IN_ORIGINAL

```TypeScript
Y_IN_ORIGINAL = 'YInOriginal'
```

水印裁剪图左上角在原始图中的Y坐标。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Image.Core

## WIDTH

```TypeScript
WIDTH = 'FragmentImageWidth'
```

水印裁剪图的宽。单位：像素（px）。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Image.Core

## HEIGHT

```TypeScript
HEIGHT = 'FragmentImageHeight'
```

水印裁剪图的高。单位：像素（px）。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Image.Core
