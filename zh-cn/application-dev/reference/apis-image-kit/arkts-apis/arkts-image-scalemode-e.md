# ScaleMode

表示图像的缩放模式的枚举。

**起始版本：** 9

**系统能力：** SystemCapability.Multimedia.Image.Core

## FIT_TARGET_SIZE

```TypeScript
FIT_TARGET_SIZE = 0
```

图像适合目标尺寸的效果。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.Multimedia.Image.Core

## CENTER_CROP

```TypeScript
CENTER_CROP = 1
```

缩放图像以填充目标图像区域并居中裁剪区域外的效果。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**卡片能力：** 从API版本12开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.Multimedia.Image.Core
