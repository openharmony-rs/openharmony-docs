# AvisPropertyKey

表示AVIS图片信息的枚举。

> **说明：**
>
> 返回字段类型具体参考[AvisMetadata](../../../../reference/apis-image-kit/arkts-apis-image-AvisMetadata.md)。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Multimedia.Image.Core

## DELAY_TIME

```TypeScript
DELAY_TIME = 'AvisDelayTime'
```

AVIS图片的每帧播放时长。

单位：毫秒（ms）。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Multimedia.Image.Core
