# DocumentSelectMode

Enumerates the types of documents selected.

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.UserFileService.FolderSelection

## FILE

```TypeScript
FILE = 0
```

文件类型（默认类型）。

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.FileManagement.UserFileService.FolderSelection
