# FileInfo（系统接口）

最近访问列表文件信息。

**起始版本：** 10

**废弃版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.FileManagement.UserFileService

**系统接口：** 此接口为系统接口。

## ctime

```TypeScript
readonly ctime: number
```

文件的创建时间。自1970年1月1日起至目标时间的秒数。

**类型：** number

**起始版本：** 10

**废弃版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.FileManagement.UserFileService

**系统接口：** 此接口为系统接口。
