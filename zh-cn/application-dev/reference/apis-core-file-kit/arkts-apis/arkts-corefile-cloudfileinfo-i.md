# CloudFileInfo

应用本地和云端文件个数以及大小信息。

**起始版本：** 20

**系统能力：** SystemCapability.FileManagement.DistributedFileService.CloudSyncManager

## bothFileCount

```TypeScript
bothFileCount: number
```

本地已上传云端的文件总个数，取值范围[0, INT32_MAX]，单位：个。

**类型：** number

**起始版本：** 20

**系统能力：** SystemCapability.FileManagement.DistributedFileService.CloudSyncManager

## bothFileTotalSize

```TypeScript
bothFileTotalSize: number
```

本地已上传云端的文件总大小，取值范围[0, INT64_MAX]，单位：Byte。

**类型：** number

**起始版本：** 20

**系统能力：** SystemCapability.FileManagement.DistributedFileService.CloudSyncManager
