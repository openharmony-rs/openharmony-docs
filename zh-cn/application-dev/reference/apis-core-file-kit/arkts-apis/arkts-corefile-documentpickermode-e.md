# DocumentPickerMode

Enumerates the modes for saving documents.

**起始版本：** 12

**系统能力：** SystemCapability.FileManagement.UserFileService

## DEFAULT

```TypeScript
DEFAULT = 0
```

标准模式。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.FileManagement.UserFileService

## DOWNLOAD

```TypeScript
DOWNLOAD = 1
```

下载模式。

**注意**： DOWNLOAD模式创建的目录仅用于保存文件，目录之间无访问隔离，不建议保存应用敏感数据。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.FileManagement.UserFileService
