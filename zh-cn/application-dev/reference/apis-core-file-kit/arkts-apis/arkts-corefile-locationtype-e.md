# LocationType

枚举，文件位置，表示该文件是否在本地或者云端存在。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO

## LOCAL

```TypeScript
LOCAL = 1 << 0
```

文件在本地存在。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO

## CLOUD

```TypeScript
CLOUD = 1 << 1
```

文件在云端存在。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO
