# @ohos.statfs

该模块提供文件系统相关存储信息的功能，向应用程序提供获取文件系统总字节数、空闲字节数的JS接口。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** [statvfs:statfs](arkts-file-statvfs.md)
