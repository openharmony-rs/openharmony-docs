# @ohos.file.cloudDiskManager

本模块是为系统文件管理应用提供获取三方网盘注册的同步根信息的能力。

**起始版本：** 21

**系统能力：** SystemCapability.FileManagement.CloudDiskManager

**系统接口：** 此接口为系统接口。
