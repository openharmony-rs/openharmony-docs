# DeviceType（系统接口）

组成RootInfo中的deviceType字段，表明此设备的类型。

**起始版本：** 9

**废弃版本：** 23

**系统能力：** SystemCapability.FileManagement.UserFileService

**系统接口：** 此接口为系统接口。

## DEVICE_LOCAL_DISK

```TypeScript
DEVICE_LOCAL_DISK = 1
```

本地c,d...盘。

**起始版本：** 9

**废弃版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.FileManagement.UserFileService

**系统接口：** 此接口为系统接口。
