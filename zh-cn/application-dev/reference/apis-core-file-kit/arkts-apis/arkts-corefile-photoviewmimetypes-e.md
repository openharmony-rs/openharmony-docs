# PhotoViewMIMETypes

Enumerates the media file types that can be selected.

**起始版本：** 9

**废弃版本：** 12

**替代接口：** PhotoViewMIMETypes

**系统能力：** SystemCapability.FileManagement.UserFileService

## IMAGE_TYPE

```TypeScript
IMAGE_TYPE = 'image/*'
```

图片类型。

**起始版本：** 9

**废弃版本：** 18

**替代接口：** IMAGE_TYPE

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.FileManagement.UserFileService
