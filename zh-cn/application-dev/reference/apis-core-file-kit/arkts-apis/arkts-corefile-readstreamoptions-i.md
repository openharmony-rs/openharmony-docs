# ReadStreamOptions

可选项类型，支持 createReadStream 接口使用。

**起始版本：** 12

**系统能力：** SystemCapability.FileManagement.File.FileIO

## end

```TypeScript
end?: number
```

表示期望读取结束的位置，单位为Byte。可选，默认文件末尾。

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.FileManagement.File.FileIO

## start

```TypeScript
start?: number
```

表示期望读取文件的位置，单位为Byte。可选，默认从当前位置开始读。

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.FileManagement.File.FileIO
