# fileExtensionInfo

该模块提供公共文件访问和管理模块中RootInfo与FileInfo中部分属性值的定义。

**起始版本：** 9

**废弃版本：** 23

**系统能力：** SystemCapability.FileManagement.UserFileService

**系统接口：** 此接口为系统接口。

## 汇总

### 命名空间

| 名称 | 说明 |
| --- | --- |
| [DeviceFlag](arkts-corefile-fileextensioninfo-deviceflag-sta-n.md) | 组成RootInfo中的deviceFlags字段，通过与运算判断是否具备某种能力。<br/> |
| [DocumentFlag](arkts-corefile-fileextensioninfo-documentflag-sta-n.md) | 组成FileInfo中的mode字段，通过与运算判断是否具备某种能力。<br/> |
