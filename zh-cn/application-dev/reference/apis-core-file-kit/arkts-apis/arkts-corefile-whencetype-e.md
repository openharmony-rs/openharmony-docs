# WhenceType

枚举，文件偏移指针相对偏移位置类型，支持lseek接口使用。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO

## SEEK_SET

```TypeScript
SEEK_SET = 0
```

文件起始位置处。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO

## SEEK_CUR

```TypeScript
SEEK_CUR = 1
```

当前文件偏移指针位置处。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO

## SEEK_END

```TypeScript
SEEK_END = 2
```

文件末尾位置处。

**起始版本：** 11

**系统能力：** SystemCapability.FileManagement.File.FileIO
