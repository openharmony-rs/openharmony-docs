# RgbaColor

定义壁纸颜色信息结构。

**起始版本：** 7

**废弃版本：** 9

**系统能力：** SystemCapability.MiscServices.Wallpaper

## alpha

```TypeScript
alpha: number
```

表示 alpha 值，范围为 0 到 255。

**类型：** number

**起始版本：** 7

**废弃版本：** 9

**系统能力：** SystemCapability.MiscServices.Wallpaper
