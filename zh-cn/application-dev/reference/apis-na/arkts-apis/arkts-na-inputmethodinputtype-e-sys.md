# InputMethodInputType（系统接口）

定义输入类型。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**系统接口：** 此接口为系统接口。

## NONE

```TypeScript
NONE = -1
```

无输入类型，面板不在任何输入类型中。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**系统接口：** 此接口为系统接口。
