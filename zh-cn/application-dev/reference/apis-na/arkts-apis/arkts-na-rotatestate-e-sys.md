# RotateState（系统接口）

定义设备的横竖屏状态枚举类型。

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。

## PORTRAIT

```TypeScript
PORTRAIT = 0
```

设备默认为竖屏状态。

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。

## LANDSCAPE

```TypeScript
LANDSCAPE = 1
```

横屏状态。

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。
