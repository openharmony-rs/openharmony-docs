# WallpaperInfo（系统接口）

定义壁纸的信息结构。

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。

## foldState

```TypeScript
foldState: FoldState
```

表示设备的折展状态。

**类型：** FoldState

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。

## rotateState

```TypeScript
rotateState: RotateState
```

表示设备的横竖屏状态。

**类型：** RotateState

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。

## source

```TypeScript
source: string
```

表示壁纸资源uri，只支持应用沙箱目录。

**类型：** string

**起始版本：** 14

**系统能力：** SystemCapability.MiscServices.Wallpaper

**系统接口：** 此接口为系统接口。
