# OH_Rect

```c
typedef struct OH_Rect {...} OH_Rect
```

## 概述

定义录屏界面的宽高以及画面信息。

**起始版本：** 10

**相关模块：** [AVScreenCapture](capi-avscreencapture.md)

**所在头文件：** [native_avscreen_capture_base.h](capi-native-avscreen-capture-base-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| int32_t x | X-coordinate of screen recording |
| int32_t y | y-coordinate of screen recording |
| int32_t width | Width of screen recording, in px |
| int32_t height | Height of screen recording, in px |
