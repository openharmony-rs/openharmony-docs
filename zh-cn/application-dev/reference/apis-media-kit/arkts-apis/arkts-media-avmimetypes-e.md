# AVMimeTypes

媒体MIME类型，通过[setMimeType](@ohos.multimedia.media:media.MediaSource.setMimeType)设置。

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Media.Core

## APPLICATION_M3U8

```TypeScript
APPLICATION_M3U8 = 'application/m3u8'
```

表示m3u8本地文件。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.Core
