# MediaType

媒体类型枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Media.Core

## MEDIA_TYPE_UNSUPPORTED

```TypeScript
MEDIA_TYPE_UNSUPPORTED = -1
```

表示未支持的类型。

**原子化服务API：** 从API version 20 开始，该接口支持在原子化服务中使用。

**起始版本：** 20

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.Core
