# PlaybackSpeed

视频播放的倍速枚举，可通过setSpeed方法作为参数传递下去。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Media.VideoPlayer

## SPEED_FORWARD_0_75_X

```TypeScript
SPEED_FORWARD_0_75_X = 0
```

表示视频播放正常播速的0.75倍。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.VideoPlayer
