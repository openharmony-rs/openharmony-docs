# PlaylistLoopMode

表示播放列表循环模式的枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Multimedia.Media.Core

## PLAYLIST_LOOP_MODE_ALL

```TypeScript
PLAYLIST_LOOP_MODE_ALL = 1
```

列表循环模式。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.Core
