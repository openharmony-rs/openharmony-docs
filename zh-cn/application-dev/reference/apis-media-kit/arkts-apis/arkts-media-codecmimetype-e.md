# CodecMimeType

Codec MIME类型枚举。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Media.Core

## VIDEO_H263

```TypeScript
VIDEO_H263 = 'video/h263'
```

表示视频/h263类型。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Media.Core

## VIDEO_AVC

```TypeScript
VIDEO_AVC = 'video/avc'
```

表示视频/avc类型。

**起始版本：** 8

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。
