# MediaErrorCode

媒体服务错误类型枚举。

> **说明：**  
> > 从API version 8开始支持，从API version 11开始废弃，建议使用[AVErrorCode](arkts-media-media-averrorcode-e.md)替代。

**起始版本：** 8

**废弃版本：** 11

**替代接口：** [AVErrorCode](arkts-media-media-averrorcode-e.md)
