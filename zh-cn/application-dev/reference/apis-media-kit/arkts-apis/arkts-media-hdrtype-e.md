# HdrType

表示视频HDR类型的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Media.Core

## AV_HDR_TYPE_NONE

```TypeScript
AV_HDR_TYPE_NONE = 0
```

表示无HDR类型。

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Media.Core

## AV_HDR_TYPE_VIVID

```TypeScript
AV_HDR_TYPE_VIVID = 1
```

表示为HDR VIVID类型。

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Media.Core
