# ContainerFormatType

表示容器格式类型的枚举，缩写为CFT。

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Media.Core

## CFT_MPEG_4

```TypeScript
CFT_MPEG_4 = 'mp4'
```

视频的容器格式，MP4。

**起始版本：** 8

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.Core

## CFT_MPEG_4A

```TypeScript
CFT_MPEG_4A = 'm4a'
```

音频的容器格式，M4A。

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.Core
