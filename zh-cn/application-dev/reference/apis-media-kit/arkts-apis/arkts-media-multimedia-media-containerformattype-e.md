# ContainerFormatType

表示容器格式类型的枚举，缩写为CFT。

**起始版本：** 8

## CFT_MPEG_4

```TypeScript
CFT_MPEG_4 = 'mp4'
```

视频的容器格式，MP4。

**起始版本：** 8

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
