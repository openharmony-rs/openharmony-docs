# MediaStream

Media Stream. AVPlayer use this for mediaData access, current version only support live stream.

**起始版本：** 19

**系统能力：** SystemCapability.Multimedia.Media.Core

## bitrate

```TypeScript
bitrate: number
```

biterate of this mediaStream.

**类型：** number

**起始版本：** 19

**元服务API：** 从API版本19开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.Core
