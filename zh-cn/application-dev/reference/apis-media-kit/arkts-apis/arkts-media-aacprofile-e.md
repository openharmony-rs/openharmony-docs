# AacProfile

高级音频编码（AAC）类型枚举。

**起始版本：** 22

**系统能力：** SystemCapability.Multimedia.Media.AVRecorder

## AAC_LC

```TypeScript
AAC_LC = 0
```

表示AAC Low-Complexity类型。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.AVRecorder

## AAC_HE

```TypeScript
AAC_HE = 1
```

表示AAC High-Efficiency类型。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.AVRecorder

## AAC_HE_V2

```TypeScript
AAC_HE_V2 = 2
```

表示AAC High-Efficiency version 2类型。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.AVRecorder
