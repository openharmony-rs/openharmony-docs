# OnPlaybackRateDone

```TypeScript
type OnPlaybackRateDone = (rate: number) => void
```

播放速率设置完成事件回调方法。

**起始版本：** 20

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Multimedia.Media.AVPlayer

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| rate | double | 是 | 播放速率。 |
