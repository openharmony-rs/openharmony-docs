# WrapType

枚举，关键资产支持的加密导入导出类型。

**起始版本：** 18

**系统能力：** SystemCapability.Security.Asset

## NEVER

```TypeScript
NEVER = 0
```

不允许加密导入导出关键资产。

**起始版本：** 18

**系统能力：** SystemCapability.Security.Asset

## TRUSTED_ACCOUNT

```TypeScript
TRUSTED_ACCOUNT = 1
```

只在登录可信账号的设备进行加密导入导出关键资产。

**起始版本：** 18

**系统能力：** SystemCapability.Security.Asset
