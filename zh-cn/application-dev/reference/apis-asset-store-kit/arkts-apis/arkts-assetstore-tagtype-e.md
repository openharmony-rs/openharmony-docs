# TagType

枚举，关键资产属性支持的数据类型。

**起始版本：** 11

**系统能力：** SystemCapability.Security.Asset

## BOOL

```TypeScript
BOOL = 0x01 << 28
```

标识关键资产属性对应的数据类型是布尔。

**起始版本：** 11

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Asset

## NUMBER

```TypeScript
NUMBER = 0x02 << 28
```

标识关键资产属性对应的数据类型是整型。

**起始版本：** 11

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Asset

## BYTES

```TypeScript
BYTES = 0x03 << 28
```

标识关键资产属性对应的数据类型是字节数组。

**起始版本：** 11

**元服务API：** 从API版本14开始，该接口支持在元服务API中使用。
