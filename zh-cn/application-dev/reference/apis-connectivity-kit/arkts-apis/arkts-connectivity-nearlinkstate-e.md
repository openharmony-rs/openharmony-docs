# NearlinkState

星闪状态的枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Communication.NearLink.Base

## STATE_TURNING_ON

```TypeScript
STATE_TURNING_ON = 0
```

表示星闪正在开启。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Communication.NearLink.Base

## STATE_ON

```TypeScript
STATE_ON = 1
```

表示星闪已开启，可供使用。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Communication.NearLink.Base

## STATE_TURNING_OFF

```TypeScript
STATE_TURNING_OFF = 2
```

表示星闪正在关闭。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Communication.NearLink.Base
