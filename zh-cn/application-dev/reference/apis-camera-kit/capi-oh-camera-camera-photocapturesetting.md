## 概述

要设置的拍照捕获选项。

**起始版本：** 11

**相关模块：** [OH_Camera](capi-oh-camera.md)

**所在头文件：** [camera.h](capi-camera-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| [Camera_QualityLevel](capi-camera-h.md#camera_qualitylevel) quality | 拍照图像质量。 |
| [Camera_ImageRotation](capi-camera-h.md#camera_imagerotation) rotation | 拍照旋转角度。 |
| [Camera_Location](capi-oh-camera-camera-location.md)* location | 拍照位置。 |
| bool mirror | 设置镜像拍照功能开关。<br> true为打开，false为关闭，默认为false。 |
