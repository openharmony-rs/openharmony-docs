## ProvisionRequest

设备证书请求。

**原子化服务API：** 从API version 14开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Multimedia.Drm.Core

| 名称      | 类型                        | 只读 | 可选 | 说明         |
| -------- | ------------------------|----|----| ------------- |
| data   | Uint8Array| 否  | 否  | 设备证书请求数据。      |
| defaultURL     | string         | 否  | 否  | Provision服务（设备证书请求服务）URL。       |

| 名称      | 类型                        | 只读 | 可选 | 说明         |

| value     | string             | 否  | 否  | 可选数据值。 |
