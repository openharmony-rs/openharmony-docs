# ServiceProfileInfo（系统接口）

服务配置信息。根据云端返回的数据填充。

**起始版本：** 15

**系统能力：** SystemCapability.DistributedHardware.DeviceManager

**系统接口：** 此接口为系统接口。

## data

```TypeScript
data?: string
```

服务数据。字符长度不超过1000个字符。默认为空。

**类型：** string

**起始版本：** 15

**系统能力：** SystemCapability.DistributedHardware.DeviceManager

**系统接口：** 此接口为系统接口。
