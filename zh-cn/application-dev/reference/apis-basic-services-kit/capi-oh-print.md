## 概述

提供打印模块的 C 接口，模块整体设计逻辑及各 API 的详细说明请参见[ohprint.h](capi-ohprint-h.md)。

**系统能力：** SystemCapability.Print.PrintFramework

**起始版本：** 12

## 文件汇总

| 名称                           | 描述                                                         |
| ------------------------------ | ------------------------------------------------------------ |
| [ohprint.h](capi-ohprint-h.md) | 声明用于发现和连接打印机、通过打印机打印文件、查询已添加打印机列表及其内部打印机信息等功能的 API。适用于需要在应用内集成打印能力的场景，帮助开发者实现打印机发现、连接、任务下发及状态监控等打印流程管理。 |
