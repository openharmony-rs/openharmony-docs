# oh_commonevent.h


## 概述

声明公共事件相关的API，包含订阅公共事件与退订公共事件。

**库：** libohcommonevent.so

**系统能力：** SystemCapability.Notification.CommonEvent

**起始版本：** 12

**相关模块：**[CommonEvent](capi-common-event.md)
