# 打印服务错误码

> **说明：**
>
> 打印服务提供打印任务管理、打印机查询与配置、打印扩展管理等能力，适用于应用内需要接入打印功能、管理打印任务和配置打印机的场景，帮助开发者便捷实现文档和图片等内容的打印输出。
>
> 以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](../errorcode-universal.md)。

## 13100001 一般打印异常

**错误信息**

Generic failure of print.

**可能原因**

该错误码表示一般打印异常，可能原因如下：
1. 打印内部创建任务对象失败

**处理步骤**

请关闭打印界面释放资源后重试；若问题持续，请重启设备以重置打印服务状态。

## 13100002 RPC 失败

**错误信息**

RPC failure.

**可能原因**

该错误码表示 RPC 失败，可能原因如下：
1. 获取 SA 服务对象失败
2. 进程间通信失败

**处理步骤**

请关闭打印界面释放资源后重试；若仍失败，请重启设备以重新获取 SA 服务对象。建议在调用打印相关接口前，确保 SA 服务对象可用。

## 13100003 打印服务异常

**错误信息**

Failure of print service.

**可能原因**

该错误码表示打印服务异常，可能原因如下：
1. 拉起打印服务失败
2. 拉起 cups 服务失败

**处理步骤**

请关闭打印界面以重新拉起打印服务后重试；若仍失败，请重启设备以重置 cups 服务。建议在调用打印相关接口前，确保打印服务和 cups 服务对象可用。

## 13100004 无效的打印扩展

**错误信息**

Invalid print extension.

**可能原因**

该错误码表示无效的打印扩展，可能原因如下：
1. 找不到打印扩展
2. 打印扩展参数错误
3. 打印扩展状态错误

**处理步骤**

请关闭打印界面释放资源后重试；若问题持续，请重启设备以重新加载打印扩展。建议在使用打印扩展前，先检查扩展参数和状态是否有效。

## 13100005 无效的打印机

**错误信息**

Invalid printer.

**可能原因**

该错误码表示无效的打印机，可能原因如下：
1. 打印机 ID 错误
2. cups 查询打印机列表失败
3. cups 查询打印机信息失败

**处理步骤**

请确认打印机 ID 及打印机信息是否正确后重试；若查询失败，请重启设备以重新初始化 cups 服务。建议在调用打印相关 API 前，先验证打印机 ID 及打印机信息是否有效。

## 13100006 无效的打印任务

**错误信息**

Invalid print job.

**可能原因**

该错误码表示无效的打印任务，可能原因如下：
1. 打印任务 ID 错误
2. 打印任务状态错误
3. 找不到打印任务

**处理步骤**

请确认打印任务 ID 和状态是否正确后重试；若任务无法找到，请取消当前任务后重新发起打印；若问题持续，请重启设备以重置打印服务。建议在操作打印任务前，先确认任务 ID 和状态是否正确。
