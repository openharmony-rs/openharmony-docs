## 概述

提供扫描模块的 C 接口，支持发现和连接扫描仪、从扫描仪扫描图片、获取图片扫描进度和设置扫描仪参数等功能。开发者可在应用内集成扫描仪相关操作。具体接口说明请参见 [ohscan.h](capi-ohscan-h.md)。

**系统能力：** SystemCapability.Print.PrintFramework

**起始版本：** 12
