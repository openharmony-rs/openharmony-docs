# BatteryStatus

触发延迟任务回调的电池状态。

**起始版本：** 9

## BATTERY_STATUS_LOW

```TypeScript
BATTERY_STATUS_LOW = 0
```

表示这个触发条件是低电告警。

**起始版本：** 9

**模型约束：** 此接口仅可在Stage模型下使用。

## BATTERY_STATUS_OKAY

```TypeScript
BATTERY_STATUS_OKAY = 1
```

**模型约束：** 此接口仅可在Stage模型下使用。

