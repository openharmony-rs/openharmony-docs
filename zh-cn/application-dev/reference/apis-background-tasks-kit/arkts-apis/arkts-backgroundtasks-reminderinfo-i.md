# ReminderInfo

代理提醒信息，包含 ReminderRequest 和 ReminderId。

**起始版本：** 12

**系统能力：** SystemCapability.Notification.ReminderAgent

## reminderId

```TypeScript
reminderId: number
```

发布提醒后返回的id。

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.Notification.ReminderAgent

## reminderReq

```TypeScript
reminderReq: ReminderRequest
```

代理提醒对象。

**类型：** ReminderRequest

**起始版本：** 12

**系统能力：** SystemCapability.Notification.ReminderAgent
