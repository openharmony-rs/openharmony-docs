# LocalDateTime

用于日历类提醒设置时指定时间信息。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

## day

```TypeScript
day: number
```

日，取值范围是[1, 31]。

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

## hour

```TypeScript
hour: number
```

**类型：** number

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

minute: number
