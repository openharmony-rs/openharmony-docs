# Native_Bundle

## 概述

Describe the functions of AbilityResourceInfo.

**起始版本：** 21
## 文件汇总

| 名称 | 描述 |
| -- | -- |
| [ability_resource_info.h](capi-ability-resource-info-h.md) | The file declares the APIs for obtaining the following ability resource information: bundle name, module name,ability name, icon, clone index, and whether the application is a default application. |
| [native_interface_bundle.h](capi-native-interface-bundle-h.md) | 提供查询应用包信息的功能，包括应用包名、应用指纹、应用appId等。 |
| [bundle_manager_common.h](capi-bundle-manager-common-h.md) | The file declares the error codes defined by BundleManager. |
