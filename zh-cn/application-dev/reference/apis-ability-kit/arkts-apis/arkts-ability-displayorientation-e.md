# DisplayOrientation

标识该Ability的显示模式。仅适用于FA模型的[PageAbility](../../../../application-models/pageability-overview.md)。

<!--Table: 40%; 10%; 50%-->

**起始版本：** 9

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## UNSPECIFIED

```TypeScript
UNSPECIFIED = 0
```

表示未定义方向模式，由系统判定。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。
