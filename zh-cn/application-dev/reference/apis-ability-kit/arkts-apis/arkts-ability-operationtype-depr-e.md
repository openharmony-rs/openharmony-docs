# OperationType

表示WantAgent支持的操作类型。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** OperationType

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## UNKNOWN_TYPE

```TypeScript
UNKNOWN_TYPE = 0
```

不识别的类型。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** UNKNOWN_TYPE

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
