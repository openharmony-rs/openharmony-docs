# ProcessInfo

定义进程信息，可以通过[getProcessInfo](../../../../reference/apis-ability-kit/js-apis-inner-app-context.md#contextgetprocessinfo7)
获取当前Ability运行的进程信息。

**起始版本：** 7

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## pid

```TypeScript
pid: number
```

进程ID。

**类型：** number

**起始版本：** 7

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## processName

```TypeScript
processName: string
```

进程名称。

**类型：** string

**起始版本：** 7

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
