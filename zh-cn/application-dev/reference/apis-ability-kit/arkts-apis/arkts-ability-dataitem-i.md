# DataItem

描述模块配置的路由表中的自定义数据。

**起始版本：** 12

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## key

```TypeScript
readonly key: string
```

标识路由表自定义数据的键。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## value

```TypeScript
readonly value: string
```

标识路由表自定义数据的值。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core
