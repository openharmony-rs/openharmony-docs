# AtomicServiceStartupRule（系统接口）

嵌入式拉起原子化服务的规则。

**起始版本：** 18

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

**系统接口：** 此接口为系统接口。

## isEmbeddedAllowed

```TypeScript
isEmbeddedAllowed: boolean
```

是否允许嵌入式拉起原子化服务。true表示允许嵌入式拉起原子化服务，false表示不允许嵌入式拉起原子化服务。

**类型：** boolean

**起始版本：** 18

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

**系统接口：** 此接口为系统接口。
