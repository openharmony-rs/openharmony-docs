# ExtensionAbility

ExtensionAbility是特定场景扩展能力的基类，继承自[Ability](arkts-ability-app-ability-ability-ability-c.md)，未新增属性和方法。不支持开发者直接继承ExtensionAbility。各类Ability的继承关系详见[继承关系说明](docroot://reference/apis-ability-kit/js-apis-app-ability-ability.md#ability的继承关系说明)。

**继承/实现关系：** ExtensionAbility extends [Ability](arkts-ability-app-ability-ability-ability-c.md)

**起始版本：** 9

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
