# ReleaseStatus

表示锁屏下敏感数据访问权限释放状态的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager

## RELEASE_DENIED

```TypeScript
RELEASE_DENIED = -1
```

释放锁屏下敏感数据访问权限被拒绝。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager

## RELEASE_GRANTED

```TypeScript
RELEASE_GRANTED = 0
```

释放锁屏下敏感数据访问权限被允许。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager
