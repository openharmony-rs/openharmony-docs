# DataType

表示锁屏下访问敏感数据类型的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager

## MEDIA_DATA

```TypeScript
MEDIA_DATA = 0x00000001
```

媒体数据类型。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager

## ALL_DATA

```TypeScript
ALL_DATA = 0xffffffff
```

所有敏感数据类型。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager
