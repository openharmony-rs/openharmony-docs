# PopupPlacement（系统接口）

表示气泡弹窗位置的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。

## LEFT

```TypeScript
LEFT = 0
```

气泡弹窗位于组件左侧，与组件左侧中心对齐。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。
