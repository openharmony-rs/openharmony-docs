# Context

```TypeScript
export type Context = _Context
```

提供Ability或Application的上下文，可用于访问应用程序的资源。

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
