# ResultCode

模态弹框请求结果码。

**起始版本：** 9

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## RESULT_OK

```TypeScript
RESULT_OK = 0
```

表示成功。

**起始版本：** 9

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## RESULT_CANCEL

```TypeScript
RESULT_CANCEL = 1
```

表示失败。

**起始版本：** 9

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
