# ResourceType

应用资源超基线的类型。

**起始版本：** 24

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

## PSS_MEMORY

```TypeScript
PSS_MEMORY = 1
```

表示应用当前超基线的资源是PSS的内存。

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore
