# ControlType（系统接口）

标识拦截指定应用程序的不同策略。

**起始版本：** 11

**系统能力：** SystemCapability.BundleManager.BundleFramework.AppControl

**系统接口：** 此接口为系统接口。

## ALLOWED_LIST

```TypeScript
ALLOWED_LIST = 1
```

允许运行指定功能的列表。

**起始版本：** 11

**系统能力：** SystemCapability.BundleManager.BundleFramework.AppControl

**系统接口：** 此接口为系统接口。

## DISALLOWED_LIST

```TypeScript
DISALLOWED_LIST = 2
```

不允许运行指定功能的列表。

**起始版本：** 11

**系统能力：** SystemCapability.BundleManager.BundleFramework.AppControl

**系统接口：** 此接口为系统接口。
