# ErrorCode

```TypeScript
export enum ErrorCode
```

ErrorCode定义启动Ability时返回的错误码，包括无效的参数、权限拒绝等。

**起始版本：** 6

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## PERMISSION_DENY

```TypeScript
PERMISSION_DENY = -3
```

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
