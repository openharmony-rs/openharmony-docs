# EventHub

```TypeScript
export type EventHub = _EventHub.default
```

EventHub是系统提供的基于发布-订阅模式实现的事件通信机制。

**起始版本：** 9

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
