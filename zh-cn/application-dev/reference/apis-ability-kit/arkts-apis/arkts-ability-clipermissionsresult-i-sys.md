# CliPermissionsResult（系统接口）

表示CLI权限查询结果。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。

## permList

```TypeScript
permList: Array<CliCommandPermissionResult>
```

CLI权限信息的列表。

**类型：** Array<CliCommandPermissionResult>

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。
