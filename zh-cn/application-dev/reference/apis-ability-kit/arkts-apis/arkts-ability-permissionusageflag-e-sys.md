# PermissionUsageFlag（系统接口）

表示使用记录的查询方式的枚举。

**起始版本：** 9

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。

## FLAG_PERMISSION_USAGE_SUMMARY

```TypeScript
FLAG_PERMISSION_USAGE_SUMMARY = 0
```

表示查询总览数据。

**起始版本：** 9

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。

## FLAG_PERMISSION_USAGE_DETAIL

```TypeScript
FLAG_PERMISSION_USAGE_DETAIL = 1
```

表示查询详细数据。

**起始版本：** 9

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。
