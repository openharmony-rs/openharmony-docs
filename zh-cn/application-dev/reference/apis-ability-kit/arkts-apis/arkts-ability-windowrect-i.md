# WindowRect

表示模态弹框的属性。

**起始版本：** 10

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## height

```TypeScript
height: number
```

弹框的高度，单位为px。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## left

```TypeScript
left: number
```

弹框边框的左上角的X坐标。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## top

```TypeScript
top: number
```

弹框边框的左上角的Y坐标。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## width

```TypeScript
width: number
```

弹框的宽度，单位为px。

**类型：** number

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
