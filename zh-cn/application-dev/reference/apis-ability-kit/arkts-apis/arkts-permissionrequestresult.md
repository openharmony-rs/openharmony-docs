# PermissionRequestResult

## 汇总

### 类

| 名称 | 说明 |
| --- | --- |
| [PermissionRequestResult](arkts-ability-permissionrequestresult-c.md) | PermissionRequestResult是权限申请的结果对象。开发者需先创建atManager实例，再调用requestPermissionsFromUser方法申请权限，该方法返回PermissionRequestResult对象，开发者可通过该对象的属性判断权限申请结果。权限申请整体流程及atManager的详细说明请参见[@ohos.abilityAccessCtrl (程序访问控制管理)](arkts-abilityaccessctrl.md)。 |
