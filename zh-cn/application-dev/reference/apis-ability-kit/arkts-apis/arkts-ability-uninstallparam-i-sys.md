# UninstallParam（系统接口）

共享包卸载需指定的参数信息。

**起始版本：** 10

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

**系统接口：** 此接口为系统接口。

## bundleName

```TypeScript
bundleName: string
```

共享包包名。

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

**系统接口：** 此接口为系统接口。

## versionCode

```TypeScript
versionCode?: number
```

指示共享包的版本号。默认值：如果不填写versionCode，则卸载该包名的所有共享包。

**类型：** number

**起始版本：** 10

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

**系统接口：** 此接口为系统接口。
