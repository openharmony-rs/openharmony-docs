# SkillInfo

提供Skill的相关信息。Skill是一个可安装的能力单位，可以被发现并由Agent框架调用。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## abilityName

```TypeScript
readonly abilityName: string
```

Skill关联的ability名称。

**类型：** string

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core
