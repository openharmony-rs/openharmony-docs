# DataAbilityOperationType

表示数据的操作类型。DataAbility批量操作数据时可以通过该枚举值指定操作类型。

**起始版本：** 7

**系统能力：** SystemCapability.Ability.AbilityRuntime.FAModel

## TYPE_INSERT

```TypeScript
TYPE_INSERT = 1
```

插入类型。

**起始版本：** 7

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.FAModel
