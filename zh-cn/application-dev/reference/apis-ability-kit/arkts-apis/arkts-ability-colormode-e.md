# ColorMode

> **说明：**
>
> 从API version 7开始支持，从API version 9开始废弃，暂无替代接口。

应用、卡片等的颜色模式。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** [null]

**系统能力：** SystemCapability.BundleManager.BundleFramework

## AUTO_MODE

```TypeScript
AUTO_MODE = -1
```

自动模式。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** COLOR_MODE_NOT_SET

**系统能力：** SystemCapability.BundleManager.BundleFramework

## DARK_MODE

```TypeScript
DARK_MODE = 0
```

黑色模式。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** COLOR_MODE_DARK

**系统能力：** SystemCapability.BundleManager.BundleFramework

## LIGHT_MODE

```TypeScript
LIGHT_MODE = 1
```

亮度模式。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** COLOR_MODE_LIGHT

**系统能力：** SystemCapability.BundleManager.BundleFramework
