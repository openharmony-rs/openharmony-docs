# AutoFillType

表示提供自动填充类型的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

## UNSPECIFIED

```TypeScript
UNSPECIFIED = 0
```

表示未指定的类型。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。
