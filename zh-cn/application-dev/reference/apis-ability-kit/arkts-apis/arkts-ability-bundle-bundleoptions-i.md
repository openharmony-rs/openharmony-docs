# BundleOptions

> **说明：**  
>  
> 从API version 7开始支持，从API version 9开始废弃，暂无替代接口。

查询选项，包含userId。

**起始版本：** 7

**废弃版本：** 9

**系统能力：** SystemCapability.BundleManager.BundleFramework

## 导入模块

```TypeScript
import { bundle } from '@kit.AbilityKit';
```

## userId

```TypeScript
userId?: number
```

用户ID。默认值：调用方所在用户，取值范围：大于等于0。

**类型：** number

**起始版本：** 7

**废弃版本：** 9
