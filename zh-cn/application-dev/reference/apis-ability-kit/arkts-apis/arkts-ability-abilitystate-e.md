# AbilityState

Ability的状态，该类型为枚举，可配合[AbilityRunningInfo](arkts-ability-abilityrunninginfo-i.md)返回Ability的状态。

**起始版本：** 14

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## INITIAL

```TypeScript
INITIAL = 0
```

表示ability为初始化状态。

**起始版本：** 14

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## FOCUS

```TypeScript
FOCUS = 2
```
