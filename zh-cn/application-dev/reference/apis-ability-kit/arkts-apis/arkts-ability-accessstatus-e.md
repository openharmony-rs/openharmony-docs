# AccessStatus

表示锁屏下敏感数据访问权限申请状态的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager

## ACCESS_DENIED

```TypeScript
ACCESS_DENIED = -1
```

申请锁屏下敏感数据访问权限被拒绝。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager

## ACCESS_GRANTED

```TypeScript
ACCESS_GRANTED = 0
```

申请锁屏下敏感数据访问权限被允许。

**起始版本：** 12

**系统能力：** SystemCapability.Security.ScreenLockFileManager
