# ShellCmdResult

本模块提供Shell命令执行结果的能力。

> **说明：**
>
> 本模块接口仅可在[单元测试框架](../../../../application-test/unittest-guidelines.md)中使用。

**起始版本：** 8

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## exitCode

```TypeScript
exitCode: number
```

Shell命令的结果码。

**类型：** number

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## stdResult

```TypeScript
stdResult: string
```

Shell命令的标准输出内容。

**类型：** string

**起始版本：** 8

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
