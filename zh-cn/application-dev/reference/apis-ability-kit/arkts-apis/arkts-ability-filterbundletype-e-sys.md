# FilterBundleType（系统接口）

表示要监听的的应用类型，该类型为枚举。可配合[AppStateFilter](arkts-ability-appstatefilter-i-sys.md)过滤想要监听的应用类型。

**起始版本：** 21

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

**系统接口：** 此接口为系统接口。

## APP

```TypeScript
APP = 1 << 0
```

应用。

**起始版本：** 21

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

**系统接口：** 此接口为系统接口。
