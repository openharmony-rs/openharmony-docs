# PopupSize（系统接口）

表示气泡弹窗的宽和高。

**起始版本：** 12

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。

## height

```TypeScript
height: number
```

气泡弹窗的高。

**类型：** number

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。

## width

width: number

**模型约束：** 此接口仅可在Stage模型下使用。
