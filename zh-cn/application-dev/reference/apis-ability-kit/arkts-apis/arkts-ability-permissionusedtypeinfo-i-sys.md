# PermissionUsedTypeInfo（系统接口）

表示某次权限使用类型的详情。

**起始版本：** 12

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。

## permissionName

```TypeScript
permissionName: Permissions
```

被访问的敏感权限名称。

**类型：** Permissions

**起始版本：** 12

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。

## tokenId

```TypeScript
tokenId: number
```

访问敏感权限的应用身份标识。

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。

## usedType

```TypeScript
usedType: PermissionUsedType
```

敏感权限使用类型。

**类型：** PermissionUsedType

**起始版本：** 12

**系统能力：** SystemCapability.Security.AccessToken

**系统接口：** 此接口为系统接口。
