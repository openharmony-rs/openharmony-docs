## FAILURE_CODE_SYSTEM_MALFUNCTION

```TypeScript
FAILURE_CODE_SYSTEM_MALFUNCTION = 0
```

表示由于系统错误（如跳转弹框崩溃）而无法打开原子化服务。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本20开始，该接口支持在原子化服务API中使用。
