# PermissionGrantState

权限授予状态。

**起始版本：** 9

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## PERMISSION_DENIED

```TypeScript
PERMISSION_DENIED = -1
```

拒绝授予权限。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## PERMISSION_GRANTED

```TypeScript
PERMISSION_GRANTED = 0
```

授予权限。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core
