# ShowMode

```TypeScript
export enum ShowMode
```

表示[EmbeddableUIAbility](arkts-ability-embeddableuiability-c.md#EmbeddableUIAbility)被拉起时的显示模式。

**起始版本：** 12

**系统能力：** SystemCapability.Ability.AbilityBase

## WINDOW

```TypeScript
WINDOW = 0
```

表示独立窗口拉起模式。

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityBase

## EMBEDDED_FULL

```TypeScript
EMBEDDED_FULL = 1
```

表示嵌入式全屏拉起模式。

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityBase

## EMBEDDED_HALF

```TypeScript
EMBEDDED_HALF = 2
```

表示嵌入式半屏拉起模式。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Ability.AbilityBase
