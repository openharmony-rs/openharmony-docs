# PermissionOptions

权限选项。

**起始版本：** 7

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## pid

```TypeScript
pid?: number
```

进程id。

**类型：** number

**默认值：** The process id

**起始版本：** 7

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## uid

```TypeScript
uid?: number
```

用户id。

**类型：** number

**默认值：** The user id

**起始版本：** 7

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
