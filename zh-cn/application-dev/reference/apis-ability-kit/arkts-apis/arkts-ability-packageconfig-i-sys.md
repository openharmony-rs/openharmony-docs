# PackageConfig（系统接口）

pack.info的包信息。

**起始版本：** 9

**系统能力：** SystemCapability.BundleManager.BundleFramework.FreeInstall

**系统接口：** 此接口为系统接口。

## deliveryWithInstall

```TypeScript
readonly deliveryWithInstall: boolean
```

是否跟随应用一起安装。true表示跟随应用一起安装，false表示不跟随应用一起安装。

**类型：** boolean

**起始版本：** 9

**系统能力：** SystemCapability.BundleManager.BundleFramework.FreeInstall

**系统接口：** 此接口为系统接口。
