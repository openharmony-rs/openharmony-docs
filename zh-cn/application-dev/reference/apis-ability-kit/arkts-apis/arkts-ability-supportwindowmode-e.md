# SupportWindowMode

标识该组件所支持的窗口模式。

**起始版本：** 9

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## FULL_SCREEN

```TypeScript
FULL_SCREEN = 0
```

窗口支持全屏显示。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## SPLIT

```TypeScript
SPLIT = 1
```

窗口支持分屏显示。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core

## FLOATING

```TypeScript
FLOATING = 2
```

支持窗口化显示，即显示悬浮窗口。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.BundleManager.BundleFramework.Core
