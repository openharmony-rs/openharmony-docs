## 函数说明

### OH_AT_CheckSelfPermission()

```
bool OH_AT_CheckSelfPermission(const char *permission)
```

**描述**

校验应用是否被授予指定的权限。

**起始版本：** 12


**参数：**

| 参数项 | 描述 |
| -- | -- |
| const char *permission | 需要校验的权限名称，合法的权限名取值可在应用权限列表中查询。 |

**返回：**

| 类型 | 说明 |
| -- | -- |
| bool | true：应用已经被授予该权限。<br>         false：应用未被授予该权限。 |
