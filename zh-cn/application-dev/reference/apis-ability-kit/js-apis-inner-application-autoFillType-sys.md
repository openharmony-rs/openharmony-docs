表示提供自动填充类型的枚举。

> **说明：**
> 
> 本模块首批接口从API version 11开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。  
> 本模块接口仅可在Stage模型下使用。
> 本模块接口为系统接口。

## 导入模块

```ts
import { autoFillManager } from '@kit.AbilityKit';
```

**系统接口**：此接口为系统接口。
