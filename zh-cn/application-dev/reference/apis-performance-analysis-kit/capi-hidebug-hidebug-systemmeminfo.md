## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| uint32_t totalMem | 系统总的内存，以KB为单位。 |
| uint32_t freeMem | 系统空闲的内存，以KB为单位。 |
| uint32_t availableMem | 系统可用的内存，以KB为单位。 |
