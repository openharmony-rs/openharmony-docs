# Param

此接口提供了所有预定义参数的参数名称常量。

| 名称 | 类型 | 可读 | 可写 | 说明 |  
| ------------------------------- | ------ | ---- | ---- | ------------------ |  
| USER_ID | string | 是 | 否 | 用户自定义ID。 |  
| DISTRIBUTED_SERVICE_NAME | string | 是 | 否 | 分布式服务名称。 |  
| DISTRIBUTED_SERVICE_INSTANCE_ID | string | 是 | 否 | 分布式服务实例ID。 |
