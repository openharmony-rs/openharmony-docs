# @ohos.hiAppEvent

本模块提供了应用事件打点能力，包括对打点数据的落盘，以及对打点功能的管理配置。

> **说明：**  
>  
> - 本模块接口从API version 9开始废弃，建议使用新接口[@ohos.hiviewdfx.hiAppEvent](arkts-performanceanalysis-hiappevent-n.md)替代。
