## 概述

OH_NativeColorSpaceManager是色彩空间管理器的原生结构体，用于表示和管理色彩空间信息。开发者可以通过该结构体在不同图形处理场景中进行色彩空间的创建、查询和转换操作，确保图像和视频内容的色彩准确性。适用于需要进行色彩空间转换、色彩管理或跨平台色彩一致性处理的场景。
