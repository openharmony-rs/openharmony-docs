# TextAlign

文本对齐方式枚举。

**起始版本：** 12

<!--Device-text-enum TextAlign--><!--Device-text-enum TextAlign-End-->

**系统能力：** SystemCapability.Graphics.Drawing

## LEFT

```TypeScript
LEFT = 0
```

文本靠左对齐。

**起始版本：** 12

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
