# TextEncoding

文本的编码类型枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_UTF8

```TypeScript
TEXT_ENCODING_UTF8 = 0
```

使用1个字节表示UTF-8或ASCII。

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_UTF16

```TypeScript
TEXT_ENCODING_UTF16 = 1
```

使用2个字节表示大部分unicode。

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_UTF32

```TypeScript
TEXT_ENCODING_UTF32 = 2
```

使用4个字节表示全部unicode。

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_GLYPH_ID

```TypeScript
TEXT_ENCODING_GLYPH_ID = 3
```

使用2个字节表示glyph index。

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
