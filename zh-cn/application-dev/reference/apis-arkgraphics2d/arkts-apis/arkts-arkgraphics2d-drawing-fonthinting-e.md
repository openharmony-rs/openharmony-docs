## NONE

```TypeScript
NONE = 0
```

不修改字型轮廓。

**起始版本：** 12

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。

```TypeScript
SLIGHT = 1
```

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
