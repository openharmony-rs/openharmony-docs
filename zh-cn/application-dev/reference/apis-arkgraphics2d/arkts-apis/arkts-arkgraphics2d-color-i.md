# Color

ARGB格式的颜色描述。

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## alpha

```TypeScript
alpha: number
```

颜色的A分量（透明度），值是0~255的整数。

**类型：** number

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
