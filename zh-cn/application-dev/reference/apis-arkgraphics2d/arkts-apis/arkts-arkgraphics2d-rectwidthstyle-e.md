# RectWidthStyle

矩形区域宽度规格枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## TIGHT

```TypeScript
TIGHT = 0
```

不设置letterSpacing时，与字形紧贴，否则包含letterSpacing的宽度。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## MAX

```TypeScript
MAX = 1
```

扩展宽度，以匹配所有行上最宽矩形的位置。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
