# Rect

矩形区域，通过2个坐标点可以描述出一个矩形区域，这2个点分别认为是矩形区域的左上角点与右下角点。

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## bottom

```TypeScript
bottom: number
```

矩形区域的右下角纵坐标，浮点数。

**类型：** number

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## left

```TypeScript
left: number
```

**类型：** number

**起始版本：** 11

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
