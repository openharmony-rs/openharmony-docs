# Color4f

ARGB格式的颜色描述。

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## alpha

```TypeScript
alpha: number
```

颜色的A分量（透明度），值是0.0~1.0的浮点数。

**类型：** number

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## blue

```TypeScript
blue: number
```

颜色的B分量（蓝色），值是0.0~1.0的浮点数。

**类型：** number

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## green

```TypeScript
green: number
```

颜色的G分量（绿色），值是0.0~1.0的浮点数。

**类型：** number

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## red

```TypeScript
red: number
```

颜色的R分量（红色），值是0.0~1.0的浮点数。

**类型：** number

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing
