# RunMetrics

描述文本行中连续文本块的布局信息和度量数据。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## fontMetrics

```TypeScript
fontMetrics: drawing.FontMetrics
```

字体度量信息。

**类型：** drawing.FontMetrics

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## textStyle

```TypeScript
textStyle: TextStyle
```

字体的样式信息。

**类型：** TextStyle

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
