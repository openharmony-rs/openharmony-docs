# TextBox

文本矩形区域，表示文本在布局时所占用的矩形空间。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## direction

```TypeScript
direction: TextDirection
```

文本方向。

**类型：** TextDirection

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## rect

```TypeScript
rect: common2D.Rect
```

矩形区域信息，单位为物理像素px。

**类型：** common2D.Rect

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
