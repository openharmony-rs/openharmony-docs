# RectStyle

矩形框样式。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## color

```TypeScript
color: common2D.Color
```

矩形框的颜色。

**类型：** common2D.Color

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## leftBottomRadius

```TypeScript
leftBottomRadius: number
```

矩形框的左下半径，单位为物理像素px。

**类型：** number

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing
