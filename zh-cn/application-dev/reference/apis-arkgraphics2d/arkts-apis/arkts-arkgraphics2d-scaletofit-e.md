# ScaleToFit

源矩形到目标矩形的缩放方式枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## FILL_SCALE_TO_FIT

```TypeScript
FILL_SCALE_TO_FIT = 0
```

将源矩形缩放以填充满整个目标矩形，可能会改变源矩形的长宽比。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## START_SCALE_TO_FIT

```TypeScript
START_SCALE_TO_FIT = 1
```

保持源矩形的长宽比进行缩放，并对齐到目标矩形的左上方。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## CENTER_SCALE_TO_FIT

```TypeScript
CENTER_SCALE_TO_FIT = 2
```

保持源矩形的长宽比进行缩放，并居中对齐到目标矩形。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## END_SCALE_TO_FIT

```TypeScript
END_SCALE_TO_FIT = 3
```

保持源矩形的长宽比进行缩放，并对齐到目标矩形的右下方。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing
