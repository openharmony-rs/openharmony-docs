# RectHeightStyle

矩形区域高度规格枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## TIGHT

```TypeScript
TIGHT = 0
```

与字形紧贴。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## MAX

```TypeScript
MAX = 1
```

扩展高度，以匹配所有行上最高矩形的位置。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## INCLUDE_LINE_SPACE_MIDDLE

```TypeScript
INCLUDE_LINE_SPACE_MIDDLE = 2
```

每个矩形的顶部和底部将覆盖行上方和行下方的一半空间。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## INCLUDE_LINE_SPACE_TOP

```TypeScript
INCLUDE_LINE_SPACE_TOP = 3
```

行间距将被添加到矩形的顶部。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## INCLUDE_LINE_SPACE_BOTTOM

```TypeScript
INCLUDE_LINE_SPACE_BOTTOM = 4
```

行间距将被添加到矩形的底部。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## STRUT

```TypeScript
STRUT = 5
```

高度按照文本的样式设置。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
