# FontFeature

文本字体特征。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## name

```TypeScript
name: string
```

字体特征键值对中的关键字标识的字符串。

**类型：** string

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## value

```TypeScript
value: number
```

字体特征键值对的值。

**类型：** number

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
