# createFilter

## createFilter

```TypeScript
function createFilter(): Filter
```

创建Filter实例用于给组件添加多种filter效果。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

**返回值：**

| 类型 | 说明 |
| --- | --- |
| Filter | 返回Filter的头节点。 |

**示例：**

```TypeScript
let filter : uiEffect.Filter = uiEffect.createFilter()

```
