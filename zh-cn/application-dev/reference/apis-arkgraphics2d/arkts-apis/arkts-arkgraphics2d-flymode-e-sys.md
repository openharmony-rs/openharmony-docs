# FlyMode（系统接口）

飞入飞出形变场景模式枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

**系统接口：** 此接口为系统接口。

## BOTTOM

```TypeScript
BOTTOM = 0
```

从底部进行飞入飞出形变。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

**系统接口：** 此接口为系统接口。

## TOP

```TypeScript
TOP = 1
```

从顶部进行飞入飞出形变。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

**系统接口：** 此接口为系统接口。
