# Point

坐标点。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## x

```TypeScript
x: number
```

横坐标，浮点数。

**类型：** number

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## y

```TypeScript
y: number
```

纵坐标，浮点数。

**类型：** number

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
