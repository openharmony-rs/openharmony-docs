# TextRectSize

文本矩形尺寸，用于描述文本的矩形宽高属性。值为浮点数，单位为物理像素px。

**起始版本：** 24

**系统能力：** SystemCapability.Graphics.Drawing

## height

```TypeScript
height: number
```

文本矩形的高度，浮点数，单位为物理像素px。

**类型：** number

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## width

```TypeScript
width: number
```

文本矩形的宽度，浮点数，单位为物理像素px。

**类型：** number

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
