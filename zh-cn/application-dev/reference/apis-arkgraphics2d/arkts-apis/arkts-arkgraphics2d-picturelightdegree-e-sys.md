# PictureLightDegree（系统接口）

图片颜色明亮度的枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Multimedia.Image.Core

**系统接口：** 此接口为系统接口。

## UNKNOWN_LIGHT_COLOR_DEGREE_PICTURE

```TypeScript
UNKNOWN_LIGHT_COLOR_DEGREE_PICTURE = 0
```

未知明亮度的图片。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**卡片能力：** 从API版本26.0.0开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.Multimedia.Image.Core

**系统接口：** 此接口为系统接口。
