# TextProcessState

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Graphics.Drawing

## INIT

```TypeScript
INIT = 0
```

初始状态，文本处理尚未开始。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
