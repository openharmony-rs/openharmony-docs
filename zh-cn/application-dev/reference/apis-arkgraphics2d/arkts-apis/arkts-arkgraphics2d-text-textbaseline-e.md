## ALPHABETIC

```TypeScript
ALPHABETIC = 0
```

用于拉丁字母的文本基线对齐。

**起始版本：** 12

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。

```TypeScript

```

**起始版本：** 12

**原子化服务API：** 从API版本22开始，该接口支持在原子化服务API中使用。
