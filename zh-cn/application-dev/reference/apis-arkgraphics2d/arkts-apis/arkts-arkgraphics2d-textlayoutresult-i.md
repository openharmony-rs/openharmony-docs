# TextLayoutResult

文本布局结果。

**起始版本：** 24

**系统能力：** SystemCapability.Graphics.Drawing

## correctRect

```TypeScript
correctRect: TextRectSize
```

布局后段落的矩形尺寸。

**类型：** TextRectSize

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## fitStrRange

```TypeScript
fitStrRange: Array<Range>
```

文本布局计算完成后能够完整显示的字符范围数组。

**类型：** Array<Range>

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
