# TextDirection

文本排版方向枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## RTL

```TypeScript
RTL = 0
```

文本从右到左排版。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## LTR

```TypeScript
LTR = 1
```

文本从左到右排版。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
