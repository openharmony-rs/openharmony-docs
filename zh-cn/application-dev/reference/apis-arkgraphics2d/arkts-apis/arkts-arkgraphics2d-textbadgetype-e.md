# TextBadgeType

文本上下标枚举。

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_BADGE_NONE

```TypeScript
TEXT_BADGE_NONE = 0
```

不使能上下标。

**起始版本：** 20

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_SUPERSCRIPT

```TypeScript
TEXT_SUPERSCRIPT = 1
```

使能上标。

**起始版本：** 20

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## TEXT_SUBSCRIPT

TEXT_SUBSCRIPT = 2
