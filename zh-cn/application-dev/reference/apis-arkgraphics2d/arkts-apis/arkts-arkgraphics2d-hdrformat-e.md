# HDRFormat

HDR格式枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Graphic.Graphic2D.ColorManager.Core

## NONE

```TypeScript
NONE = 0
```

不支持HDR类型。

从API version 12开始，该接口支持在原子化服务中使用。

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphic.Graphic2D.ColorManager.Core
