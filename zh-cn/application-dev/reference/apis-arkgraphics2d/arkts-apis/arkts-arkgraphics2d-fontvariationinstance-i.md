# FontVariationInstance

字体可变实例信息，存放预设的可变字体样式信息。

**起始版本：** 24

**系统能力：** SystemCapability.Graphics.Drawing

## coordinates

```TypeScript
coordinates: Array<FontVariation>
```

可变字体属性数组。

**类型：** Array<FontVariation>

**起始版本：** 24

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## localName

```TypeScript
localName: string
```

字体可变实例的本地化名称，可以为空。

**类型：** string

**起始版本：** 24

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## name

```TypeScript
name: string
```

字体可变实例的英文名称。

**类型：** string

**起始版本：** 24

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
