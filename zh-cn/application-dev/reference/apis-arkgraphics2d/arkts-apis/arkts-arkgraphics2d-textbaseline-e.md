# TextBaseline

文本基线类型枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## ALPHABETIC

```TypeScript
ALPHABETIC = 0
```

用于拉丁字母的文本基线对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## IDEOGRAPHIC

```TypeScript
IDEOGRAPHIC = 1
```

用于CJK（中文，日文，韩文）的文本基线对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
