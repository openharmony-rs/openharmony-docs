# PointMode

绘制数组点的方式的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## POINTS

```TypeScript
POINTS = 0
```

分别绘制每个点。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## LINES

```TypeScript
LINES = 1
```

将每对点绘制为线段。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## POLYGON

```TypeScript
POLYGON = 2
```

将点阵列绘制为开放多边形。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing
