# FontHinting

字型轮廓效果类型枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## NONE

```TypeScript
NONE = 0
```

不修改字型轮廓。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## SLIGHT

```TypeScript
SLIGHT = 1
```

最小限度修改字型轮廓以改善对比度。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## NORMAL

```TypeScript
NORMAL = 2
```

修改字型轮廓以提高对比度。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## FULL

```TypeScript
FULL = 3
```

修改字型轮廓以获得最大对比度。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
