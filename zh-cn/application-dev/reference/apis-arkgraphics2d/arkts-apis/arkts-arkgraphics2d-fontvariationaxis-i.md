# FontVariationAxis

字体可变轴信息。

**起始版本：** 24

**系统能力：** SystemCapability.Graphics.Drawing

## defaultValue

```TypeScript
defaultValue: number
```

字体可变轴的默认值。

**类型：** number

**起始版本：** 24

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## flags

```TypeScript
flags: number
```

字体可变轴的标志位，用于标记某个可变轴是否应该对用户隐藏，值为0或1。值为0时表示该轴对用户可见，值为1时表示该轴应隐藏。

**类型：** number

**起始版本：** 24

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
