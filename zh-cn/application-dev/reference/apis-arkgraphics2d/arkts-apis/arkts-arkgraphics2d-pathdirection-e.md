# PathDirection

添加闭合轮廓方向的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## CLOCKWISE

```TypeScript
CLOCKWISE = 0
```

顺时针方向添加闭合轮廓。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## COUNTER_CLOCKWISE

```TypeScript
COUNTER_CLOCKWISE = 1
```

逆时针方向添加闭合轮廓。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing
