# TextAlign

文本对齐方式枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## LEFT

```TypeScript
LEFT = 0
```

文本靠左对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## RIGHT

```TypeScript
RIGHT = 1
```

文本靠右对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## CENTER

```TypeScript
CENTER = 2
```

文本居中对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## JUSTIFY

```TypeScript
JUSTIFY = 3
```

文本两侧对齐，对最后一行无效。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## START

```TypeScript
START = 4
```

基于文本的方向[TextDirection](arkts-arkgraphics2d-textdirection-e.md)，文本靠开头方向对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## END

```TypeScript
END = 5
```

基于文本的方向[TextDirection](arkts-arkgraphics2d-textdirection-e.md)，文本以结束方向对齐。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
