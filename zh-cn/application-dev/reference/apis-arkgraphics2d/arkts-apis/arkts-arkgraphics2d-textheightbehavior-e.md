# TextHeightBehavior

文本高度修饰符模式枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## ALL

```TypeScript
ALL = 0x0
```

高度修饰符设置为段落中第一行上升、最后一行下降。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## DISABLE_FIRST_ASCENT

```TypeScript
DISABLE_FIRST_ASCENT = 0x1
```

高度修饰符设置为禁止段落中第一行上升。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## DISABLE_LAST_ASCENT

```TypeScript
DISABLE_LAST_ASCENT = 0x2
```

高度修饰符设置为禁止段落中最后一行下降。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## DISABLE_ALL

```TypeScript
DISABLE_ALL = 0x1 | 0x2
```

高度修饰符设置为禁止段落中第一行上升、最后一行下降。

**起始版本：** 12

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Graphics.Drawing
