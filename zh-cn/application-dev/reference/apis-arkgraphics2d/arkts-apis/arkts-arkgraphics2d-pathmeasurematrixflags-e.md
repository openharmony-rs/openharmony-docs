# PathMeasureMatrixFlags

路径测量中的矩阵信息维度枚举，常用于控制物体沿路径移动的动画场景。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## GET_POSITION_MATRIX

```TypeScript
GET_POSITION_MATRIX = 0
```

获取位置信息对应的矩阵。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## GET_TANGENT_MATRIX

```TypeScript
GET_TANGENT_MATRIX = 1
```

获取切线信息对应的矩阵。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## GET_POSITION_AND_TANGENT_MATRIX

```TypeScript
GET_POSITION_AND_TANGENT_MATRIX = 2
```

获取位置和切线信息对应的矩阵。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing
