# drawing_canvas.h

## 概述

文件中定义了与画布相关的功能函数。<br>画布自带一个黑色，开启抗锯齿，不具备其他任何样式的默认画刷，当且仅当画布中主动设置的画刷和画笔都不存在时生效。

**起始版本：** 8

**相关模块：** [Drawing](capi-drawing.md)
