# 常量

## MAX_KEY_LENGTH

```TypeScript
const MAX_KEY_LENGTH: 80
```

key的最大长度限制为80字节。

**起始版本：** 6

**废弃版本：** 9

## MAX_VALUE_LENGTH

```TypeScript
const MAX_VALUE_LENGTH: 8192
```

value的最大长度限制为16MB。

**起始版本：** 6

**废弃版本：** 9
