# 常量

## MAX_KEY_LENGTH

```TypeScript
const MAX_KEY_LENGTH: number
```

Key的最大长度限制为1024个字节。

**起始版本：** 12

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。

## MAX_VALUE_LENGTH

```TypeScript
const MAX_VALUE_LENGTH: number
```

Value的最大长度限制为16MB。

**起始版本：** 12

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。
