# 常量

## MAX_KEY_LENGTH

```TypeScript
const MAX_KEY_LENGTH: number
```

Key的最大长度限制为1024个字节。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.DistributedDataManager.Preferences.Core

## MAX_VALUE_LENGTH

```TypeScript
const MAX_VALUE_LENGTH: number
```

Value的最大长度限制为16MB。

**起始版本：** 12

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.DistributedDataManager.Preferences.Core
