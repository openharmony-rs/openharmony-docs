# BindInfo

数据库的绑定信息。当前版本只支持关系型数据库的绑定。

**起始版本：** 11

**系统能力：** SystemCapability.DistributedDataManager.DataObject.DistributedObject

## assetName

```TypeScript
assetName: string
```

待绑定资产在所属的数据库中的资产名。

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.DistributedDataManager.DataObject.DistributedObject

## field

```TypeScript
field: string
```

待绑定资产在所属的数据库中的列名。

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.DistributedDataManager.DataObject.DistributedObject

## primaryKey

```TypeScript
primaryKey: commonType.ValuesBucket
```

待绑定资产在所属的数据库中的主键。

**类型：** commonType.ValuesBucket

**起始版本：** 11

**系统能力：** SystemCapability.DistributedDataManager.DataObject.DistributedObject

## storeName

```TypeScript
storeName: string
```

待绑定资产在所属的数据库中的库名。

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.DistributedDataManager.DataObject.DistributedObject

## tableName

```TypeScript
tableName: string
```

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.DistributedDataManager.DataObject.DistributedObject
