# AutoSyncTriggerMode

自动同步触发模式枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.DistributedDataManager.CloudSync.Client

## ACCOUNT_LOGIN

```TypeScript
ACCOUNT_LOGIN = 0
```

账号登录触发模式。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.DistributedDataManager.CloudSync.Client
