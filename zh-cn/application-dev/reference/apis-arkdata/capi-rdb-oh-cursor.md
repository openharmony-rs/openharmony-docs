## 概述

表示结果集。提供通过查询数据库生成的数据库结果集的访问方法。

**起始版本：** 10

**相关模块：** [RDB](capi-rdb.md)

**所在头文件：** [oh_cursor.h](capi-oh-cursor-h.md)

```

**描述**

```

| 参数项              | 描述                                                         |
| ------------------- | ------------------------------------------------------------ |
| OH_Cursor *cursor   | 表示指向OH_Cursor实例的指针                                  |
