## 枚举类型说明

### Rdb_EncryptionAlgo

```
enum Rdb_EncryptionAlgo
```

**描述**

数据库加密算法。

**起始版本：** 20

| 枚举项              | 描述                                |
| ------------------- | ----------------------------------- |
| RDB_AES_256_GCM = 0 | 表示数据库使用RDB_AES_256_GCM加密。 |
| RDB_AES_256_CBC     | 表示数据库使用RDB_AES_256_CBC加密。 |

enum Rdb_HmacAlgo

| RDB_HMAC_SHA256   | RDB_HMAC_SHA256算法。 |
| RDB_HMAC_SHA512   | RDB_HMAC_SHA512算法。 |
