# RunningMode（系统接口）

设备管理的运行模式。

**起始版本：** 19

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

**系统接口：** 此接口为系统接口。

## DEFAULT

```TypeScript
DEFAULT = 0
```

默认用户运行模式，表示应用在首次开机后的用户下运行。

**起始版本：** 19

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

**系统接口：** 此接口为系统接口。

## MULTI_USER

```TypeScript
MULTI_USER = 1
```

多用户运行模式，表示应用能够在多个用户下同时运行。

**起始版本：** 19

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

**系统接口：** 此接口为系统接口。
