# DockInfo

快捷栏中的应用信息。

**起始版本：** 24

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## abilityName

```TypeScript
abilityName: string
```

应用的Ability名称。

**类型：** string

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## bundleName

```TypeScript
bundleName: string
```

应用的包名。

**类型：** string

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## index

```TypeScript
index: number
```

应用在快捷栏中的位置索引。
取值范围为全体整数。

**类型：** number

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager
