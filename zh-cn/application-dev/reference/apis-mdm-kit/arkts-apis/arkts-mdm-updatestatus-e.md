# UpdateStatus

系统更新状态。

**起始版本：** 12

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## NO_UPDATE_PACKAGE

```TypeScript
NO_UPDATE_PACKAGE = -4
```

指定版本系统更新包不存在。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager
