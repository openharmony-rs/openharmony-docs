# UsbPolicy

USB读写策略的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## READ_WRITE

```TypeScript
READ_WRITE = 0
```

可读可写。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## READ_ONLY

```TypeScript
READ_ONLY = 1
```

只读。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## DISABLED

```TypeScript
DISABLED = 2
```

禁用。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager
