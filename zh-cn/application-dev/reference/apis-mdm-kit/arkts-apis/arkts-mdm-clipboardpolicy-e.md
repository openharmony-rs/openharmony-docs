# ClipboardPolicy

设备剪贴板策略。

**起始版本：** 12

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## DEFAULT

```TypeScript
DEFAULT = 0
```

默认，表示无策略。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## IN_APP

```TypeScript
IN_APP = 1
```

剪贴板可在同一应用使用。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## LOCAL_DEVICE

```TypeScript
LOCAL_DEVICE = 2
```

剪贴板可在同一设备使用。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager

## CROSS_DEVICE

```TypeScript
CROSS_DEVICE = 3
```

剪贴板可跨设备使用。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Customization.EnterpriseDeviceManager
