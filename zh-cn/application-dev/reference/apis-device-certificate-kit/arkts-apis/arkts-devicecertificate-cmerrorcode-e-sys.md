# CMErrorCode

表示调用证书管理相关API的错误码。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_ERROR_NOT_SYSTEM_APP

```TypeScript
CM_ERROR_NOT_SYSTEM_APP = 202
```

表示应用程序不是系统应用程序。

此接口为系统接口。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

**系统接口：** 此接口为系统接口。

## CM_ERROR_PASSWORD_IS_ERR

```TypeScript
CM_ERROR_PASSWORD_IS_ERR = 17500008
```

表示密码错误。

此接口为系统接口。

26.0.0

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.CertificateManager

**系统接口：** 此接口为系统接口。
