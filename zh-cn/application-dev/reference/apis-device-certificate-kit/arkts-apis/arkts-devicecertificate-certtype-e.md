# CertType

表示证书类型。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager

## CA_CERT_SYSTEM

```TypeScript
CA_CERT_SYSTEM = 0
```

表示系统CA证书。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager

## CA_CERT_USER

```TypeScript
CA_CERT_USER = 1
```

表示用户CA证书。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager
