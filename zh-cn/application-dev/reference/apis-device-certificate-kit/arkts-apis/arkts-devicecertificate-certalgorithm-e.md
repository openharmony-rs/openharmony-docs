# CertAlgorithm

表示证书的算法类型。

**起始版本：** 20

**系统能力：** SystemCapability.Security.CertificateManager

## INTERNATIONAL

```TypeScript
INTERNATIONAL = 1
```

表示国际密码算法，如RSA、NIST ECC等。

**起始版本：** 20

**系统能力：** SystemCapability.Security.CertificateManager

## SM

```TypeScript
SM = 2
```

表示商用密码算法，如SM2、SM4等。
海外设备不支持使用该算法的证书。

**起始版本：** 20

**系统能力：** SystemCapability.Security.CertificateManager
