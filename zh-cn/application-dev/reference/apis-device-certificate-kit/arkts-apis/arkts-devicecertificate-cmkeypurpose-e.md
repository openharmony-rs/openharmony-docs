# CmKeyPurpose

表示密钥使用目的的枚举，用于签名、验签。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_KEY_PURPOSE_SIGN

```TypeScript
CM_KEY_PURPOSE_SIGN = 4
```

签名。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_KEY_PURPOSE_VERIFY

```TypeScript
CM_KEY_PURPOSE_VERIFY = 8
```

验签。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager
