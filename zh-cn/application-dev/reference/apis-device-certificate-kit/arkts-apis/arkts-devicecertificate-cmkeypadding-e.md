# CmKeyPadding

表示签名、验签使用的填充方式的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_PADDING_NONE

```TypeScript
CM_PADDING_NONE = 0
```

无填充。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_PADDING_PSS

```TypeScript
CM_PADDING_PSS = 1
```

PSS方式填充。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_PADDING_PKCS1_V1_5

```TypeScript
CM_PADDING_PKCS1_V1_5 = 2
```

PKCS1_V1_5方式填充。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager
