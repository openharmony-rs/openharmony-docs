# CertFileFormat

表示证书文件格式。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Security.CertificateManager

## PEM_DER

```TypeScript
PEM_DER = 0
```

表示证书文件格式为PEM或DER。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.CertificateManager

## P7B

```TypeScript
P7B = 1
```

表示证书文件格式为P7B。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.CertificateManager
