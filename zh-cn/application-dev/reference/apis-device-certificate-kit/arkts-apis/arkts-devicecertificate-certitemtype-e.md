# CertItemType

表示获取证书字段的枚举。

**起始版本：** 10

**系统能力：** SystemCapability.Security.Cert

## CERT_ITEM_TYPE_TBS

```TypeScript
CERT_ITEM_TYPE_TBS = 0
```

表示获取证书的待签名信息。

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
