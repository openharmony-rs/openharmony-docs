# CertChainValidationResult

表示证书链校验的返回值。

**起始版本：** 11

**系统能力：** SystemCapability.Security.Cert

## entityCert

```TypeScript
readonly entityCert: X509Cert
```

表示实体证书。

**类型：** X509Cert

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
