# AuthStorageLevel

表示凭据的存储级别。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager

## EL1

```TypeScript
EL1 = 1
```

EL1级别，表示设备启动后可以访问。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager

## EL2

```TypeScript
EL2 = 2
```

EL2级别，表示设备首次解锁后可以访问。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager

## EL4

```TypeScript
EL4 = 4
```

EL4级别，表示设备解锁时可以访问。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManager
