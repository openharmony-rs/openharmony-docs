# CmKeyDigest

表示签名、验签使用的摘要算法的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager

## CM_DIGEST_NONE

```TypeScript
CM_DIGEST_NONE = 0
```

选用此项时，表示由应用程序对待签名、验签的数据进行摘要计算。

**起始版本：** 11

**系统能力：** SystemCapability.Security.CertificateManager
