# ExtensionEntryType

表示获取扩展域中对象类型的枚举。

**起始版本：** 10

**系统能力：** SystemCapability.Security.Cert

## EXTENSION_ENTRY_TYPE_ENTRY

```TypeScript
EXTENSION_ENTRY_TYPE_ENTRY = 0
```

表示获取整个对象。

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## EXTENSION_ENTRY_TYPE_ENTRY_CRITICAL

```TypeScript
EXTENSION_ENTRY_TYPE_ENTRY_CRITICAL = 1
```

表示获取对象的critical属性。

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## EXTENSION_ENTRY_TYPE_ENTRY_VALUE

```TypeScript
EXTENSION_ENTRY_TYPE_ENTRY_VALUE = 2
```

表示获取对象的数据。

**起始版本：** 10

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
