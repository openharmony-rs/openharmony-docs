# CertChainBuildResult

用于指定证书链创建结果。

**起始版本：** 12

**系统能力：** SystemCapability.Security.Cert

## certChain

```TypeScript
readonly certChain: X509CertChain
```

生成的证书链对象。

**类型：** X509CertChain

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## validationResult

```TypeScript
readonly validationResult: CertChainValidationResult
```

指定最终证书链的最大长度。

**类型：** CertChainValidationResult

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.Cert
