# OcspDigest

表示OCSP摘要算法的枚举。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Security.Cert

## SHA1

```TypeScript
SHA1 = 0
```

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。
