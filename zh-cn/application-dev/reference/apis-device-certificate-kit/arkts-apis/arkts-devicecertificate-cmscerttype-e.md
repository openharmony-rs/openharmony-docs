# CmsCertType

从CMS中获取证书不同类型的枚举。

**起始版本：** 22

**系统能力：** SystemCapability.Security.Cert

## SIGNER_CERTS

```TypeScript
SIGNER_CERTS = 0
```

签名者证书。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## ALL_CERTS

```TypeScript
ALL_CERTS = 1
```

全部证书。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
