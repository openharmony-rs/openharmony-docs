# EncodingBlob

带编码格式的证书二进制数组。

**起始版本：** 9

**系统能力：** SystemCapability.Security.Cert

## data

```TypeScript
data: Uint8Array
```

传入的证书数据。

**类型：** Uint8Array

**起始版本：** 9

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## encodingFormat

```TypeScript
encodingFormat: EncodingFormat
```

指明证书编码格式。

**类型：** EncodingFormat

**起始版本：** 9

**原子化服务API：** 从API版本12开始，该接口支持在原子化服务API中使用。

**系统能力：** SystemCapability.Security.Cert
