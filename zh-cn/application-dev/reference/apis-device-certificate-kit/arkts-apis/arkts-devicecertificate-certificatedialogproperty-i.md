# CertificateDialogProperty

表示证书管理对话框的属性。

**起始版本：** 18

**系统能力：** SystemCapability.Security.CertificateManagerDialog

## showInstallButton

```TypeScript
showInstallButton: boolean
```

表示是否显示安装证书的按钮，true为显示，false为不显示。

**类型：** boolean

**起始版本：** 18

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.CertificateManagerDialog
