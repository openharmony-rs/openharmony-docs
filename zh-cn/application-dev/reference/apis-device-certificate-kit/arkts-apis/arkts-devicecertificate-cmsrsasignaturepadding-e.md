# CmsRsaSignaturePadding

表示RSA类型CMS签名填充方式的枚举。

**起始版本：** 22

**系统能力：** SystemCapability.Security.Cert

## PKCS1_PADDING

```TypeScript
PKCS1_PADDING = 0
```

PKCS1填充方式。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## PKCS1_PSS_PADDING

```TypeScript
PKCS1_PSS_PADDING = 1
```

PKCS1 PSS填充方式。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
