# CsrAttribute

定义CSR属性表示。

CSR属性字段，当前仅支持字符串类型的属性字段，属性值添加到CSR中编码为utf-8。常见的type为challengePassword。

**起始版本：** 18

**系统能力：** SystemCapability.Security.Cert

## type

```TypeScript
type: string
```

PKCS #9指定的扩展类型。

**类型：** string

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## value

```TypeScript
value: string
```

属性值。

**类型：** string

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
