# CmsContentType

表示Cms内容类型的枚举。

**起始版本：** 18

**系统能力：** SystemCapability.Security.Cert

## SIGNED_DATA

```TypeScript
SIGNED_DATA = 0
```

签名数据。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## ENVELOPED_DATA

```TypeScript
ENVELOPED_DATA = 1
```

封装数据。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
