# CertificateType

表示安装证书的类型。

**起始版本：** 14

**系统能力：** SystemCapability.Security.CertificateManagerDialog

## CA_CERT

```TypeScript
CA_CERT = 1
```

CA证书。

**起始版本：** 14

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Security.CertificateManagerDialog
