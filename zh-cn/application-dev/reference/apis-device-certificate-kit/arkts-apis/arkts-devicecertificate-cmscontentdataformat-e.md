# CmsContentDataFormat

表示Cms内容数据格式的枚举。

**起始版本：** 18

**系统能力：** SystemCapability.Security.Cert

## BINARY

```TypeScript
BINARY = 0
```

表示二进制数据格式。

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert

## TEXT

TEXT = 1

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Security.Cert
