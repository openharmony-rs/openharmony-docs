# AutoAdComponent

本模块提供展示轮播广告的能力。

**起始版本：** 11

**系统能力：** SystemCapability.Advertising.Ads

## build

```TypeScript
build(): void
```

用于创建AutoAdComponent对象的构造函数。

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Advertising.Ads
