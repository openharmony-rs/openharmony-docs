# InputWindowInfo

输入法软键盘的窗口信息。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## userId

```TypeScript
userId?: number
```

显示输入法窗口的用户ID。

该属性仅系统应用可以使用。

**类型：** number

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**系统接口：** 此接口为系统接口。
