# EnterKeyType

Enter键的功能类型。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## UNSPECIFIED

```TypeScript
UNSPECIFIED = 0
```

未指定。

**使用场景：**编辑框不指定Enter键具体功能时使用。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## NONE

```TypeScript
NONE
```

NONE。

**使用场景：**Enter键无特定行为，仅作为换行或普通按键使用。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## GO

```TypeScript
GO
```

前往。

**使用场景：**适用于URL输入框，Enter键触发"前往"操作，如打开链接。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## SEARCH

```TypeScript
SEARCH
```

查找。

**使用场景：**适用于搜索框，Enter键触发搜索操作。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## SEND

```TypeScript
SEND
```

**使用场景：**适用于消息发送框，Enter键触发发送操作。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## NEXT

```TypeScript
NEXT
```

下一步。

**使用场景：**适用于多步骤表单，Enter键跳转到下一个输入框。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

```TypeScript
DONE
```
