# CapitalizeMode

枚举，定义了文本首字母大写的不同模式。

| 名称 | 值 | 说明 |
| -------- | -- | -------- |
| NONE | 0 | 不进行任何首字母大写处理。|
| SENTENCES | 1 | 每个句子的首字母大写。|
| WORDS | 2 | 每个单词的首字母大写。|
| CHARACTERS | 3 | 每个字母都大写。|

**起始版本：** 20

**系统能力：** SystemCapability.MiscServices.InputMethodFramework
