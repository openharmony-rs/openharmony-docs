# RequestKeyboardReason

枚举，请求键盘输入的原因。

| 名称 | 值 | 说明 |  
| ------------ | -- | ------------------ |  
| NONE | 0 | 表示没有特定的原因触发键盘请求。 |  
| MOUSE | 1 | 表示键盘请求是由鼠标操作触发的。 |  
| TOUCH | 2 | 表示键盘请求是由触摸操作触发的。 |  
| OTHER | 20 | 表示键盘请求是由其他原因触发的。 |

**起始版本：** 19
