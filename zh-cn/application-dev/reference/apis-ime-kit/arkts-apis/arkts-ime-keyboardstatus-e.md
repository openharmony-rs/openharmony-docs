# KeyboardStatus

输入法软键盘状态。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## NONE

```TypeScript
NONE = 0
```

NONE。

**使用场景：**表示键盘状态尚未确定或无法判断时使用。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## HIDE

```TypeScript
HIDE = 1
```

隐藏状态。

**使用场景：**表示当前软键盘处于隐藏状态。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## SHOW

```TypeScript
SHOW = 2
```

显示状态。

**使用场景：**表示当前软键盘处于显示状态。

**起始版本：** 10

**系统能力：** SystemCapability.MiscServices.InputMethodFramework
