# KeyEvent

按键属性值。

**起始版本：** 8

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## keyAction

```TypeScript
readonly keyAction: number
```

按键事件类型。

- 当值为2时，表示按下事件；
- 当值为3时，表示抬起事件。

**类型：** number

**起始版本：** 8

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## keyCode

```TypeScript
readonly keyCode: number
```

按键的键值。键码值说明参考[KeyCode](../../apis-input-kit/arkts-apis/arkts-input-keycode-e.md)。

**类型：** number

**起始版本：** 8

**系统能力：** SystemCapability.MiscServices.InputMethodFramework
