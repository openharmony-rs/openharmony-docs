
未启用。

## BASIC_MODE

```TypeScript
BASIC_MODE
```

基础模式。

**使用场景：**输入法已启用但处于基础模式，仅具备基础输入能力，不支持高级功能（如自定义通信）。

**起始版本：** 15

## FULL_EXPERIENCE_MODE

```TypeScript
FULL_EXPERIENCE_MODE
```

完整体验模式。

**使用场景：**输入法已启用且处于完整体验模式，支持所有功能（包括自定义通信、预上屏等）。

**起始版本：** 15
