# FluidLightMode（系统接口）

枚举，输入法流光模式。

| 名称 | 值 | 说明 |
| ------------ | -- | ------------------ |
| NONE | 0 | 不使用流光模式。 |
| BACKGROUND_FLUID_LIGHT | 1 | 开启背景流光模式。系统面板变为透明，流光效果由编辑框宿主应用实现。 |

**起始版本：** 20

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**系统接口：** 此接口为系统接口。

## NONE

```TypeScript
NONE = 0
```

不使用流光模式。

**起始版本：** 20

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**系统接口：** 此接口为系统接口。

## BACKGROUND_FLUID_LIGHT

```TypeScript
BACKGROUND_FLUID_LIGHT = 1
```

开启背景流光模式。此时系统面板会变为透明，流光效果需要由编辑框宿主应用实现。

**起始版本：** 20

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

**系统接口：** 此接口为系统接口。
