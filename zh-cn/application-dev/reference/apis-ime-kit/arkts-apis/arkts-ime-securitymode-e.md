# SecurityMode

输入法的安全模式，如BASIC或FULL。

**起始版本：** 11

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## BASIC

```TypeScript
BASIC = 0
```

基础访问模式，基础打字模式，会限制网络访问。

**起始版本：** 11

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## FULL

```TypeScript
FULL
```

完全访问模式，不做限制，可以访问网络。

**起始版本：** 11

**系统能力：** SystemCapability.MiscServices.InputMethodFramework
