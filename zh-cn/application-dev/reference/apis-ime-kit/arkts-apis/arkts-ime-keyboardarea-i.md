# KeyboardArea

面板中的键盘区域。

**起始版本：** 15

**系统能力：** SystemCapability.MiscServices.InputMethodFramework

## bottom

```TypeScript
bottom: number
```

键盘区域的下边界到面板区域下边界的距离，单位为px，该参数为整数。

**类型：** number

**起始版本：** 15

**系统能力：** SystemCapability.MiscServices.InputMethodFramework
