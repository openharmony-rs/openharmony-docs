# AccessibilityFocusScene（系统接口）

Enumeration of scenes of accessibility focus.

**起始版本：** 26.0.0

**系统能力：** SystemCapability.BarrierFree.Accessibility.Core

**系统接口：** 此接口为系统接口。

## HOVER_FOCUS

```TypeScript
HOVER_FOCUS = 1
```

Hover to move the accessibility focus.

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.BarrierFree.Accessibility.Core

**系统接口：** 此接口为系统接口。
