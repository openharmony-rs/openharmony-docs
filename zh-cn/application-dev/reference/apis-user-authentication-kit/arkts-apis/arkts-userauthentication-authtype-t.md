# AuthType

```TypeScript
type AuthType = 'ALL' | 'FACE_ONLY'
```

表示认证类型。

**起始版本：** 6

**废弃版本：** 8

**替代接口：** [UserAuthType](arkts-userauthentication-userauthtype-e.md)

**系统能力：** SystemCapability.UserIAM.UserAuth.Core

| 类型 | 说明 |
| --- | --- |
| 'ALL' | 预留参数，当前版本暂不支持ALL类型的认证。 |
| 'FACE_ONLY' | 人脸认证。 |
