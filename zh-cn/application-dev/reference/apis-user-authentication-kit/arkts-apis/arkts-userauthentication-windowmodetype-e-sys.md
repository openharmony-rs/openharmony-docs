# WindowModeType（系统接口）

用户认证界面的显示类型枚举。该枚举定义了认证界面可使用的显示模式，用于控制系统身份认证组件的窗口样式

**起始版本：** 10

**系统能力：** SystemCapability.UserIAM.UserAuth.Core

**系统接口：** 此接口为系统接口。

## DIALOG_BOX

```TypeScript
DIALOG_BOX = 1
```

对话框类型。身份认证界面以对话框形式显示，适用于大多数认证场景，用户体验较好。

**起始版本：** 10

**系统能力：** SystemCapability.UserIAM.UserAuth.Core

**系统接口：** 此接口为系统接口。

## FULLSCREEN

```TypeScript
FULLSCREEN = 2
```

全屏类型。身份认证界面以全屏形式显示，适用于需要沉浸式认证体验或认证信息较多需要更大展示空间的场景。

**起始版本：** 10

**系统能力：** SystemCapability.UserIAM.UserAuth.Core

**系统接口：** 此接口为系统接口。
