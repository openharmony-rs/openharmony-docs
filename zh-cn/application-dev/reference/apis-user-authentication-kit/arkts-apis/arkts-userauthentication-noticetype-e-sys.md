# NoticeType（系统接口）

用户身份认证的通知类型枚举。该枚举定义了系统支持的通知类型，用于标识通知的来源。

**起始版本：** 10

**系统能力：** SystemCapability.UserIAM.UserAuth.Core

**系统接口：** 此接口为系统接口。

## WIDGET_NOTICE

```TypeScript
WIDGET_NOTICE = 1
```

表示该通知由系统统一身份认证控件发出，用于通知用户认证框架相关事件。

**起始版本：** 10

**系统能力：** SystemCapability.UserIAM.UserAuth.Core

**系统接口：** 此接口为系统接口。
