# DoNotDisturbType（系统接口）

> **说明：**
> 从 API version 8开始支持，从API version 9开始废弃。建议使用
[notificationManager.DoNotDisturbType](arkts-notification-donotdisturbtype-e-sys.md)替代。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** DoNotDisturbType

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_NONE

```TypeScript
TYPE_NONE = 0
```

非通知勿扰类型。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** TYPE_NONE

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_ONCE

```TypeScript
TYPE_ONCE = 1
```

以设置时间段(只看小时和分钟)一次执行勿扰。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** TYPE_ONCE

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_DAILY

```TypeScript
TYPE_DAILY = 2
```

以设置时间段(只看小时和分钟)每天执行勿扰。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** TYPE_DAILY

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_CLEARLY

```TypeScript
TYPE_CLEARLY = 3
```

以设置时间段(明确年月日时分)执行勿扰。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** TYPE_CLEARLY

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。
