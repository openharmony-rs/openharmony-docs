# PriorityNotificationType

描述通知的优先级类型。

**起始版本：** 23

**系统能力：** SystemCapability.Notification.Notification

## OTHER

```TypeScript
OTHER = 'OTHER'
```

表示通知优先级类型为默认。

**起始版本：** 23

**系统能力：** SystemCapability.Notification.Notification

## PRIMARY_CONTACT

```TypeScript
PRIMARY_CONTACT = 'PRIMARY_CONTACT'
```

表示通知优先级类型为重要联系人。

**系统能力：** SystemCapability.Notification.Notification
