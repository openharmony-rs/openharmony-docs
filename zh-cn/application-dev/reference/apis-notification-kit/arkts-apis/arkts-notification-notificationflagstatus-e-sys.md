# NotificationFlagStatus（系统接口）

描述通知标志状态。

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_NONE

```TypeScript
TYPE_NONE = 0
```

默认标志，与TYPE_OPEN效果相同。

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_OPEN

```TypeScript
TYPE_OPEN = 1
```

通知标志打开。

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## TYPE_CLOSE

```TypeScript
TYPE_CLOSE = 2
```

通知标志关闭。

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。
