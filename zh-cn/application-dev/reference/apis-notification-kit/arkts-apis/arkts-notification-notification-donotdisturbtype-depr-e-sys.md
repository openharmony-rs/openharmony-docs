# DoNotDisturbType（系统接口）

> **说明：**  
> 从 API version 8开始支持，从API version 9开始废弃。建议使用[notificationManager.DoNotDisturbType](arkts-notification-notificationmanager-donotdisturbtype-e-sys.md)替代。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** DoNotDisturbType
