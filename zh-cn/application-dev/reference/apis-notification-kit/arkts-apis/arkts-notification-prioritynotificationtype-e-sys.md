# PriorityNotificationType

描述通知的优先级类型。

**起始版本：** 23

**系统能力：** SystemCapability.Notification.Notification

## PAYMENT_DUE

```TypeScript
PAYMENT_DUE = 'PAYMENT_DUE'
```

表示通知优先级类型为缴费还款。

**起始版本：** 23

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。
