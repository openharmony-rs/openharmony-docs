# RemoveReason（系统接口）

通知删除原因。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## CLICK_REASON_REMOVE

```TypeScript
CLICK_REASON_REMOVE = 1
```

点击通知后删除通知。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。
