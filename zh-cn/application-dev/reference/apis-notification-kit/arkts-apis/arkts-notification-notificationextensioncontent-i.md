# NotificationExtensionContent

通知扩展内容。
> **说明：**  
>  
> 本模块首批接口从API version 22开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

**起始版本：** 22

## text

```TypeScript
text: string
```

通知内容。不能为空且不能超过3072字节，超出内容将被截断。

**类型：** string

**起始版本：** 22

## title

```TypeScript
title: string
```

通知标题。不能为空且不能超过1024字节，超出内容将被截断。

**类型：** string

**起始版本：** 22
