# ContentType

通知内容类型。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.Notification

## NOTIFICATION_CONTENT_BASIC_TEXT

```TypeScript
NOTIFICATION_CONTENT_BASIC_TEXT
```

普通文本类型通知。

**起始版本：** 9

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Notification.Notification

```TypeScript

```
