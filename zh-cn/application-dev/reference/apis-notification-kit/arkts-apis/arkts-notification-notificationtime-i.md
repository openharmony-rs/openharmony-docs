# NotificationTime

描述通知计时信息。

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

## initialTime

```TypeScript
initialTime?: number
```

计时起始时间。单位：ms。

**类型：** number

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

## isCountDown

```TypeScript
isCountDown?: boolean
```

是否倒计时。默认为false。

- true：是。
- false：否。

**类型：** boolean

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

## isInTitle

```TypeScript
isInTitle?: boolean
```

时间是否展示在title中。默认为false。

- true：是。
- false：否。

**类型：** boolean

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification

## isPaused

```TypeScript
isPaused?: boolean
```

是否暂停。默认为false。

- true：是。
- false：否。

**类型：** boolean

**起始版本：** 11

**系统能力：** SystemCapability.Notification.Notification
