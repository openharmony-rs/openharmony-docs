# VoiceContentOptions（系统接口）

描述订阅通知的语音播报内容配置项。

**起始版本：** 26.0.0

<!--Device-unnamed-export interface VoiceContentOptions--><!--Device-unnamed-export interface VoiceContentOptions-End-->

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## enabled

```TypeScript
enabled?: boolean
```

是否订阅通知语音播报内容。  
- true：订阅。  
- false：不订阅。默认值为false。

**类型：** boolean

**默认值：** false

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。
