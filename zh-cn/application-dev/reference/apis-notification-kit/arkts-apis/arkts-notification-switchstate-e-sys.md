# SwitchState（系统接口）

描述通知相关开关的设置状态。

**起始版本：** 20

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## USER_MODIFIED_OFF

```TypeScript
USER_MODIFIED_OFF = 0
```

表示用户设置的关闭状态。

**起始版本：** 20

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## USER_MODIFIED_ON

```TypeScript
USER_MODIFIED_ON = 1
```

表示用户设置的开启状态。

**起始版本：** 20

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## SYSTEM_DEFAULT_OFF

```TypeScript
SYSTEM_DEFAULT_OFF = 2
```

表示在用户设置前的初始关闭状态。

**起始版本：** 20

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。

## SYSTEM_DEFAULT_ON

```TypeScript
SYSTEM_DEFAULT_ON = 3
```

表示在用户设置前的初始开启状态。

**起始版本：** 20

**系统能力：** SystemCapability.Notification.Notification

**系统接口：** 此接口为系统接口。
