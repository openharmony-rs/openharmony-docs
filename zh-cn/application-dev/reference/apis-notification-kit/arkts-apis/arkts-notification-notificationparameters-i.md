# NotificationParameters

**起始版本：** 24

**系统能力：** SystemCapability.Notification.Notification

## wantAction

```TypeScript
wantAction?:string
```

应用在创建wantAgent时，传入的want的action字段，具体含义请参考[action](../../apis-ability-kit/arkts-apis/arkts-ability-want-c.md)。

**类型：** string

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Notification.Notification
