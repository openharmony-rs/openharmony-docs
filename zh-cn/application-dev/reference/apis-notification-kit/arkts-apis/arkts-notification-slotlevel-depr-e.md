# SlotLevel

通知级别。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** SlotLevel

**系统能力：** SystemCapability.Notification.Notification

## LEVEL_NONE

```TypeScript
LEVEL_NONE = 0
```

表示关闭通知功能。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** LEVEL_NONE

**系统能力：** SystemCapability.Notification.Notification

## LEVEL_MIN

```TypeScript
LEVEL_MIN = 1
```

表示通知功能已启用，但状态栏中不显示通知图标，且没有横幅或提示音。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** LEVEL_MIN

**系统能力：** SystemCapability.Notification.Notification

## LEVEL_LOW

```TypeScript
LEVEL_LOW = 2
```

表示通知功能已启用，且状态栏中显示通知图标，但没有横幅或提示音。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** LEVEL_LOW

**系统能力：** SystemCapability.Notification.Notification

## LEVEL_DEFAULT

```TypeScript
LEVEL_DEFAULT = 3
```

表示通知功能已启用，状态栏中显示通知图标，没有横幅但有提示音。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** LEVEL_DEFAULT

**系统能力：** SystemCapability.Notification.Notification

## LEVEL_HIGH

```TypeScript
LEVEL_HIGH = 4
```

表示通知功能已启用，状态栏中显示通知图标，有横幅和提示音。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** LEVEL_HIGH

**系统能力：** SystemCapability.Notification.Notification
