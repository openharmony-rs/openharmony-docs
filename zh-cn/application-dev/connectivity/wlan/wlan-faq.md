## 是否有API支持切换网络，以解决在特定情况下无法连接外网的问题

connection.setAppNet API 支持切换网络，可以解决特定网络无法连接外网的问题。可参考（https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-connection#connectionsetappnet9-1）
