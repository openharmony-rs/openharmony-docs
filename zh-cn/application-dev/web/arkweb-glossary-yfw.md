# ArkWeb术语解释

## A

### App Side；应用侧

运行应用原生代码的一侧环境。

## C

### Cache Mode；缓存模式

Web组件加载资源的缓存策略。

### Cross-origin Access；跨域访问

从一个源访问另一个源资源的行为。

## E

### Extension Area，扩展区

三方应用可以扩展的User-Agent字段区域。

## I

### Incognito Mode；无痕模式

不保留浏览记录和数据的浏览模式。

### Intelligent Tracking Prevention；智能防跟踪

自动拦截跟踪型网站的功能。

## J

### JSBridge

JavaScript桥接，应用与Web页面之间进行JavaScript交互的机制。

## L

### Local Storage

本地存储，保存在应用目录下的持久化数据。

### Local Offline Resource；本地离线资源

存储在本地的离线网页资源。

## M

### Message Communication；消息通信

前端页面和应用侧之间的双向通信。

### Message Event；消息事件

消息端口接收消息时触发的事件。

### Message Port；消息端口

用于双向通信的消息通道。

### MixedContent Policy；混合内容策略

控制HTTP和HTTPS混合内容的机制。

## N

### Network Permission；网络权限

应用访问网络所需的权限声明。

## P

### Page Loop Redirect；页面循环跳转

页面跳转逻辑陷入死循环的问题。

### Persistent Storage；持久化存储

数据写入本地永久保存的存储方式。

### Privacy Mode；隐私模式

不保留浏览数据的隐私浏览模式。

## R

### rawfile

原始文件目录，应用资源目录下的原始文件存储位置。

## S

### SameSite

同站属性，Cookie的安全属性用于控制跨站请求。

### Scheme Handler

Scheme处理器，用于拦截和处理Web组件发起的特定协议网络请求。

### SDK

软件开发工具包，提供开发接口的软件包。

### session cookie

会话Cookie，仅在会话期间有效的临时Cookie。

### Session Storage

会话存储，DOM Storage中的临时存储机制。

### Static Web Page

静态网页，内容固定不变的网页类型。

### Sync Method；同步方法

同步执行的JSBridge方法。

## T

### Tracking Website；跟踪型网站

用于跟踪用户行为的第三方网站。

## U

### UA Compatibility；UA兼容性

User-Agent与网页适配的问题。

### UI Thread；UI线程

负责处理用户界面更新的线程。

## V

### ValidCallback

有效回调，Native侧注册的示例生命周期回调。

### viewport

视口，网页的可视区域。

### View Layer；视图层

小程序架构中负责界面渲染的部分。

## W

### WebDriver

Selenium的Web自动化驱动接口，通过ChromeDriver与Web内核通信。

### WebDriverSetupTool

WebDriver设置工具类，用于初始化和管理ChromeDriver连接。

### webTag

Web标签，用于标识Web组件的唯一标识符。

### Window Object；window对象

浏览器中的全局窗口对象。

## X

### XMLHttpRequest

XMLHttpRequest对象，用于发送HTTP请求的JavaScript对象。

## 其他

### 2in1 Device；2in1设备

支持触屏和键盘输入的二合一设备类型。
