# 全球化子系统Changelog

## cl.notofonts.1 notofonts三方件小语种字体升级变更

**访问级别**

公共能力

**变更原因**

当前版本存在缺失错别字和字符缺失的问题，变更后可修复错别字，补充字符。

**变更影响**

变更前：蒙古语、天城文、缅甸语显示错别字，部分数学符号字体显示偏小。

变更后：蒙古语、天城文、缅甸语显示正常，部分数学符号字体显示偏大。

**起始 API Level**

26.0.0

**变更发生版本**

从OpenHarmony SDK 7.0.0.21开始。

**变更的接口/组件**

不涉及

**适配指导**

蒙古语、天城文、缅甸语应用无需做适配即可显示正常；符号字体偏大，显示效果会更好更清晰，但是由于字体变大，可能出现界面变动，需要应用根据实际情况调整界面排版。
